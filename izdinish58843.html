ئۇيغۇرچە javascript قوللانمىسى

ئەسسەلامۇ ئەلەيكۇم، تۆۋەندە javascript دىن قىسقىغىنە چۈشەنچە بېرىمەن
javascript بولسا ئىنتىرنىتتا ئەڭ مودا بولغان بىر قوليازما تېلى بولۇپ، بۇ ئاساسلىق توركۆرگۈچلەردە خىزمەت قىلىدۇ ،جۈملىدىن Ie ,Firefox.Safair chrome قاتارلىقلار سىز بۇ تىل بىلەن پروگىرامما يىزىشنى باشلاشدىن بۇرۇن چوقۇم HTML / XHTML نى بىلىشىڭىز كىرەك ، js بەتتىكى ھەربىر ئىلىمىنتى باشقۇرۇش خۇسۇسىيتىغا ئىگە،دىمەكچىمەنكى javascript ئارقىلىق بەتتىكى ھەربىر dom ئىلىمىنىتنى خالىغانچە كونتىرول قىلالايسىز،شۇنداقلا بۇ تىل يانفۇن ۋە كومپيۇتىرنىڭ قاتتىق دىتاللىرىغا كىرىش ئىقتىدارىغا ئىگە ،جۈملىدىن كامىرا،مىكروفون،قاتتىق دىسكىدىكى ھۆججەت قاتارلىقلار. مەقسەتكە كەلسەك بۇ قوللانمىنى يىزىشتىكى مەقسەت قانداقتۇر javascript نى يڭىدىن باشلىغۇچىلار ئۈچۈن پايدىلىنىش ماتىريالى قىلىش ئەمەس،بەلكى javascript دىكى دائىملىق خاتا چۈشەنچىلەر ۋە بىرقسىم مۇرەككەپ مەسىلىلەرنى يىشىپ بىرىش،شۇڭلاشقا ئەگەر سىز javascript دىن ئاز تولا خەۋرىڭىز بولماي تۇرۇپ ،بۇ قوللانمىدىن پايدىلانماقچى بولسىڭىز كۆزلىگەن مەقستىڭىزگە ئىرىشەلمەسلىكىڭىز مومكىن،چۈنكى بۇ قوللانما javascript دىن ئوتتۇرھال سەۋىيەگە ئىگە بولغانلارغا ماس كىلىدۇ، بىشىدىن باشلىماقچى بولسىڭىز ئىنتىرنىتتىكى باشقا قوللانملارنى بىرقاتار كۆرۈڭ ۋە ئەمەلى سىناپ يىزىپ ئاساسى چۈشەنچىگە ئىگە بولۇڭ،ئاندىن بۇ قوللانما سىزگە ،سىزJavascript دا ئۇچىرىغان بىشىڭىزنى قايدۇرغان قىيىن مەسىلىلەرنى ھەل قىلىشتا زور ياردىمى تىگىدۇ، ئەگەر ئاللاھ خالىسا...

بۇ قوللانمىنى تۆۋەندىكى بىكەتكە كىرىىپ كۆرسىڭىز بولىدۇ،پات ئارىدا ئاللاھ خالىسا كىتاب نەشىرىنىمۇ ئىلان قىلىشىم مومكىن،بۇ قوللانمىنىڭ مەزمۇنلىرى داۋاملىق تۈردە قوشۇلۇپ ماڭىدۇ،قوللانمىنىڭ تېخىمۇ كۆپ ئۈنۈمى ۋە ئۇزۇن مۇددەتلىك پايدىسى بولۇشى ئۈچۈن مەخسۇس بىكەت ياسالدى
