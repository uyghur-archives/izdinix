CPU نىڭ ساختىلىقىنى قانداق پەرىق ئىتىمىز؟

Intel نىڭ CPU ( مەركىزى بىر تەرەپ قىلغۇچ )سىنىڭ ساختا ياكى ھەقىقىيلىقىنى قانداق بىلىمىز؟
ھەممىمىزگە مەلۇم CPU بولسا كومپيۇتېرنىڭ ئەڭ يادرولۇق قىسىمى ، بىز ئۇنداق بولسا Intel شىركىتىنىڭ CPU ( مەركىزى بىر تەرەپ قىلغۇچ )سىنىڭ ساختا ياكى ھەقىقىي ئىكەنلىكىنى قانداق پەرىق ئىتىمىز؟
Intel شىركىتى بۇ مەسلىنى ھەل قىلىش ئۈچۈن مەخسۇس ئۆزىنىڭ CPU نىڭ ساختىلىقىنى پەرىق ئىتىدىغان يۇمشاق دىتالدىن بىرنى تەمىنلىگەن ، سىز بۇم يۇمشاق دىتال ئارقىلىق CPU ڭىزنىڭ ئەھۋالى بىلەن تونۇشۇپ چىقالايسىز.
ئىسمى : Intel(R) Processor ID Utility
چۈشۈرۈش ئادرىسى : http://www.orsoon.com/Soft/2652.html
ئۇ قوللايدىغان CPU تىپلىرى بۇ يەردە : http://support.intel.com/support/processors/tools/piu/sb/CS-015472.htm
