مەنWindows 8 توغرىسىدا ياردەمگە مۇھتاج بولۇپ قالدىم، مۇشۇ يەردىلا دەۋالاي:

6- ئايدا باشقىلاردىن 248 يۈەنگە Windows 8 سېتىۋالغان، ساتقۇچى ئىزدىنىشنىڭ بىرسوم پىلانىغا پۇل ئىئانە قىلامسىز دەپتى خۇشاللىق بىلەن 50 سوم پۇلمۇ ئىئانە قىلغان، بىرەر ئايغىچە ئىشلىتىپ كۆنەلمەي كونا سېستىمىنى قاچىلاپ، كونا سېستىما يەنىلا Windows 8 گە يەتمەيدىكەن دەپ قاراپ توركەش دېگەن تور بەتتىن Windows 8 نى چۈشۈرۈپ قاچىلىسام ھېچقانداق مەخپى نومۇر سورىماي قاچىلاندى. لىكىن مەن سېتىۋالغان Windows 8 نىڭ ھوزۇرۇنى كۆرەلمەي. بولدىلا ئۆزەمنىڭ ھالال نەشىردىكى سېستىمامنى ئىشلىتەي دەپ، توردىن Windows 8 نى چۈشۈرۈپ قاچىلىسام قاچىلىغان كۈنۈم سېستىما ياخشى ماڭدى. ئەتىسىدىن باشلاپ تەخمىنەن ھەر تۆت سائەتتە بىر قېتىم بۇ ئاچقۇچ نومۇرنى باشقىلار ئىشلىتىۋاتىدۇ، قايتىدىن يېڭى ئاچقۇچ نومۇر كىرگۈزۈڭ دەپ ئۇچۇر چىقىۋالدى. مەن سېتىۋالغان ئاچقۇچ نومۇرنى كىرگۈزسەم يەنىلا ئوخشاش. سوراي: مەن قانداق قىلسام ئۆزەم سېتىۋالغان ئاچقۇچ نومۇرىنى ئىشلىتەلەيمەن؟

يەنە بىر باش قېتىنچىلىقىم: Windows 8 نىڭ ئەپ دۇكىنىغا كىرگەندە يۇمتال چۈشۈرمەكچى بولساق ئېلخەت ئادرىسى بىلەن مەخپى نومۇر سورايدىكەن. بۇ يەرگە Windows 8 نى سېتىۋالغاندىكى ئىشلەتكەن ئېلخەت ساندۇقىنى توشتۇرۇمىزمۇ ياكى قانداق خەت ساندۇقى، قانداق مەخپى نومۇر بولسا بولامدۇ؟

Windows 8 نى سېتىۋالغان چاغدىكى ئېلخەت ساندۇقى مېنىڭ ئەمەس ماڭا ساتقۇچىنىڭ ئىدى، ئەگەر Windows 8 نى سېتىۋالغان چاغدىكى ئېلخەت ساندۇقىنى تولدۇرۇدىغان ئىش بولسا مېنىڭ ئىشلىتىۋاتقان ئېلخەت ساندۇقۇمنى ئەسلىدە سېتىۋالغان چاغدىكى ئېلخەت ساندۇقى بىلەن ئالماشتۇرغىلى بولامدۇ؟



جاۋاپ سۇئالدىن كىيىن بولسىمۇ ئالدى بىلەن رەخمەت دېگەچ تۇراي: كۆپتىن كۆپ رەخمەت!
