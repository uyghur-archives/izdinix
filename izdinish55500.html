32 بىتلىق Win8 نى 64 بىتلىق Win8.1 گە كۆتىرىش


(ئىزدىنىش مۇنبىرى- باچكا)



«كاش.......!» قىلغان ئاۋاز بىلەن، كونا كومپيۇتېرىمنىڭ توك مەنبەسى كۆيۈپتۇ.  چۇۋۇپ، بىخەتەر سىمنى ئالماشتۇرۇپ توك چاتسام يەنە «كاش...!». يېڭى توك مەنبە ساندۇقى سېتىۋېلىشنى ئويلاشمىدىم. شۇنداق قىلىپ، كومپيۇتېرىمنى يېڭىلاش ئارزۇيۇمغا تېزلا يېتىپ، ئەسلىدىكى Windows8 ھەقىقىي نەشىر ئاچقۇچى بىلەن قىلچە ئالاقىسى بولمىغان يېڭى ساندۇق ئېكرانغا چېتىلدى.


دۇكاندا ئۇنىڭغا 64 بىتلىق كۆزنەك 7 نى قاچىلاتتىم، ئەسلى بۇ بىر توغرا تاللاش ئىكەنتۇق..



شۇنىڭ بىلەن ئىككى ۋەزىپە چۈشتى: بىرى، ئاچقۇچنى يېڭى كومپيۇتېرغا قايتا ئاكىتىپلاش؛ يەنە بىرى، 32 بىتلىق سىستېما بىلەن خوشلىشىپ، 64 بىتلىق Windows 8.1 ئىشلىتىش.



ئەسلى، بۇ ئىشنىڭ قەدەم باسقۇچى توغرا بولسا، ئاددىي ۋە تېزلا (ئىككى كۈندە) تۈگەيدىغان ئىشكەنتۇق، يەنى:


1- قەدەم. 64 بىتلىق Win 7 دە تۇرۇپ، خەت ساندۇقنى ئېچىپ، چۈشۈرۈشنى بېسىپ، كونا كومپيۇتېرغا ئىشلەتكەن ئاچقۇچ بىلەن، Win 8 نى چۈشۈرۈپ قاچىلاش، بۇنىڭدا ئاپتوماتىك ھالدا 64 بىتلىق Win 8 چۈشۈپ قاچىلىنىدۇ. ئەمما ئاچقۇچ يېڭى كومپيۇتېرغا ئاكىتىپلانمايدۇ. ئەگەر، ئەسلى قاچىلىغىنىڭىز Win8 نىڭ 64 بىتلىق نەشىرى بولسا، 3- قەدەمدىن باشلاڭ.


Windows 7 دە تۇرۇپ، مىكروسوفت ئورگان بېتىدىكى 8 ۋە 8.1 نى چۈشۈرىدىغان ئىككى ئۇلانمىدىن، بىۋاستە 8.1 نى بېسىپ، چۈشۈرۈش پروگراممىسىغا 8 نىڭ ئاچقۇچىنى كىرگۈزسەك تونۇمايدىكەن. باشقىلارنىڭ بېشىغا قانداق كەلدى بىلمىدىم.



2- قەدەم. سىستېما خاسلىقى كۆزنىكىدىن،  ئاكتىپلاش ئۇلانمىسىنى چېكىپ، مۇلازىمەتكە تېلېفۇن قىلىش. ئېكراندا تۇنجى نومۇر 800 دىن باشلانغان، ئۇنىڭغا ئۇرسىڭىز قۇرۇق نومۇر چىقىدۇ، سىز 800 نى 400 گە ئۆزگەرتىپ، كېيىنكى خانىسىنى ئۆز ئەينى بېسىپ، مۇلازىمەتكە ئۇلاپ، قاتتىق دېتال بۇزۇلغانلىق بىلەن ئاچقۇچنى قايتا ئاكىتىپلىماقچى بولغانلىقىڭىزنى ئېيتىسىز. قارشى تەرەپ 9 خانىلىق نومۇرنى سورايدۇ، ئاندىن «قانچە كومپيۇتېرغا ئىشلىتىسىز؟» سۇئالىغا «بىرلا كومپيۇتېرغا ئىشلىتىمەن، بۇرۇنقىسىنى ئىشلەتمەيمەن» جاۋابىنى بېرىپ، قارشى تەرەپ سىزگە دەپ بېرىدىغان 9 خانە نومۇرنى خاتىرلەيسىز. ئۇنى ئاكتىپلاش كۆزنىكىگە كىرگۈزىسىز- دە، تامام، پۈتۈنلەي يېڭى قاتتىق دېتال بۇرۇنقى ئاچقۇچتا يېڭىدىن ئاكتىپلىنىدۇ. بۇ جەرياندا دىققەت قىلىدىغان ئىش: سىز خەنچىگە راۋان بولسىڭىز بولىدۇ؛ خاتىرە دەپتەرنى ئېچىپ، ئۇلار دەپ بېرىدىغان 9 گۇرۇپپا ساننى كۇنۇپكىدا ئۇرۇپ كىرگۈزۈڭ، قول تېز بولسا قەغەزگە يازسىڭىزمۇ بولىدۇ، لېكىن مەن يەنىلا كۇنۇپكىدا خاتىرە دەپتەر ئەپىگە خاتىرلىشىڭىزنى تەۋسىيە قىلىمەن.



3- قەدەم. win8  نى يېڭىلاش. ئەگەر تېخى يېڭىلانمىغان بولسا، ئەپ دۇكىندا Windows 8.1 گە يېڭىلاش ئۇچۇرى چىقمايدۇ!



4- قەدەم. ئەپ دۇكىنىنى ئېچىش. باش بەتتىلا، Windows 8.1  گە ھەقسىز يېڭىلاش بېتى كۆرۈنىدۇ. چۈشۈرۈپ قاچىلاشقا باشلىسىڭىز بولىدۇ. چۈشۈرۈپ بولغاندىن كېيىن، ئاپتوماتىك قايتا قوزغىلىشتىن بۇرۇن 10 نەچچە مىنۇت ۋاقىت بېرىپ، ساقلايدىغان نەرسە بار- يوقلىقىنى بىر تەرەپ قىلىشنى سورايدىكەن. قاچىلاش جەريانمۇ نەچچە سائەت ۋاقىتنى يەيدىكەن.  قاچىلاپ تۈگەيدىغان چاغدا، خەت ساندۇقىغا تىزىملىتىش باسقۇچى قانداق بولدىكىن، بىرەر سائەتكە يېقىن تىزىملاش ھالىتىدە تۇرۇپ ۋاقىت يىدى. «ھازىرقى ھېساۋات نامىنى ئىشلىتىش» كۇنۇپكىسى كۆرۈنگەندە بېسىپ داۋاملاشتۇردۇم. دەل ھازىر قاچىلاش تامام بولدى. Win+Pause كۇنۇپكىسى بىلەن، سىستېما خاسلىقىنى تەكشۈردۈم. «Windows 8.1 كەسپىي نۇسخىسى، 64 بىتلىق مەشغۇلات سىستېمسى، x64 رامكىسىدىكى بىر تەرەپ قىلغۇچ» ئۇچۇرلىرىنى كۆرۈپ، جەزىملەش تامام بولدى، يەنى،  64 بىتلىق Windows 8.1 نى قاچىلاش تامام.



.....


لېكىن، بۇرۇن 32 بىتلىق Windows 7 دە تۇرۇپ چۈشۈرۈپ ياسىغان قاچىلاش دېسكىامغا ئارزۇغا ئاساسەنلا ئىشىنىپ كېتىش سەۋەبىدىن، يەنى، قاچىلاش دېسكىسىنىڭ 64 بىتلىق ئەمەسلىكىنى بىلمەسلىك سەۋەبىدىن، يېڭى كومپيۇتېرغا، 64 بىتلىق Windows 7  تا تۇرۇپ قاچىلىغان Windows 8 سىستېمام 32 بىتلىق بولۇپ قالسا بولىدۇ.. «پەندىيات!...».  بۇنىڭ بىلەن، 64 بىتلىق Windows 8.1 گە ئېرىشىش مۇمكىن بولمىدى.



قولۇمدىكى64 بىتلىق Windwos7 قاچىلاش دېسكامدا كاشىلا باركەن، يەنى كومپيۇتېر قوزغاتقاندا، ئاۋال «FATAL ERROR....Press Any Key...» ئۇچۇرى چىقىدۇ، خالىغان كۇنۇپكا چەكسەم ئاندىن قوزغىلىدۇ، ئۇنىڭدا Windows 8 نى چۈشۈرۈپ بولغاندا، «خاتالىق كۆرۈلۈپ»، iso ياكى  usb ياساش مۇمكىن بولمىدى. ... ئەگەر،  32 بىتلىق ساق Windows7 قاچىلاش دېسكامنى ئىشلەتسەم، پىلان مەڭگۈ ئورۇنلانمايدۇ، يەنى، Windows 8.1 نىڭ 64 بىتلىقىغا مەڭگۈ ئېرىشەلمەيمەن.  بۇ جەرياندا ئىسراپ بولغان ۋاقىتنىڭ كۆپلۈكىنى دەيمۇ.. قىزىل بايراقتىن يەنە سېتىۋېلىپ كەلگەن ئىككى Windows7  دېسكىسىمۇ نورمال قاچىلانمىدى... توردىن چۈشۈرگەن ئىككى Windows 7 ISO نى بارماق دېسكىغا ئۇيۇپ (مەخسۇس قۇرالدا ياسىلىدۇ) قاچىلىساممۇ، قاچىلاش داۋاملاشمىدى.. ئىسراپ كېچىلەر...


ئاخىر، شۇ نورمال قاچىلانمىغان سىستېما بار بارماق دېسكدىكى Ghost پروگراممىسى بىلەن Windows7.gho نى يەشكەندىن كېيىن، كومپيۇتېر قوزغالغاندا «FATAL ERROR..» چىقمىدى. ئۇف.. ئاخىر ئىشلار يۈرۈشۈپ، «64 بىتلىق Win7 دە تۇرۇپ 64 بىتلىق Win8 نى قاچىلاش، يېڭىلاش، 64 بىتلىق Win8 دە تۇرۇپ 64 بىتلىق Win8.1 نى قاچىلاش» ئوڭۇشلۇق ئاخىرلاشتى.



.....


64 بىتلىق مەشغۇلات سىستېمىسى بىلەن، 32 بىتلىق مەشغۇلات سىستېمىسىنىڭ پەرقىنى چۈشەندۈرسەك، مەسىلەن، سىز 4 گېگا بايىتتىن چوڭ سىغىمدىكى ئىچكى ساقلىغۇچ ئىشلەتكەن بولسىڭىز، 32 بىتلىق سىستېمىدا، ئىچكى ساقلىغۇچنىڭ 4GB دىن ئاشقان قىسىمى تامغا سىزىلغان ئىشكاپقا ئوخشاش ئىشلەتكىلى بولمايدۇ. شۇڭا، كۆپچىلىكنىڭ، ئەگەر قاتتىق دېتال 64 خانىلىق بولسىلا، چوقۇم 64 خانىلىق سىستېما قاچىلاپ ئىشلىتىشىنى كۈچلۈك تەۋسىيە قىلىمەن، مەيلى سىزنىڭ ئىچكى ساقلىغۇچىڭىز 2GB بولغان تەقدىردىمۇ.. چۈنكى، 64 بىتلىقنىڭ ئارتۇقچىلىقى يالغۇز ئىچكى ساقلىغۇچ ئىشلىتىش بىلەنلا تۈگەپ قالمايدۇ.. غول يولدا 32 رەت ماڭغانغا قارىغاندا، 64 رەت ماڭسا، كومپيۇتېرنىڭ خىزمەت ئۈنۈمى بىر ھەسسسىگە يېقىن ئاشقان بولىدۇ..

كۆك كەپتەر ئىنىمنىڭ يارىدىمىدە كۆزنەك8دىن 3 يۈرۈش  سېتىۋىلىپ، 32 ۋە 64لىكلىرىنى ھەم ئۇلارنىڭ ئۇيغۇرچىلاشتۇرغۇچلىرىنى توردىن  چۈشۈرۈپ نۇرتەخسىگە ئويىۋالغان ئىدىم، سەپلىمىسى تۆۋەنرەك كومپيۇتېرغا قاچىلاش ئۈچۈن 64لىكنى ئويغان دىسكانى سالسام، كومپيۇتېرىمغا 64لىكنىڭ ماس كەلمەيدىغانلىق ئۇچۇرى چىقتى، 32لىكنى سالسام نورمال قاچىلىدى، ھەپتىدىن كېيىن سۆرىيەلمەي ئىنجىقلىغىلى تۇردى. يېڭىدىن خاتىرە كومپيۇتېر سېتىۋىلىشقا توغرا كەلدى، ئۇنىڭغا ئاۋۇ كومپيۇتېر سۆرىيەلمىگەن سېستىمىنى يۆتكەپ ئىشلىتىش ئىستىكىدە، ئالدى بىلەن يېڭى كومپيۇتېرنىڭ قاتتىق دىسكىسىنى كونا ئۇسۇلۇم بويىچە رايۇنلارغا ئايرىدىم، ئۇنىڭدىن كېيىن كۆزنەك8نىڭ 32لىكىنى قاچىلاشنى ياكى  64لىكىنى قاچىلاشنى بىكىتەلمەي، تەۋەككۇل قىلىپ، 64لىكى ئويالغان نۇردىسكىنى پۈتۈنلەي ئاق ھالەتتىكى كومپيۇتېرنىڭ نۇردىسكا قوزغاتقۇچىسىغا سېلىپ قاچىلاشقا باشلىدىم،  ئاچقۇچ  كۆزنىكىگە كونا كومپيۇتېرغا قاچىلىغاندىكى پىچىرلارنى كىرگۈزسەم نورمال قاچىلاپ كەتتى، تولۇق قاچىلاپ بولغاندىن كېيىن ئاكىتىپلاشتا توردارشلارنىڭ  http://www.izdinix.com/forum.php?mod=viewthread&tid=55027 دىكى يارىدىمى بىلەن تېلېفۇن ئارقىلىق ئاكىتىپلىۋالدىم، تىل جەھەتتە بىرئاز قىيىنچىلىق تارتتىم،  قايسى سانلارنى سورىغانلىقىنى باشتا بىلەلمىدىم، تېلېفۇننى ئالغان يولداش خىلى سەۋىرچانكەن،  ئونىڭ سورىغان سىپىرلىرىنى تېپىپ بىر-بىرلەپ دەپ بەردىم، كېيىن ئۇ يولداش بىرگۇرۇپا سانلارنى دەپ بەردى، ئارىلىقتا بىر كاتەكتىكى ساندا خاتالىق كۆرۈلىۋىدى، مەلۇم قىلسام شۇ كاتەكتىكى رەقەملەرنى قايتا دەپ بەردى، تولۇق كىرگۈزۈپ بولغاندىن كېيىن مۇۋەپپىقىيەتلىك بولدى(رەھمىتىمنى ئېيتىپ تېلېفۇننى قويىۋەتتىم)، ئەمما ئۇيغۇرچىلاشتۇرۇش بۇيرىقىنى بەرسەم، ئىجرا بولمىدى، بۇرۇن چۈشۈرىۋالغان تىل يامىقىنى قاچىلاپ ئۇيغۇرچىلاشتۇرىۋالدىم. ئۇيغۇرچىغا شۇنداق كۆنۈپلا كەتتىم، ئوففىس 2010نىڭ بۇيرۇق ئىقتىدارلىرىنى تاپالماي، كوچانىڭ سوپىنى ياخشى دېگەندەك، ئوففىس 2003نىمۇ قاچىلاپ، ئۇيغۇرچە  كىرگۈزگۈچكە ئۇيغۇرچىنى ئەرەپچىنىڭ قولتىقىغا كىرگۈزۈپ ئىشلىتىۋاتىمەن، مۇشۇ ئىزدىنىشقا ئىنكاس يازغاندا نۇر بەلگىنىڭ پات-پات ئۇيەر-بۇيەرگە چاچراپ كېتىپ، باشقا يەرلەردىكى سۆزلەرنى بۇزىۋىتىدىغان خۇيىنى ھېساپقا ئالمىغاندا بۇ كۆزنەك 8نىڭ ئۇيغۇرچىسىدا ھىچقانداق مەسىلە كۆرۈلمىدى.
