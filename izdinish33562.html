HIV يۇقۇمدارلىرى سانى 33 مىليۇن ئىكەنلىكى مۆلچە


ئەيدىز ( HIV ) يۇقۇمدارلىرى سانى 33 مىليۇن ئىكەنلىكى مۆلچەرلەنمەكتە

<< يۈكسەل AIDS تور تەشۋىقات گۇرۇپپىسى >> دىن كروران تەييارلىدى


دۇنيا بويىچە ئەيدىز ( HIV ) دىن يۇقۇملانغان كىشىلەرنىڭ سانى 7 مىليۇن كىشى ئارتۇق مۆلچەرلىنىپ قالغان

بىرلەشكەن دۆلەتلەر تەشكىلاتى ( UN ) دۇنيا بويىچە ئەيدىز ( HIV ) بىلەن يۇقۇملانغانلارنىڭ مۆلچەر سانىىنى40 مىليوندىن33 مىليونغا چۈشۈردى. سەيشەنبە كۈنى ئېلان قىلنغان بىر خەۋەردە دېيىلىشىچە، بىرلەشكەن دۆلەتلەر تەشكىلاتى ( UN )ھىندىستاندىكى ۋىروسلانغۇچىلارنىڭ كۆپ ساندا ئازلىغانلىقى سەۋەبىدىن مۆلچەرگە ئۆزگەرتىش كىرگۈزگەن. مەزكۇر تەشكىلاتنىڭ ئېيتىشىچە، ئەيدىز ( HIV ) ئاتلىق ھازىرچە داۋالاش مومكىن بولمايدىغان ۋىرۇستىن يۇقۇملانغۇچىلارنىڭ سانىيۇقىرى مۆلچەرلىنىپ قالغان. ئېنىق بولغان سانلىق مەلۇماتلارغا ئاساسلانغاندا، يۇقۇم ئەھۋالى بىزنى قورقۇتقان ھېلىقىدەك ئەمەس ئىكەن.


خەۋەردە ئېيتىلىشىچە، « مۆلچەر ساننىڭ بۇنداق ئازلىشىدىكى بىردىنبىر چوڭ سەۋەب شۇكى،ھىندىستاندىكى ئەيدىز ( HIV )ۋىروسىنى ئېلىپ يۇرگۈچىلەرگە ئېلىپ بېرىلغان ئاكتىپ ياردەم بېرىش پائالىيىتنىڭ نەتىجىسىدە بۇ دۆلەتتىكى مۆلچەر سان زور دەرىجىدە ئازلىغان »
ئىلگىرى ھىندىستاندا 5.7 مىللىيون ئادەم يۇقۇملانغان دىيىلگەن بولسا، ھازىر بىرلەشكەن دۆلەتلەر تەشكىلاتى پەقەت 2.5 مىلىيون يۇقۇمدار بار دەپ مۆلچەرلىمەكتە.
يۇقۇمدارلار سانى يەنىلا ناھايىتى كۆپ
دۇنيادىكى ئەيدىز ( HIV ) يۇقۇمدارلىرىغا ئائىت مۆلچەرنىڭ 7 مىليۇن كىشى ئارتۇق مۆلچەرلىنىپ قېلىنغان بولسىمۇ، سانلىق ئۇچۇرلاردىن كۆرە، ئەيدىز ( HIV ) يۇقۇمى يەنىلا ناھايىتى ئېغىر ، يۇقۇمدارلار سانى يەنىلا ناھايىتى كۆپ ، ئەيدىز ( HIV ) نىڭ ئالدىنى ئېلىش ۋە ئۇنى تېزگىنلەش خىزمىتىنى يەنىمۇ جىددىي ئىشلەش كېرەك، دەيدۇ بىرلەشكەن دۆلەتلەر تەشكىلاتى ئەيدىز - پىلان مەھكىمىسى ( UN AIDS ) ئەمەلدارى.
« ئۆزگەرتىش كىرگۈزۈلگەن بۇ سانلىق مەلۇماتلار دىن بىز ئەيدىز ( HIV ) نىڭ يۇقۇم ئەھۋالىنى تېخىمۇ ئېنىق چۈشىنىپ يېتەلەيمىز. كۆز ئالدىمىزدا خىرىس ۋە پۇرسەت تەڭ مەۋجۇد بولۇپ تۇرماقتا، دەيدۇ UN AIDS نىڭ ئىجرائىيە مەسئۇلىPeter Piot . - «ئېنىقكى،بىز بەدىلىمىزنىڭ(سالغان مەبلىغىمىزنىڭ) نەتىجىسىنى كۆرۈشكە باشلىدۇق. بىراق ھەر كۈنى يېڭىدىن يۇقۇملىنىۋاتقان 6800 دىن كۆپرەك يېڭى يۇقۇمدارلار، ۋە ھەر كۈنى ئەيدىز سىەۋەبىدىن ئۈلۈپ كېتىۋاتقان 5700 دىن كۆپرەك ئادەمنى نەزەرگە ئالغاندا، بىزئەيدىز ( HIV ) پۈتۈن دۇنياغا يامرىشىنىڭ ئالدىنى ئېلىش ئۈچۈن يەنىمۇ زور كۈچ بىلەن تىرشىشىمىز لازىم».
UN AIDS نىڭ ئېيتىشىچە بۇ يىلسەھرايى كەبىرنىڭ جەنۇبىدىكى ئافىرىقىدا 1.7 مىليۇن ئادەم ئەيدىز ( HIV ) دىن يۇقۇملانغان. بۇ2001 - يىلىغا قارىغاندا كۆرۈنەرلىك ئازلىغان، بىراق ئافرىقا يەنىلا ئەيدىز ( HIV ) يۇقۇم ئەھۋالى ئېغىر بولغان رايون بولۇپ،22.5 مىليون ئەيدىز ( HIV ) يۇقۇمدارى مۇشۇ يەردە. «بۇ رايوندىكى 8 دۆلەت ھازىر دۇنيادىكى يېڭىدىن ئەيدىز ( HIV ) بىلەن يۇقۇملانغۇچىلار ۋە ئۆلۈپ كېتىۋاتقانلارنىڭ 3 دىن 1 نى ئىگەللەيدۇ دەپ قارالماقتا » دەيدۇ، UN AIDSبىر بايانىدا.
ئالدىنى ئېلىش مۇۋاپپەقىيەتلىك بولغان
 باياندا مۆلچەرنىڭ چۈشۈشىگە مۇنداق ئىككى سەۋەب كۆرسىتىلدى. بىرى ئېنىق - توغرا بولغان سانلىق مەلۇمات، يەنە بىرى يېڭى يۇقۇمدارلارنىڭ ھەقىقەتەن ئازلىغانلىقى. ھەر يىلى يېڭىدىنئەيدىز ( HIV )بىلەن يۇقۇملانغۇچىلارنىڭ سانى 1990 يىلى 3مىليون بۇلۇپ تارىختىكى ئەڭ يوقىرى سەۋىيەگە يەتكەن بولسا، بۇ سان 2007- يىلى 2.5 مىلىيون دەپ مۆلچەرلەندى.
خەۋەردە ئېيتىلىشىچە، « بۇلار يۇقۇمدارلاردىكى تەبىئي يۇزلىنىشنى كۆرسىتىپ بېرىدۇ، شۇنداقلا ئەيدىز ( HIV ) دىن مۇداپىئەلىنىش تىرىشچانلىقىمىزنىڭ ئۈنۈمىنى ئەكىس ئەتتۈرىدۇ. 2006 -ۋە 2007 -يىلدىكى پۈتۈنلەي ئوخشاش بولمىغان سانلىق مەلۇمات مۆلچەرلىنىشتىكى ئۆزگىرىش مۇنۇ 6 دۆلەتتىن كەلگەن،ئۇلار:ئانگولا،ھىندىستان،كېنىيە،موزامبېك،نگېرىيە ۋە زىمبابۇۋى».
يەنە بىر دىققەت قىلىشقا تېگىشلىك نۇقتا شۇكى،UN يەنە بىر مۆلچەرىنى ئۆزگەرتتى . ئۇ بولسىمۇ ناۋادا ئەيدىز ( HIV ) بىمارى داۋالانمىسا 9 يىلغىچە ئۈلۈپ كېتىدۇ ، دىگەن مۆلچەرىنى 11 يىلغىچە شۇنداق بولۇشى مۇمكىن، دەپ ئۆزگەرتتى.
