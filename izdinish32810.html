كىيىنكى ئەۋلات مەشغۇلات سېستىمىسى Windows "7 " توغرى

كىيىنكى ئەۋلات مەشغۇلات سېستىمىسى Windows "7" توغرىسىدا سۇئال جاۋابلار
Windows Vista ئاللىقاچان بىزنىڭ يېنىمىزغا كىلىپ بولدى ، شۇنىڭ بىلەن كىشىلەرنىڭ نەزىرى كىيىنكى ئەۋلات مەشغۇلات سېستىمىسىغا ئاغدۇرۇلماقتا ، ئۇ مەشغۇلات سېستىمىسى دەل بىز دەۋەتقان Windows “7” (مايكروسوفىت بۇنى  Vienna ياكى “Windows Seven” دەپ ئاتىماقتا ). ھازىر بۇ توغرۇلۇق دەلىلەش ئۇچۇرلىرى كامچىل بولسىمۇ ، تۆۋەندىكى ئىككى سەۋەب بىزنىڭ بۇ يېڭى تۈرنى تالاش-تارتىش قىلىشىمىزغا سەۋەب بولىدۇ:
Q: مايكروسوفىت ھازىر Windows Vista نىڭ ئىزباسارىنى ئىشلەۋاتامدۇ؟
A: شۇنداق . كىيىنكى ئەۋلات مەشغۇلات سېستىمىسى Windows ئەسلىدە “Blackcomb” دەپ ئاتىلاتتى، 2006.يىلىنىڭ بېشىدا مايكروسوفىت شىركىتى ئۇنىڭغا “Windows Vienna”دەپ ئىسىم قويدى ، يېقىندىن بۇيان "Windows Seven" ياكى "Windows 7" دەپ ئاتاۋاتىدۇ.
Q: نىمە ئۈچۈن Windows 7 دەپ ئاتىلىدۇ؟
A: چۈنكى Windows Vista ئەمەلىيەتتە Windows 6.0 دىن ئبارەت، شۇڭا كىينكىسى Windows 7 يەنى Windows نىڭ 7.0 نەشىرىدىن ئىبارەت.
Q: Windows 7 ئەڭ ئاخىرقى ئاتىلىشىمۇ؟
A: ياق . خۇددى Vienna غا ئوخشاش Windows 7 مۇ كود نامى ، ھەققىي تارقىتىلغاندا يېڭى ئىسىم قويۇلىدۇ.
Q: ئاڭلاشلارغا قارىغاندا Windows Vista ئەڭ ئاخىرقى مۇھىم بولغان مايكروسوفىت مەشغۇلات سېستىمىسىى ، شۇنداقمۇ؟
A: ياق. Windows نى ئاساس قىلغان PC لار كەلگۈسىدە بىزنىڭ سىفىرلىق تۇرمۇشىمىز بىلەن بىرلىشىپ كىتىدۇ، يېقىندا ، مايكروسوفىتنىڭ يۇقۇرى دەرىجىلىك ئەمەلدارلىىرى Windows نىڭ كۆپ قىسىم جايلىرىغا كۆپ مىقداردا ئۆزگەرتىش ئېلىپ باردىغانلىقنى كۆرسەتتى. مەسلەن : ئاۋاز سىزىش ۋە ساقلاش قاتارلىقلار...
Q: ئۇنداقتا ، Windows 7 كەلگۈسىدىكى مۇھىم Windows نەشىرىمۇ؟
A:شۇنداق . Windows Vista بولسا بىر مۇھىم نەشىرى، Widnows 7 مۇ بىر مۇھىم يېڭىلىنىش بولىدۇ، ئادەتتە مايىكروسوفىت شىركىتى بىر قانچە نەشىردە بىر قېتىم مۇھىم نەشىرنى تارقىتىدۇ، شۇڭا Vista بىلەن Windows 7 ئارسىدا يەنە بىر قېتىم كىچىك تىپتىكى يېڭىلاش بار.
Q: Windows 7 قاچان تارقىتىلىدۇ؟
A: مايكىروسوفىت (Microsoft) نىڭ پىلانى بويىچە بولغاندا ، Windows 7 نۇسخىسى 2010-يىلى تاقىتىلىدۇ.
Q: Windows 7 دە قانداق يېڭىلىقلار بار؟
A: مايكىروسوفىت (Microsoft) ھازىرغىچە تېخى Windows 7 نىڭ ئىقتىدارلىرىنى ئاشكارلىمىدى، ئۇلار ھازىر كىيىنكى Windows قا قانداق تەڭشەش ئېلىپ بېرىشنى قارار قىلىۋاتىدۇ.
بىراق بىزيەنە بىر قىسىم Windows 7 گە مۇناسىۋەتلىك ئۇچۇرلارنى بىلىمىز: يېڭى نۇسخىدىكى Windows نىڭ مەنبە باشقۇرغۇچىسىنى (Explorer)  ئېچىش ، Office 2007Ribbon UI ئېچىش گورۇپپىسىغا تاپشۇرۇلغان ، چوقۇم Windows Server 2008 دا قوللىنىلغان "Hypervisor" (Windows نى مەھۇملاشتۇرۇش ) تېخنىكىسى قوللىنىلغان ، يەنە چوقۇم WinFS( Windows Future Storage ، Windows كەلگۈسى ساقلاش)—بۇ تېخنىكا ئەسلىدە Vista دا تارقىتىش پىلانلانغان ئىدى.
Q: Windows 7 نىڭمۇ 32 ۋە 64 بىتلىق ئىككى خىل نەشىرى بارمۇ؟
A: شۇنداق ئىككى خىل نۇسخسى تارقىتىلىدۇ.
Q: مايكىروسوفىت (Microsoft)  Vista  بىلەن Windows 7 ئارسىدا Windowsنىڭ نەشىرىنى يېڭىلامدۇ؟
A: شۇنداق . Windows Vista SP1 (كود ئىسمى : Fiji) بەلكىم 2008-يىلنىڭ بېشىدا Windows Server 2008 بىلەن بىرلا ۋاقىتتا تارقىتىشى مۇمكىن.
