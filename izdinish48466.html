رەسىم پارچىلاش پروگراممىسى

يېقىندىن بۇيان ئىزدىنىشقا ئېلان ئېلىش ئىشلىرىنىڭ ئېلىپ بېرىلىشىغا ئەگىشىپ بەزى مەسىلىلەرمۇ تەڭ پەيدا بولدى. يەنى ئېلان بەرگۈچىنىڭ تور بېتى بولمىغان ئەھۋال ئاستىدا بىز ئېلان بېتىدىن بىرنى ياساپ بېرىشىمىز كېرەك بولدى، يەنە بەزى ئەھۋاللاردا ئېلان بەرگۈچى ئۆزى html بېتىنى ياساپ بىزگە بېرىدىغان ھالەتمۇ مەۋجۇت. مانا مۇشۇ بەتلەر ئاساسەن چوڭ رەسىملەرنى ئۆز ئىچىگە ئالغان بولىدۇ. بۇ چوڭ رەسىملەرنى ئۆز ئىچىگە ئالغان بەتلەرنى ئاچقاندا رەسىم چوڭ بولۇش سەۋەپلىك ئېچىلىش سۈرئىتى ئاستا بولىدۇ. شۇ سەۋەپ ۋەجىدىن مەن تۆۋەندىكى رەسىم پارچىلاش پروگراممىسىنى تۈزۈپ قويغان، بۇ پروگراممىنى ئىزدىنىش شىركىتى ئېلان بۆلىمىدىكىلەر ئىشلىتىۋاتىدۇ. باشقا تور بەت ياسايدىغان ۋە تور بېتى بارلارمۇ بۇ خىل قورالغا ھاجىتى چۈشىشى مۇمكىن، شۇڭا بۇ پروگراممىنىڭ چۈشۈرۈش ئادرىسىنى تەمىنلىدىم. بەك كۆپ قۇرۇق گەپ قىلىپ كەتتىم، تۆۋەندە ئىشلىتىش ئۇسۇلىنى تونۇشتۇراي.


بۇ پروگراممىنى قاچلاشتىن بۇرۇن Microsoft .NET Framework 3.5 SP1 چۈشۈرۈپ قاچىلايسىز( بۇرۇن قاچىلاپ بولغان بولسىڭىز قايتا قاچىلاش ھاجەتسىز).


چۈشۈرۈش ئادرىسى نى بېسىپ بۇ پروگراممىنى چۈشۈرگەندىن كېيىن، يېشىپ قاچىلايسىز. پروگراممىنى قوزغاتقاندا تۆۋەندىكىدەك كۆزنەك چىقىدۇ.



رەسىم ئادرىسى دېگەن قۇرنىڭ سول تەرىپىدىكى ئېچىش دېگەن كۇنۇپكىنى بېسىپ تۆۋەندىكى ئېچىش كۆزنىكىنى ئاچىسىز:



بۇ يەردىن لازىملىق رەسىمنى تاللاپ ئېچىش كۇنۇپكىسىنى باسسىڭىز تۆۋەندىكىدەك ئۇچۇرلار كۆرىنىدۇ:



بۇ كۆزنەكنىڭ ئەسلى رەسىم ئۇچۇرلىرى دېگەن يەردە سىز باياتىن ئاچقان رەسىمىڭىزنىڭ كەڭلىكى ۋە ئىگىزلىك ئۇچۇرلىرى كۆرىنىدۇ.


پارچىلاش پارامېتىرى دېگەن تۈرنىڭ ئىچىدىكى كەڭلىك، ئىگىزلىك دېگىنىمىز سىزنىڭ پارچىلىغاندا ھاسىل بولىدىغان رەسىملەرنىڭ كەڭلىك ئىگىزلىكى.


HTML دىكى رەسىم مۇندەرىجىسى دېگەن ئورۇنغا پارچىلانغان رەسىملەرنى قويىدىغان مۇتلەق ئادرىسنى يازسىڭىزمۇ ياكى مەلۇم بىر ھۆججەت قىسقۇچنىڭ نامىنى يازسىڭىزمۇ بولىدۇ( شەرتى: پارچىلانغان رەسىملەرنى شۇ ھۆججەت قىسقۇچقا ياكى شۇ مۇتلەق ئادرىسقا يوللىشىڭىز كېرەك!).


يۇقىرىقى پارامېتىرلارنى كىرگۈزۈپ بولۇپ، پارچىلاش دېگەن كۇنۇپكىنى باسسىڭىز ئاستى تەرەپتىكى ھاسىل بولغان HTML كودى دېگەن ئورۇنغا HTML كودى ھاسىل بولىدۇ. دېمەك رەسىم پارچىلاش ئاخىرلاشتى.



ئەمدى كۆچۈرۈش دېگەن كۇنۇپكىنى بېسىپ ھاسىل بولغان HTML كودىنى كۆچۈرۈپ ئالىسىز.


پارچىلانغان رەسىملەر ئەسلى رەسىم قايسى ھۆججەت قىسقۇچتا بولسا شۇ ئورۇنغا ھاسىل بولىدۇ:



ئەمدى سىز HTML تەھرىرلىگۈچتىن بىرنى تېپىپ، يېڭى بەتتىن بىرنى ھاسىل قىلسىڭىز:



بۇ HTML كودى ئىچىدىكى <body> بىلەن </body> نىڭ ئارىسىغا باياتىن كۆچۈرىۋالغان HTML كودىنى چاپلايسىز. شۇنىڭ بىلەن كىچىك رەسىملەردىن قۇراشتۇرۇلغان بىر HTML بېتى تەييار بولغان بولىدۇ.



مۇشۇنچىلىك كوتۇلدىسام بولا.
