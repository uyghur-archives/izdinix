Windows8 دە net 2.0. پروگرامملىرىنى ئىجرا قىلدۇرۇش

بۇ يازمىنى ئاخىرىدا   heqteyur تەھرىرلىگەن. ۋاقتى2014-10-20 09:29

ئەسسالامۇ ئەلەيكۇم:

ھەممىمىزگە مەلۇم . .net 2.0. دە يېزىلغان پروگراممىلارنى ئىجرا قىلىش ئۈچۈن چوقۇم.net 2.0 مۇھىتىنى قاچىلاش كېرەك، يەنى Windows XP دا .net 2.0 مۇھىتى بولمىغاچ ئادەتتە چوقۇم ئىشلەتكۈچى ئۆزى ئايرىم قاچىلىشى كېرەك، ئەمما Windows 7 دە بولسا .net 3.5 نەشرى سىستېما بىلەن بىرگە قاچىلانغان ھالەتتە بولغاچقا كەڭ پروگراممېرلار ۋە ئىشلەتكۈچىلار ئۈچۈن كۆپ قولايلىقلارنى ئېلىپ كەلگەن ئىدى، ئەپسۇس بۇ ئەھۋال Windows 8/8.1 دىن باشلاپ ئۆزگىرىپ قالدى، يەنى Windows 8/8.1 دە .net نىڭ 4.0 بولغاچقا 2.0 دە يېزىلغان پروگراممىلارنى ئىجرا قىلىش ئۈچۈن 2.0 مۇھىتىنى قاچىلاش كېرەك بولدى، يەنە كېلىپ Windows 8/8.1 مۇھىتىغا 2.0 مۇھىتىنى قاچىلاش ناھايىتىمۇ ئاۋارىچىلىق بولۇپ ئىشلەتكۈچىلارگە نۇرغۇن قولايسىزلىقلارنى ئېلىپ كەلدى، چۈنكى.net مۇھىتىدا يېزىلغان نۇرغۇنلىغان قوللىنىشچان يۇمشاق دېتاللار 2.0مۇھىتىدا يېزىلغان بولغاچقاWindows 8/8.1 مۇھىتىدا بۇلارنى ئىشلىتىش بىر باش ئاغرىقى بولدى.

گەرچە يۇقۇرى نەشرىدىكى مۇھىت تۆۋەن نەشرىدىكى مۇھىتتا تۈزۈلگەن پروگراممىنى قوللاش ناھايىتى ئەقەللى بىر ئىش بولسىسمۇ 4.0 نەشرىگە كەلگەندە ئەھۋالدا ئازىراق ئۆزگىرىش بولغان، يەنى 2.0 دە يېزىلغان پروگراممىلار 4.0 مۇھىتىدا بىۋاستە ئىجرا بولماي 2.0 مۇھىتىنى ھازىرلىشىمىزنى تەلەپ قىلىدۇ.

يۇقۇرىدىكى بىۋاستە سۆزگە دىققەت قىلغان بولسىڭىز شۇنى ھېس قىلىسزكى بۇنىڭ بەزى ۋاستىلەر ئارقىلىق ئەمەلگە ئاشۇرۇش يوللىرىنىڭ بارلىقىنى، تۆۋەندە شۇ ۋاستىلىك ئۇسۇلنى تونۇشتۇرۇپ ئۆتەي:

سىزنىڭ ئىجرا قىلدۇرماقچى بولغان 2.0 مۇھىت نەشرىدىكى پروگراممىڭىزنى نامىنى TestProgram.exe دەپ تۇرايلى.

1،ئۇنداقتا سىز ئاشۇ پروگراممىڭىز تۇرغان مۇندەرىجە ئىچىگە TestProgram.exe.config دېگەن نامدا بىردا نە تېكىست ھۆججىتى قۇرۇڭ.(يەنى ھۆججەتنىڭ كېڭەيتىلگەن نامى config بولىدۇ، txt بولۇپ قالسا بولمايدۇ)

2، ئاندىن يۇقۇرىدا قۇرغان ھۆججەتنىڭ ئىچىگە تۆۋەندىكى مەزمۇننى چاپلايسىز. (يازسىڭىزمۇ مەيلى، ھى ھى)


   
   

3، يازغان مەزمۇننى ساقلاپ چېكىنىپ، TestProgram.exe دېگەن پروگراممىنى قوش چېكىپ قوزغاتسىڭىز نورمال قوزغىلىدۇ.



يۇقۇردىكى مەزمۇننى مەنبەسىنى ئەسكەرتىپ خالىغانچە كۆچۈرۈپ ئىشلەتسىڭىز بولىدۇ. كۆكلەن NET. پاراڭجايى بلوگىدىن ئېلىندى.
