IE10 ۋە HTML5 نى قانچىلىك چۈشىنىسىز





دېسەم دىمىسەم سىز بۇ چاغقىچە HTML5 نى ئاڭلاپ باقتىڭىز، چۈنكى ھازىر ھەممە يەردە مۇشۇ ئوقەتنىڭ
تاتلىق گېپى، يەنە Internet Explorer10 ۋە Windows8نىڭمۇ قۇلىقىنى
قىززىتىپ، ئازتۇلا سىلاپ باققاچ سىناپ باقتىڭىزغۇ دەيمەن. ئەمىسە بۇ يېڭى ئوقەتلەر
بىزگە نېمىلەرنى ئەكىلىدۇ؟ Internet Explorer10 ۋە Windows8مىكروسوفت شىركىتىنىڭ
تارقىتىلىش ئالدىدا تۇرغان مەشغۇلات سىستېمىسى ۋە تور كۆرگۈسى، ئۇنىڭغا WP8 دە گەۋدىلىك بولغان Modern UI (يەنى بىز داۋاملىق دەيدىغان Metro ئۇسلۇبى)
قوللىنىلغان. ئۇنىڭ ئۈستىگە پات ئارىدا تارقىتىلىدىغان WP8 سىستېمىسى  HTML5 ياغدەك قوللاپ قويىدۇ. شۇڭلاشقىمۇ ئۇنىڭدا يۇمشاق دېتال ئېچىش
ئىنتايىن بەك قولايلىق.


Internet Explorer10نى 9Internet
Explorer غا سېلىشتۇرساق، ئۇنىڭ HTML5 ئالاھىدىلىكلىرىنى دەھشەت قوللاپ قويىدۇ. مەسىلەن: Application
cache, Css 3D transform, web socket قاتارلىقلار. قاتتىق دېتالنى تېزلەشتۈرۈش، سۈرەت ئىقتىدارىمۇ
ناھايىتى ياخشى، شۇڭا ئۇ HTML5 تور بەتلىرىنى ئىنتايىن راۋان قوللاپ قويىدۇ. ناۋادا سىزنىڭ تور
بېتىڭىز Internet Explorer10 بىلەن سىغىشالىسا، ئۇنداقتا باشقا ئالى دەرىجىلىك تور
كۆرگۈلەردىمۇ بەك راۋان يۈرۈشەلەيدۇ.


HTML5 بولسا 1999-يىلى بېكىتىلگەن 4.01HTML ۋە 1.0HTML ئۆلچىمىنىڭ ئورنىنى ئالغان يېڭى ئۆلچەم. HTML5 نىڭ ئىككى چوڭ ئالاھىدىلىكى بار، Web تور بېتىنىڭ ئىپادىلەش ئىقتىدارى ۋە يەرلىك سانداننى قوشۇش قاتارلىقWeb قوللىنىشچان ئىقتىدارلىرى تېخىمۇ كۈچەيتىلگەن. HTML5 نىڭ تۇنجى نۇسخىسى 2008-يىلى 1-ئاينىڭ 22-كۈنى ئېلان قىلىنغان،
نۆۋەتتە HTML5 تېخىمۇ
مۇكەممەللىشىش باسقۇچىدا تۇرسىمۇ، لېكىن كۆپ ساندىكى تور كۆرگۈلەر ئاللىقاچان HTML5 نىڭ بەزى ئىقتىدارلىرىنى قوللاشنى ئىشقا ئاشۇرۇپ بولدى.


HTML5 ئىشلەتكۈچىلەر كۆرۈپ بېقىش ئىقتىدارىنى يۇقىرى كۆتۈرۈش
ئىقتىدارىنى كۈچەيتتى: Canvas ئېلمىنتىدا ئاچقۇچىلار بىر ئالاھىدە رايوندا تۈرلۈك مۇرەككەپ
شەكىللەرنى سىزالايدۇ، ھەمدە Javascript ئارقىلىق سىزغان شەكىلنىڭ ھەرىكەت ئۈنۈمىنى كونترول قىلالايدۇ. ئىشلەتكۈچىلەر
كۆچمە ئېغىز ۋە PC ئېغىزى ئارقىلىق بەلگىلىك كۆرۈم تۇيغۇسىدىن بەھرىمەن بۇلالايدۇ؛
قېنى ئەمىسە سىز بۇرۇن ئىشلىتىپ باققان بىرە تور پروگراممىسى ئارقىلىق، توردا
رەسىم بىر تەرەپ قىلىدىغان پروگرامما ئارقىلىق ئاددىيغىنە رەسىم بىر تەرەپ قىلىپ
بېقىڭ، لېكىن ئۇنى ھەرگىز Photoshop نىڭ ئورنىغا دەسسەتمەڭ، چۈنكى ئۇنىڭ ئىنتېرەكتىۋلىقى،
ماسلىشىشچانلىقى ۋە ئىقتىدارچانلىقى يەرلىك پروگراممىغا يەتمەيدۇ. لېكىن غەم
قىلماڭ، HTML5 بۇ مەسىلىنى ناھايىتى ئومدان ھەل قىلغان. ئىنتېرەكتىۋلىققا نىسبەتەن، ئۇ قوللايدىغانmenu ئېلمىنتى يەرلىكتىكى تىزىملىك ئىستونى، قۇرال ئىستونى، تىزىملىك
ئىستونى قاتارلىق قوللىنىشچان بۆلەكلەرگە تەقلىد قىلىنغان. Drag-and-drop ئىقتىدارى «ھۆججەتنى سۆرەپ ئەخلەتخانىغا ئەكىرىش» مەشغۇلاتىغا
تەقلىد قىلىغان. ماسلىشىشچانلىقىغا نىسبەتەن، Programmable HTTP
Cache and Serving ئىقتىدارى web
app مەزمۇن ئۆزگىرىشلىرىنى
تور كۆرگۈ ساقلىغۇچىغا ئوقۇيدۇ، app نىڭ يەرلىككە ماسلىشىش ئىقتىدارىنى زور دەرىجىدە ئاشۇرىدۇ؛Web
Worker ئىقتىدارى app نىڭ كۆپ ۋەزىپە ئۆتۈشىگە رۇخسەت قىلىپ، ئىشلەتكۈچىنىڭ ئىش ئۈنۈمى
ۋە مەشغۇلاتنىڭ مۇقىملىقىنى يۇقىرى كۆتۈرىدۇ. Offline Web
Application ئىقتىدارى app نىڭ تورسىز ھالەتتە سانلىق مەلۇماتلارنى ساقلىشىغا رۇخسەت قىلىدۇ،
ھەم بۇنى نورمال زىيارەت قىلغىلى بولىدۇ.


HTML5 سىزنىڭ سەھنىڭىز، سەپەردىشىڭىز. Chorme, FireFox,
Safari قاتارلىقلار سىزگە ئويۇن
قۇيۇشقا تەييارلىنىپ بولدى. ئەمدىلىكتە Internet Explorer10 مۇ سىز ئۈچۈن ئويۇن قۇيۇشقا ئىلتىماس قىلدى. قېنى ئەمىسە سىزمۇ
بۇلارنى سىناشقا تەييارلىنىپ بولدىڭىزمۇ؟
