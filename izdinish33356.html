ئەيدىز ۋە شىنجاڭدىكى ئالىي مەكتەپ ئوقۇغۇچىلىرى

شىنجاڭدىكى ئالىي مەكتەپ ئوقۇغۇچىلىرىنىڭ ئەيدىز ساۋاتلىرىنى بىلىش ئەھۋالى ۋە ئەيدىزگە تۇتقان پوزىتسىيىسى ئۈستىدىكى تەكشۈرۈش دوكلاتى

شىنجاڭ ئۇيغۇر ئاپتونوم رايونىدا 1995 - يىلى ئەيدىز ( HIV ) يۇقۇمدارى بايقالغان بولۇپ، تەڭرىتاغ تورى ( 天山网 ) نىڭ 5 - ئاينىڭ 29 - كۈنى بەرگەن خەۋىرىگە كۆرە ، 2008 - يىلى 3- ئايغىچە بولغان ئارىلىقتا، شىنجاڭ ئۇيغۇر ئاپتونوم رايونى بويىچە ئومۇمىي دوكلات قىلىنغان ئەيدىز يۇقۇمدارى 22756 نەپەرگە يەتكەن. ئومۇمىي يۇقۇملانغۇچىلار سانى جەھەتتە جۇڭگو بويىچە 4 - ئورۇندا تۇرغان. چۈشەندۈرۈشلەرگە قارىغاندا، ئەيدىز يۇقۇمى ئۇيغۇر ئاپتونوم رايون تەۋەسىدىكى 14 ۋىلايەت، ئوبلاست، شەھەردىكى 91 ناھىيە ، شەھەر، بازارغا تارقالغان بولۇپ، يۇقۇملانغان كىشىلەر سانى 60 مىڭ ئەتراپىدا مۆلچەر قىلىنىۋېتىپتۇ. ئەيدىز يۇقۇم ۋەزىيىتىدە يۇقۇملىنىش نىسبىتى يۇقىرى بولۇش، يۇقۇم سۈرئىتى تېز بولۇش، كەڭ كۆلەمدە يۇقۇش يوشۇرۇن خەتىرى زور بولۇشتەك كەسكىن ھالەت شەكىللەنگەن. ئەيدىز نۆۋەتتە شىنجاڭ ئۇيغۇر ئاپتونوم رايونىنىڭ ئىجتىمائىي مۇقىملىقى ۋە تەرەققىياتىغا تەسىر كۆرسىتىۋاتقان ئاساسلىق ئاممىۋىي سەھىيە مەسىلىسىگە ئايلانغان بولۇپ، تېز سۈرئەتتە تارقىلىش ۋەزىيىتى تېخى ئۈنۈملۈك تېزگىنلەنمىگەننىڭ ئۈستىگە، ئەيدىز يۇقۇمى يۇقۇش خەتىرى يۇقىرى كىشىلەر توپىدىن ئادەتتىكى كىشىلەر توپىغا قاراپ كېڭەيمەكتە. دۇنيادىكى ھەرقايسى ئەللەرنىڭ ئەيدىزدىن مۇداپىئەلىنىش تەجرىبىلىرىنى يەكۈنلەشتىن مەلۇم بولۇشىچە، ئەيدىزنىڭ ئالدىنى ئېلىش تەربىيە - تەشۋىقاتى ۋە ئارىلىشىش خاراكتېرلىك ئالدىنى ئېلىش تەدبىرى ئەيدىز يۇقۇمىنى تېزگىنلەشتىكى ئەڭ ئۈنۈملۈك ئۇسۇل ئىكەن. ۋاھالەنكى، شىنجاڭ ئۇيغۇر ئاپتونوم رايونىمىزدا تۈرلۈك سەۋەبلەر تۈپەيلى بۇ ھەقتىكى تەربىيە - تەشۋىقاتنىڭ سالمىقى يەنىلا يېتەرسىز ھالەتتە تۇرىۋاتىدۇ. شۇ ۋەجىدىن دىيارىمىزدىكى كىشىلەردە ئەيدىز ( HIV ) نى چۈشەنمەسلىك، ئەيدىز ( HIV ) يۇقۇمدارلىرى ۋە بىمارلىرىنى كەمسىتىش ئەھۋالى ئېغىر بولىۋاتىدۇ. بۇ  بىزگە ئەيدىز ( HIV ) ھەققىتىكى تەربىيە - تەشۋىقات سالمىقىنى يەنىمۇ زور كۈچ بىلەن ئاشۇرۇش، ئارىلىشىپ ئالدىنى ئېلىش چارە - تەدبىرلىرىنى تېخىمۇ مۇكەممەللەشتۈرۈشنىڭ ئىنتايىن زۆرۈر ۋە تەخىرسىز ئىكەنلىكىنى كۆرسىتىپ بېرىدۇ.
يۇقىرىقىدەك ئارقا كۆرۈنۈش ئاستىدا شىنجاڭ يېزائىگىلىك ئۇنىۋېرسىتېتى ئادىمىيەت - جەمئىيەتشۇناسلىق ئىنستىتۇتىنىڭ قابىل ئوقۇتقۇچىلىرىدىن  康菊花 بىلەن 朱 华 ئىككەيلەن شىنجاڭدىكى ئالىي مەكتەپ ئوقۇغۇچىلىرىنىڭ ئەيدىز ساۋاتلىرىنى بىلىش ئەھۋالى ۋە ئەيدىزگە تۇتقان پوزىتسىيىسى ئۈستىدە مەخسۇس تەكشۈرۈش تەتقىقاتى ئىشلىگەن ھەمدە بۇ ھەقتىكى تەكشۈرۈش دوكلاتىنى << ئۈرۈمچى كەسپىي ئۇنىۋېرسىتېت ئىلمىي ژۇرنىلى ( 乌鲁木齐职业大学学报 ) >> نىڭ 2008 - يىللىق 2 - سانىدا ئېلان قىلغان. تۆۋەندە مەزكۇر تەكشۈرۈش دوكلاتىنىڭ قىسقىچە مەزمۇنىنى كۆپچىلىكنىڭ ھوزۇرىغا سۇنىمىز:
بۇ قېتىملىق تەكشۈرۈش پائالىيىتىدە ، ئۈرۈمچى شەھىرىدىكى 520 نەپەر ئالىي مەكتەپ ئوقۇغۇچىسى زىيارەت قىلىنغان بولۇپ، بۇلارنىڭ ئىچىدە ئوغۇللار  292 نەپەر ( خەنزۇ ئوقۇغۇچىلار 139 نەپەر، ئۇيغۇر ئوقۇغۇچىلار 153 نەپەر )، قىزلار 222 نەپەر  ( خەنزۇ ئوقۇغۇچىلار 122 نەپەر، ئۇيغۇر ئوقۇغۇچىلار 100 نەپەر ) ؛ تەكشۈرۈش نەتىجىلىرىنى ئانالىز قىلغاندا :
بىرىنچى، شىنجاڭدىكى ئالىي مەكتەپ ئوقۇغۇچىلىرى ئەيدىز ھەققىدە مەلۇم چۈشەنچىلەرگە ئىگە ئىكەن. ئەمما، بۇ ئەيدىز ھەققىدىكى چۈشەنچىلىرى بىرقەدەر يۈزە ، ئېنىقسىز ئىكەن.
مەسىلەن، %92 ئوقۇغۇچى ئەيدىز دەيدىغان بىر كېسەللىكنىڭ بارلىقىنى بىلگەن - ئاڭلىغان ئىكەن. بۇلار ئەيدىزنىڭ AIDS ۋە HIV  غا ئوخشاش ئېنگلىزچە قىسقارتىلما يېزىلىشنى بىلىدىكەن. ئەيدىزنىڭ تولۇق ئاتىلىشىنىڭ << ئېرىشىشچان ئىممۇنىتېت كەمتۈكلىكى ئۇنىۋېرسال كېسىلى ( 获得性免疫缺陷综合症 ) >> ئىكەنلىكىنى پەقەت %52 ئوقۇغۇچى بىلىدىكەن. دۇنيا ئەيدىز كۈنىنىڭ ھەريلى 12 - ئاينىڭ 1 - كۈنى ئىكەنلىكىنى پەقەت % ئوقۇغۇچى بىلىدىكەن. %39 ئوقۇغۇچى ئەيدىزنى ئۆزىدىن يىراق - مۇناسىۋەتسىز دەپ قارايدىكەن، بۇ نىسبەت ئوغۇللاردا 55% گە يېتىدىكەن.
ئىككىنچى، ئەيدىزنىڭ يۇقۇش يوللىرىنى توغرا بىلىدىغان ئوقۇغۇچىلار %80 گىمۇ يەتمەيدىكەن.
ئەيدىز ( HIV ) يۇقۇش يولى پەقەت 3 بولۇپ 1. جىنسىي مۇناسىۋەت ئارقىلىق يۇقىدۇ. 2. قان ئارقىلىق يۇقىدۇ. 3. ئاندىن بالىغا ئېمىتىش ۋە تۇغۇت جەريانىدا يۇقىدۇ. ۋاھالەنكى ئوقۇغۇچىلاردىن << ئەيدىز ( HIV ) جىنسىي ئالاقىدىن يۇقامدۇ؟ ئەيدىز ( HIV ) قان ۋە قان ياسالمىلىرىنى ئىشلىتىشىتىن يۇقامدۇ ؟ ئەيدىز ( HIV ) ھامىلىدارلىق ، تۇغۇت، ئېمىتىش مەزگىلىدە ئانىدىن بالىغا يۇقامدۇ؟ >> دەپ سورالغاندا، خەنزۇ ئوقۇغۇچىلارنىڭ %75.5 ، %79.5 ، %75.5 توغرا جاۋاب بەرگەن. ئازسانلىق مىللەت ئوقۇغۇچىلىرىدىن %65 ،  %67 ، %65 تى توغرا جاۋاب بېرەلىگەن. ئازسانلىق مىللەت قىز ئوقۇغۇچىلىرىنىڭ تېخىمۇ تۆۋەن بولۇپ ئاران %58 ،  %57 ، %55 تى توغرا جاۋاب بەرگەن.
زەھەر چەككەندە شىپىرىسنى ئورتاق ئىشلىتىش ئەيدىز ( HIV ) نىڭ يۇقۇشىدىكى ئەڭ خەتەرلىك قىلمىش بولسىمۇ ، << زەھەر چېكىش جەريانىدا شىپىرسنى ئورتاق ئىشلىتىشتىن ئەيدىز ( HIV )  يۇقامدۇ؟ >> دېگەن سوئالغا يەنىلا %18.9 ئوقۇغۇچى << يۇقمايدۇ >> دەپ جاۋاب بەرگەن بولسا، %10.1 ئوقۇغۇچى << بىلمەيمەن >> دەپ جاۋاب بەرگەن. بۇ ئىككىسىنى قوشقاندا % 30 دىن ئىبارەت قورقۇنچلۇق نىسبەت كېلىپ چىققان. كىشىنى چۈچىتىدىغىنى شۇكى بۇ نىسبەت ئاز سانلىق مىللەت ئوقۇغۇچىلاردا %46 گە يەتكەن. ئەپسۇسلىنارلىقى شۇكى، 4/1 ئوقۇغۇچى << ئەيدىز زەھەر چېكىش ياكى ۋېنادىن زەھەر ئۇرۇش جەريانىدا   يۇقمايدۇ / بىلمەيمەن >> دەپ جاۋاب بەرگەن.

ئۈچىنچى، ئەيدىز ( HIV ) نىڭ يۇقۇش يولى ھەققىدىكى گەپ سۆزلەر مۇبالىغىلەشتۈرۈلۈپ، جەمئىيەتتە ئەيدىز ( HIV )  دىن ئورۇنسىز قورقۇش << ۋەھىمىسى >> پەيدا بولغان.
ئەيدىز ( HIV ) ۋىرۇسى ناھايىتى ئاجىز ۋىرۇس بولۇپ، ناھايىتى ئوڭاي ئۆلۈپ قالىدۇ. شۇڭا، ھاۋا ، يېمەكلىك، سۇ قاتارلىق ئادەتتىكى تۇرمۇشىمىزدا ئۇچرىشىدىغان نەرسىلەر ئارقىلىق يۇقمايدۇ. بۇلاردىن باشقا، ئەيدىز ( HIV )  ۋىرۇسى پاشا قاتارلىقلارنىڭ تېنى ياكى قېنىدا ياشىيالمايدۇ. شۇڭا، پاشىنىڭ چېقىشىدىن ئەيدىز ( HIV )  يۇقمايدۇ. ئەيدىز ( HIV ) نىڭ يۇقۇشى ئىنسانلارنىڭ قىلمىشى بىلەن مۇناسىۋەتلىك بولۇپ، كىشىلەرنىڭ قىلمىشىنى تۈزەش ياكى تەرتىپكە سېلىش ئارقىلىق ئەيدىز ( HIV ) نىڭ ئالدىنى تامامەن ئالغىلى بولىدۇ.
ئەمما، بۇ قېتىملىق تەكشۈرۈشتىن مەلۇم بولۇشىچە، %25 ، %62 ، %27 ، %19.6 ، %21.8  ۋە %11.7 ئوقۇغۇچى << يۆتىلىش، چۈشكۈرۈش، پاشا چېقىش، ئەيدىز ( HIV ) يۇقۇمدارى بىلەن قۇچاقلىشىش، سۆيۈشۈش، قول ئېلىشىپ كۆرۈشۈش،  بىرگە تاماق يېيىش، سۇ ئۈزۈش، ئوبورنى ئىشلىتىش، بىر سىنىپتا دەرس ئاڭلاش >> ئارقىلىق ئەيدىز ( HIV )  يۇقىدۇ ، دەپ قارىغان. << بىلمەيمەن >> دەپ جاۋاب بەرگەن ئوقۇغۇچىلارنىڭ سانىنى قوشقاندا بۇ نىسبەت تېخىمۇ يۇقىرى بولغان.

تۆتىنچى ، ئالىي مەكتەپ ئوقۇغۇچىلىرىنىڭ ئالىي مەكتەپ ئوقۇش ھاياتىدا ئەيدىز ( HIV ) بىلىملىرىگە ئالاقىدار بولغان تەربىيە - مائارىپ يېتەرسىز ئىكەن.
پەقەت %50 ئوقۇغۇچى ئوقۇش داۋامىدا ئەيدىز ( HIV )  ھەققىدىكى تەربىيە - لېكسىيىنى قوبۇل قىلغانلىقىنى ئېيتقان. تەڭ يېرىم ئوقۇغۇچىلار بۇ خىل تەربىيە ياكى لېكسىيىنى ئاڭلاش پۇرسىتىنىڭ يوقلىقىنى تىلغا ئېلىشقان. %57 ئوقۇغۇچى ئۆزلىرى ئىگىلىگەن ئەيدىز ( HIV )  ھەققىدىكى بىلىملەرنى باشقىلارغىمۇ سۆزلەپ بېرىشنى خالايدىغانلىقىنى بىلدۈرگەن.
بەشىنچى، گېزىت - ژۇرنال، رادىئو - تېلېۋىزور، تور قاتارلىق تاراتقۇلار ئالىي مەكتەپ ئوقۇغۇچىلىرنىڭ ئەيدىز ( HIV )  ھەققىدىكى ئۇچۇرلارنى ئىگىلىشىدىكى ئاساسلىق ۋاسىتىلەر ئىكەن.
ئالتىنچى، ئالىي مەكتەپ ئوقۇغۇچىلىرىنىڭ ئەيدىز ( HIV )  بىمارى ۋە يۇقۇمدارلىرىغا تۇتقان پوزىتسىيىسىدىكى پەرق زور بولۇپ، كۆپ قىرلىق بولۇش ۋەزىيىتى مەۋجۇد ئىكەن. %68 ئوقۇغۇچى << ئەيدىز ( HIV )  بىمارى - يۇقۇمدارلىرىغا كۆيۈنۈش، ھېسداشلىق قىلىش كېرەك >> دەپ قارايدىكەن. %21 ئوقۇغۇچى بۇ خىل قاراشقا قارشى ئىكەن. %60 ئوقۇغۇچى يېنىدا ئەيدىز ( HIV )  بىمارى - يۇقۇمدارى پەيدا بولۇپ قالسا ، ئادەتتىكى كىشىلەرگە ئوخشاش مۇئامىلە قىلالمايدىغانلىقىنى بىلدۈرگەن.
يەتتىنچى، << ئۆزىنى قەدىرلەپ ، پاك دىيانەتلىك بولۇش >> تەربىيىسىنى يەنىمۇ كۈچەيتىش كېرەك ئىكەن.
تەكشۈرۈشتىن مەلۇم بولۇشىچە، << تويدىن ئىلگىرىكى جىنسىي مۇناسىۋەت >> نى قوللايدىغان ئوقۇغۇچىلار %24.1 ، << بەرىبىر >> دەپ قارايدىغان ئوقۇغۇچىلار %15.5 بولۇپ، بۇ ئىككى نىسبەتكە << بىلمەيمەن >> دەپ جاۋاب بەرگەن %11.7 ئوقۇغۇچىنى قوشقاندا، %51.4 ئوقۇغۇچى << تويدىن ئىلگىرىكى جىنسىي مۇناسىۋەت ئۆتكۈزۈش قىلمىشى >> غا قارشى ئەمەسكەن. << كۆپ جىنسىي جۆرىسى بولۇشقا قانداق قارايسىز >> دېگەن سوئالغا % 9.1 ئوقۇغۇچى << قوللاش>> ، %11.3 ئوقۇغۇچى << بەرىبىر >> دەپ ئاۋاز بەرگەن . بۇ ئىككى نىسبەتنى قوشقاندىمۇ %20 دىن ئېشىپ كەتكەن.

مەزكۇر تەكشۈرۈش تەتقىقاتىغا ئائىت تەپىسىلىي ستاتىستېكا جەدۋەللىرىنى ئەسلى مەنبە بولغان << يۈكسەل بلوگى >> دىن كۆرگەيسىز : http://yuksel.blogbus.com

يۇقىرىقى تەكشۈرۈش دوكلاتى سەمىمىزگە شۇنى سالدىكى،  دىيارىمىز ياشلىرى جۈملىدىن ئالىي مەكتەپ ئوقۇغۇچىلىرىنىڭ ئەيدىز ( HIV )  ھەققىدىكى بىلىملىرى يەنىلا ئىنتايىن يېتەرسىز ھالەتتە بولۇپ، بۇ ھەقتىكى تەشۋىقات - تەربىيە خىزمىتىنى يەنىمۇ ئوبدان ئىشلەش كېرەك ئىكەن.  بولۇپمۇ << يۈكسەل بلوگى >> ھۆكۈمەت ئورۇنلىرىنىڭ بۇ ھەقتىكى چاقىرىقلىرىغا ئاكتىپ ئاۋاز قېتىپ، ئۇلارنىڭ خىزمەتلىرىگە يېقىندىن ھەمدەمدە بولۇپ،  ئۆزىنىڭ ئەيدىز ئۇچۇرلىرى ۋە ساۋاتلىرىنى تورداشلارغا يەتكۈزۈش ، ئەيدىزگە قارشى كۆرەش سېپىدە ئۆزىنىڭ تېگىشلىك ھەسسىسىنى قوشۇشتەك مەۋقەسىدە ئۇزۇنغىچە چىڭ تۇرۇپ، بۇ جەھەتتە تېخىمۇ زور تىرىشچانلىق كۆرسىتىشى كېرەك ئىكەن.
