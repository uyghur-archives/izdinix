يېڭى مىنىستىرلىرىمىز ۋەزىپىگە ئولتۇردى[تەرجىمە]

جۇڭگو ئاخبارات تورىنىڭ 16 - مارت خەۋىرى :


12 - نۆۋەتلىك مەملىكەتلىك خەلق قۇرۇلتىيى 1 - يىغىنى 16 - مارت چۈشتىن كىيىن 6 - قېتىملىق ئومۇمىي يىغىن ئۆتكۈزدى . يىغىندا بېلەت تاشلاش شەكلى ئارقىلىق جاڭ گاۋلى ، ليۇ يەندوڭ ، ۋاڭ ياڭ ، ما كەي قاتارلىق تۆت كىشى دۆلەت ئىشلىرى مەھكىمىسىنىڭ ( گوۋۇيۈەننى شۇنداق ئاتامدا ؟ ) مۇئاۋىن زوڭلىلىقىغا سايلاندى .


بۇ يىغىندا يەنە ھەر قايسى مىنىستىرلىكلەرنىڭ مىنىستىرلىرى ، كومىتېت مۇدىرلىرى ، جۇڭگو خەلق بانكىسىنىڭ باشلىقى ، باش مۇپەتتىش ۋە باش كاتىپلار تەيىن قىلىندى .


ياڭ جىڭ ( 杨晶 ) دۆلەت ئىشلىرى مەھكىمىسىنىڭ ( گوۋۇيۈەننىڭ ) باش كاتىپلىقىغا تەيىنلەندى .


ۋاڭ يى ( 王毅 ) دېپلوماتىيە مىنىستىرلىكىگە تەيىنلەندى .


چاڭ ۋەنچۈەن ( 常万全 ) دۆلەت مۇداپىئە مىنىستىرلىكىگە تەيىنلەندى .


شۈ شاۋشى ( 徐绍史 ) دۆلەت تەرەققىيات - ئىسلاھات كومىتېتىنىڭ مۇدىرلىقىغا تەيىنلەندى .


يۈەن گۈيرېن ( 袁贵仁 ) مائارىپ مىنىستىرلىكىگە تەيىنلەندى .


ۋەن گاڭ ( 万钢 ) پەن - تېخنىكا مىنىستىرلىكىگە تەيىنلەندى .


مياۋ ۋېي ( 苗圩 ) سانائەت ۋە ئۇچۇرلاشتۇرۇش مىنىستىرلىكىگە تەيىنلەندى .


ۋاڭ جېڭۋېي ( 王正伟 ) دۆلەت مىللەتلەر ئىشلىرى كومىتېتىنىڭ مۇدىرلىقىغا تەيىنلەندى .


گۇ شېڭكۈن ( 郭声琨 ) جامائەت خەۋپسىزلىك مىنىستىرلىكىگە تەيىنلەندى .


گېڭ خۈيچاڭ ( 耿惠昌 ) دۆلەت خەۋپسىزلىك مىنىستىرلىكىگە تەيىنلەندى .


خۇاڭ شۇشيەن ( 黄树贤 ) رېۋىزىيە مىنىستىرلىكىگە تەيىنلەندى .


لى لىگو ( 李立国 ) خەلق ئىشلىرى مىنىستىرلىكىگە تەيىنلەندى .


ۋۇ ئەييىڭ ( 吴爱英 ) ئەدلىيە مىنىستىرلىكىگە تەيىنلەندى .


لۇ جىۋېي ( 楼继伟 ) مالىيە مىنىستىرلىكىگە تەيىنلەندى .


يىن ۋېيمىن ( 尹蔚民 ) ئادەم كۈچى بايلىقى ۋە ئىجتىمائىي كاپالەت مىنىستىرلىكىگە تەيىنلەندى .


جياڭ دامىڭ ( 姜大明 ) دۆلەت زىمىن بايلىقى مىنىستىرلىكىگە تەيىنلەندى .


جوۋ شېڭشيەن ( 周生贤 ) موھىت ئاسىراش مىنىستىرلىكىگە تەيىنلەندى .


جياڭ ۋېيشىن ( 姜伟新 ) تۇرالغۇ ۋە شەھەر - يېزا قۇرۇلۇشى مىنىستىرلىكىگە تەيىنلەندى .


ياڭ چۈەنتاڭ ( 杨传堂 ) قاتناش - ترانسپورت مىنىستىرلىكىگە تەيىنلەندى .


چېن لېي ( 陈雷 ) سۇ ئىشلىرى مىنىستىرلىكىگەتەيىنلەندى .


خەن چاڭفۇ ( 韩长赋 ) يېزا ئېگىلىكى مىنىستىرلىكىگە تەيىنلەندى .


گاۋ خۇچېڭ ( 高虎城 ) سودا ئىشلىرى مىنىستىرلىكىگە تەيىنلەندى .


سەي ۋۇ ( 蔡武 ) مەدەنىيەت مىنىستىرلىكىگە تەيىنلەندى .


لى بىن ( 李斌 ) دۆلەت سەھىيە ۋە پلانلىق تۇغۇت خىزمىتى كومىتېتىنىڭ مۇدىرلىقىغا تەيىنلەندى .


جوۋ شياۋچۈەن ( 周小川 ) جۇڭگو خەلق بانكىسىنىڭ باشلىقلىقىغا تەيىنلەندى .


ليۇ جيايى ( 刘家义 ) مۇپەتتىش مەھكىمىسىنىڭ باش مۇپەتتىشلىكىگە تەيىنلەندى .
