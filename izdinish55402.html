Android

Android دېگەن سۆزنىڭ ئەسلى مەنىسى: ماشىنا ئادەم شۇنداقلا Google شىركىتى 2007-يىلى 11-ئاينىڭ 5-كۈنى ئېلان قىلغان Linux سۇپىسى ئاساسىدىكى ئوچۇق سىستېمىنىڭ نامى ھېسابلىنىدۇ. بۇ خىل سۇپىدا سىستېما، ۋاستىچى ئۈسكىنە، كۆرۈنمە يۈز، قوللىنىشچان يۇمشاق دېتاللار ۋە ئابونت زىچ بىرلەشتۈرۈلگەن. نۆۋەتتە ئەڭ يېڭى نەشىرلىرىدىن Android 2.4 Gingerbreadۋە Android 3.0 Honeycomb قاتارلىقلار بار.

سىستېما تەرجىمىھالى

Android نىڭ ئاساسى بولسا ئېچىۋېتىلىشچانلىق يادرولۇق Linux سىستېمىسى بولۇپ، Google شىركىتى 2007-يىلى 11-ئاينىڭ 5-كۈنى ئېلان قىلغان يانفون مەشغۇلات سىستېمىسى. ئەڭ دەسلەپكى مەزگىلدە Android ناملىق شىركەت بۇ سىستېمىنى ئاچقان، 2005-يىلى گۇگېل qAndroid.lnc; نى سېتىۋالغاندىن كېيىن، Android نى داۋاملىق ئاچقان ۋە تەرەققىي قىلدۇرغان ۋە يۇمشال دېتال قەۋىتىنىڭ (soft ware stack) نىڭ قۇرۇلمىسىنى ئىشلىتىشكە باشلىغان بولۇپ، ئاساسلىقى ئۈچ قىسىمغا بۆلۈنگەن. ئاساسى قىسمىنى Linux يادروسى ئەڭ ئاساسلىق ئىقتىدارلىرى ئارقىلىق تەمىنلەيدۇ، باشقا قوللىنىشچان دېتاللارنى ھەرقايسى شىركەتلەر ئېچىپ تەرەققىي قىلدۇرىدۇ، قىسمەن پروگراممىلارنى JAVA ئارقىلىق ياسايدۇ.

2011-يىلى دەسلەپكى قەدەمدىكى سانلىق مەلۇماتلارغا ئاساسلانغاندا، Android بازارغا چىقىپ تېز سۈرئەتتە تەرققىي قىلىپ ئون يىل زومىگەر دەپ ئاتالغان سىمبياننى تەدرىجى بازاردىن سىقىپ چىقارغان ھەمدە يەرشارى بويىچە ئەڭ كۆپ ئالقىشقا ئېرىشىۋاتقان ئەقلىي ئىقتىدارلىق سىستېمىسىغا ئايلانغان. نۆۋەتتە Android يانفون ساھەسىدىلا ئەمەس بەلكى تاختا كومپىيۇتېر ساھەسىدىمۇ تېز سۈرئەتتە كېڭەيمەكتە، بىرلا ۋاقىتتا MP4 ساھەسىگىمۇ يۈزلەنمەكتە. ھازىر Android ئىشلىتىدىغان شىركەت–كارخانىلاردىن تەيۋەن HTC  (گۇگېلنىڭ تۇنجى يانفونى G1 نى ۋاكالىتەن ئىشلەپچىقارغان)، كورىيە سامسۇڭ، ئامېرىكا موتورولا، ئەنگىلىيە سونى ئېرىكسون قاتارلىق خەلقئارالىق شىركەتلەر بار، جۇڭگو چوڭ قۇرۇقلۇقىدىن خۇاۋېي، مېيزۇ، جوڭشىڭ، ليەنشياڭ قاتارلىقلار بار.

سىستېما قۇرۇلمىسى

قوللىنىشچان پروگراممىلار: Android بولسا JAVA ئارقىلىق تۈزۈلىدۇ، ئۇلىنىش ئېغىزىدىن تارتىپ ئىقتىدارلىرىغىچە ئىنتايىن كۆپ ئۆزگەرتىلگەن. ھەر بىر ئىقتىدارغا بىر Activity مەسئۇل بولىدۇ، ئىككىلىكلەر بولسا Connection ceServis ۋە AIDL ئارقىلىق بىرىكىدۇ. بەلگىلەنگەن سانلىق مىقدارغا يەتكەندە قوزغىلىش ئۈنۈمى يۈز بېرىدۇ.

پروگرمما تۈزۈش تىلى

Android بولسا Linux kernel دا قوزغىلىدۇ، لېكىن GNU/Linux ئەمەس. چۈنكى GNU/Linux قوللايدىغان Cairo, X11, Alsa, FFmpeg, GTK, Pango قاتارلىقلارنى Android قوللىمىغاچقا چىقىرۋېتىلگەن. شۇڭلاشقا bionic نى Glibc نىڭ ئورنىغا، Skia نى Cairo نىڭ ئورنىغا، opencore نى FFmpeg نىڭ ئورنىغا ئىشلىتىلگەن. نۆۋەتتە  Linux kernel نىڭ Android دىكى كونتروللىقى بولسا ئاساسلىقى بىخەتەرلىك، ساقلىغۇچ باشقۇرۇش، پروگرامما باشقۇرۇش، تور، قوزغىتىش مودىلى قاتارلىقلار. Android نىڭ ئەسلى كودىنى چۈشۈرۈشتىن بۇرۇن، بىرىكتۈرۈش دېتالى Repo نى قاچىلىشى كېرەك. Repo بولسا Android نىڭ Git خىزمىتىنىڭ ياردەمچى قورالى." N0                                               

بىخەتەرلىك ۋە ھوقۇق چەكلىمىسى

Android بولسا بىرخىل ھوقۇق چەكلىمىسىگە ئايرىلغان مەشغۇلات سىستېمىسى. بۇ خىل سىستېمىدا ھەر بىر قوللىنىلمىلارغا بىردىن سىستېما سالاھىيەت تونۇشى ئارقىلىق قوزغىلىدۇ. سىستېمىنىڭ ھەر قايسى قىسىملىرىمۇ ئۆزىگە خاس تونۇش شەكلىگە ئىگە. Linux مانا مۇشۇنداق قوللىنىلما بىلەن قوللىنىلما، قوللىنىلما بىلەن سىستېما ئايرىۋېتىلگەن سىستېمىدۇر. سىستېمىنىڭ تېخىمۇ كۆپ بىخەتەرلىكىگە نىسبەتەن ھوقۇق چەكلەش يولغا قويۇلغان. ھوقۇق چەكلەش مەلۇم خىل ئالاھىدە بەلگىلەنگەن ئالاھىدە مەشغۇلاتنى چەكلەيدۇ ھەمدە ھەر بىر ئالاھىدە بەلگىلەنگەن URI سانلىق مەلۇماتىنى زىيارەت قىلىشنىمۇ چەكلەيدۇ.

Android نەشىرى




2008-يىلى 9-ئايدا Android 1.1 ئېلان قىلىنغان؛

2009-يىلى 4-ئايدا Android 1.5 ئېلان قىلىنغان؛

2009-يىلى 9-ئايدا Android 1.6 ئېلان قىلىنغان؛

2009-يىلى 10-ئاينىڭ 26-كۈنى Android 2.0 ئېلان قىلىنغان؛

2009-يىلى 10-ئينىڭ 26-كۈنى يەنە بىر ئاساسلىق يېڭى نەشىرى ئىنتايىن تېز سۈرئەتتە ئېلان قىلىنغان. بۇ قېتىم نەشىرى بىۋاستە 2.1 گە كۆتۈرۈلگەن بولۇپ، ئاساسلىق ئالاھىدىلىكى: قاتتىق دېتال تېزلىكى، ئېكران چوڭلۇقى، ئېكران رەسىمى، كۆرۈنمە يۈزنى ئەلالاشتۇرۇش، مەۋھۇم كونۇپكا تاختىسى، كۆك چىش 2.1 گۇگېل خەرىتىسى Android گۇگېل 2010-يىلى 5-ئاينىڭ 20-كۈنى كەچ سائەت 10:30 دا سان–فرانسىسكو Moscone يىغىن–كۆرگەزمە زالىدا Google I/O 2010 قۇرۇلتىيىنىڭ 2-كۈنى Google ئۆزىنىڭ ئەڭ يېڭى نەشىرى بولغان Android 2.2 نى ئېلان قىلغان. بۇ نەشىردە بۇرۇنقى نەشىرلىرىدە يوق ئالاھىدىلىكلەر بار بولۇپ، ئۇلار: 1. ئىقتىدارى كەڭ كۆلەملىك يۇقىرى كۆتۈرۈلگەن؛ 2 .3G تورى ھەمبەھىر ئىقتىدارى؛ 3 . Flash نى قوللاش؛ 4.  App2sd ئىقتىدارى؛ 5.  پۈتۈنلەي يېڭى بولغان يۇمشاق دېتال بازىرى؛ 6. تېخىمۇ كۆپ web قوللىنىشچان API ئېغىزىنىڭ تەتقىقاتى؛ 2010-يىلى 12-ئاينىڭ 7-كۈنى تاڭ سەھەردە، گۇگېل رەسمىي يوسۇندا 2.3 نەشىرىنى، يەنى Android gingerbread نى ئېلان قىلغان.
