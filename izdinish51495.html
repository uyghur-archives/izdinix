Windows 8 Pro نى ئاكتىپلاش[تەرجىمە]

Windows 8 چىققاندىن كېيىن نۇرغۇن دوسلار ھەقىقىي نۇسخىسىنى سېتىۋالغان بولسىمۇ، كۆپلىگەن كىشىلەر پۇل تۆلىيەلمەي Windows 8 نى ئىشلىتەلمەي قىينالماقتا. بىر نەچچە كۈن بۇرۇن 百度贴吧 دا Windows 8 Pro نى ئاكتىپلاش ئۇسۇلى تونۇشتۇرۇلغان بىر يازمىنى كۆرۈپ تازا ئىشەنمىگەنتىم، قايسى كۈنى شىنخۇا تورى قاتارلىق چوڭ-چوڭ تور بەتلەردىمۇ بۇ توغرىلىق خەۋەرلەرنى كۆرۈپ بىر سىناپ بېقىش قارارىغا كېلىپ، ئاشۇ يازمىدا دېيىلگەن ئۇسۇل بويىچە مەشغۇلات قىلىپ راستىنلا مۇۋاپىقىيەتلىك ئاكتىپلاشتۇردۇم. ھەقىقەتەن 100% ھەقىقىي نەشىرىگە ئايلىنىدىكەن.


مۇناسىۋەتلىك خەۋەر:http://news.xinhuanet.com/yzyd/tech/20121123/c_113778340.htm


ھازىر بۇ خەق ھەممىسى مۇشۇ ئۇسۇل بىلەن ئاكتىپلاپ ئىشلىتىۋېتىپتۇ.بەيدۇدىن windows8 kms 激活 دەپ ئىزدىسەڭلار مۇشۇ ئۇسۇل چىقىدۇ.


ئەگەر باشقۇرغۇچىلار مېنىڭ بۇ ئۇسۇلنى تونۇشتۇرىشىمنى توغرا كۆرسە بۇ تېمىنى تەستىقلاپ قويساڭلار تەپسىلىي قەدەم باسقۇچلىرى بىلەن يازاي، بولمىسا ئۆچرىۋەتسەڭلار بولدى.

ئالدى بىلەن http://windows.microsoft.com/zh-CN/windows-8/feature-packs بۇ تور بەتتىن Windows 8 Media Center Pack نىڭ نۇمۇرىدىن بىرنى ئېلتىماس قىلىمىز(بىر email ئادىرسى بولسىلا بولىدۇ). بۇ نۇمۇرنى تاپشۇرىۋېلىغقا خېلى جىق ۋاقىت كېتىدىكەن، مەنمۇ بىر كۈن ساقلاپ ئاران تاپشۇرىۋالدىم.

ئەمدى توردىن Windows 8 Pro نۇسخىسىنى تېپىپ چۈشۈرۈپ قاچلايمىز. چۈشۈرۈش ئادېرسى توردىن ئىزدىسىڭىز خېلى جىق. ماۋۇ يەردە 32بىتلىق نۇسخىسى باركەن لازىم بولغانلار مۇشۇنى چۈشۈرسەڭلارمۇ بولىدۇ.http://pan.baidu.com/share/link?shareid=130519&uk=4096017782

قاچلىغان چاغدا توردىن خالىغان بىرWindows 8 Pro نىڭ نۇمۇرىنى تېپىپ كىرگۈزسىڭىز قاچلىنىۋېرىدۇ.

قاچلاپ بولغاندىن كېيىن ） Win + X 选择 命令提示符（管理员نى تاللايمىز.

ئېچىلغان كۆزنەككە

slmgr /ipk NG4HW-VH26C-733KW-K6F98-J8CK4

slmgr -skms kms.djxmmx.net

slmgr -ato

ماۋۇ بۇيرۇقنى كىرگۈزسەك، بىز قاچلىغان ۋىندوۋس8 180 كۈنلۈك سىناق نۇسخىغا ئۆزگىرىدۇ، ھەم ئاكتىپلاش مۇۋاپىقىيەتلىك بولغانلىق ئۇچۇرى چىقىدۇ.

ئاندىن بىز باشتا بىلەن ئىلتىماس قىلىپ قويغان نۇمۇرنى كۆچۈرۈپ، win+q كۇنۇپكىسىنى باسساق ئىزدەش رامكىسى چىقىدۇ، ئۇ يەرگە 添加功能 دەپ يېزىپ ئىزدپ، 向 Windows 8 添加功能 نى تاللايمىز. چىققان كۆزنەككە بايامقى نۇمۇرنى كىرگۈزۈپ بىردەم ساقلاپ تۇرساق كومپىيوتېر قايتا قوزغىلىدۇ، شۇنىڭ بىلەن بىزنىڭ سېستىمىمىز ھەقىقىي نەشىرگە ئايلىنىدۇ.



بۇ ئۇسۇل بويىچە مەن ئۈلۈشكۈن ئۆزەمنىڭ كومپىيوتېرىنى مۇۋاپىقىيەتلىك ئاكتىپلىدىم. لازىم بولغانلار سىناپ بېقڭىلار.



ئاۋۇ بۇيرۇقلارنى خاتىرە دەپتەرگە كۆچۈرىۋېلىپ، ساقلايدىغان چاغدا bat ھۆججىتى قىلىپ ساقلىسىڭىز، ئاشۇنى بىۋاستە باسسىڭىزلا 180 كۈنلۈك سىناق نۇسخىغا ئۆزگىرىدۇ.



مەنبەسىنى ئەسكەرتىپ باشقا يەرگە كۆچۈرسەڭلەر بولىۋېرىدۇ.
