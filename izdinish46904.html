‹‹ئۇبۇنتۇ››نى ئىشلىتەمدۇق ‹‹كۆزنەك8 ››نىمۇ؟[كۆچۈرۈلمە]




مانا ئۇيغۇرچە تىل (تىل رايۇنى بار ) قوشۇلغان سىستېما 2 بولدى. ئالدىمىزدا بىر مەيدان مەشغۇلات سېستىمىسى ئىنقىلابى كۈتۈپ تۇرىدۇ. تاللاش ئۆزىمىزدە، بىرى ھەقلىق، بىرى ھەقسىز.

1 - ئۇبۇنتۇ (ubuntu linux ) نىڭ 12 نەشرىنىڭ سىناق نۇسخىسى تارقىتىلدى.   

بۇ پۈتۈنلەي ھەقسىز دىتال بولۇپ بۇنىڭدىكى libre office  ھەم ھەقسىز ھەم پۈتۈنلەي ئۇيغۇرچە. بۇنى غەيرەت توختى كەنجى، ساھران قاتارلىق يىگىتلىرىمىز خالىس ئۇيغۇرچىلاشتۇرغان بولۇپ تەرجىمە جەريانىدا، نۇرغۇن قىيىنچىلىق، جاپا-مۇشەققەتلەر يېڭىلىپ، كۆپ ئەجىرلەر سىڭدۈرۈلۈپ، يۇقۇرقىدەك خوشاللىنارلىق نەتىجىلەر قولغا كەلگەن بولسىمۇ، ئەمما بۇ سىستېمىنىڭ ئۇيغۇرچە تەرجىمىسى يەنىلا بەزى نۇقسان-سەۋەنلىكلەردىن خالىي ئەمەس. ئۇبۇنتۇ ئۇيغۇر تەرجىمە گۇرۇپپىسى بۇ سىستېمىنى تەرجىمە قىلىش، سىناش جەريانىدا ھەر ۋاقىت كۆپچىلىكنىڭ ياردىمى، قوللىشىغا مۇھتاج. ئۆزىمىزنىڭ سىستېمىسى، تىل-يېزىقى، ئۆز قېرىنداشلىرىمىز ئۈچۈن، پەقەت ۋە پەقەت ئۆزىمىزلا ياردەم قىلالايمىز. قېنى، بۇ ئەھمىيەتلىك ئىشقا سىز قانچىلىك ياردەم قىلىشنى ئويلاۋاتىسىز؟




2- كۆزنەك8 (windows8 ) بولۇپ بۇ بىز نەچچە ۋاقنىڭزى كۆنگەن سىستېما ئەمما ھەقلىق.

بۇنىڭ ئۇيغۇرچە تىل بولىقىنى  uighursoft شىركىتىنىڭ قۇرغۇچىسى ئالىم ئەخەت ئەپەندىم ئىشلىگەن بولۇپ رەسمىي نۇسخىسى تارقىتىلىشقا ئاز قالدى.

باھاسى ھەققىدىكى پەرەز

Windows 7 Starter(دەسلەپكى نەشرى) جوڭگۇدا سېتىلمايدۇ

Windows 7 Home Basic (ئائىلە نەشرى) باھاسى 399元

Windows 7 Home Premium (ئائىلە ئالىي نۇسخىسى ) 699元

Windows 7 Professional(كەسپىي نۇسخىسى ) 1380元

Windows 7 Enterprise (كارخانا نۇسخىسى ) توپ سېتىلىدۇ

Windows 7 Ultimate (بايراقدار نۇسخىسى ) 2460元

Ubuntu نىڭ نەشىرى ھەر قىتىم يېڭىلانغاندا قاچىلاپ باقىمەن(مەن ھازىرغىچە ئۇچراتقان قاچىلاش ئەڭ ئاسان مەشغۇلات سىستېمىسى). ئىككى ئۆتە-ئۆتمەي كومپىيۇتېردىن ئۆچۈرىمەن، سەۋەبى خاتىرە كومپىيۇتىرىمنى بەكلا قىزىتۋىتىدۇ(Asus,HP قاتارلىق خاتىرە كومپىيۇتېرلارغا قاچىلاپ باققان).

Ubuntu توختىماي نەشىر يېڭىلاشنى ئاستىلىتىپ كومپىيۇتىرلارنىڭ قوزغاتقۇچ پروگراممىسىغا كۈچىسە بولاتتى.



ھازىرغا قەدەر مەشغۇلات سىستېمىسى ساھەسىدە Microsoft كۆرسەتكەن تىرىشچانلىقنى ھىچكىم تىخى كۆرسەتمىدى، بۇنداق دىيىشىمنىڭ سەۋەبى:

دۇنيادىكى بارلىق قاتتىق دېتاللارنىڭ Windows مەشغۇلات سىستېمىسىدىكى قوزغاتقۇچ پروگراممىسى بار، كومپىيۇتېردا ئاۋاز تونۇمايدىغان، ئىكران ئېنىقلىقى ماسلاشمايدىغان، قىزىپ كىتىدىغان ئىش يوق.

يۇمشاق دېتال بايلىقى زىيادە مول. ھەرخىل ساھەدىكى ھەرخىل يۇمشاق دېتاللارنىڭ Windows نۇسخىسى ئاساسەن بار، بولۇپمۇ ھەرقانداق ئويۇننىڭ Windows نۇسخىسى بار.



Ubuntu مەشغۇلات سىستېمىسى ئويۇن ئوينىمايدىغانلار، كەسپىي ئىش قىلمايدىغانلارنىڭ ئېھتىياجىنى قامدايدۇ(پروگرامما ئاچقۇچىلار ئۈچۈن ياخشى تاللاش)
