ASP.NET دەرسلىكى(1-قىسىم مۇھىت ھازىرلاش ۋە كىرىش)

  ASP.NET دا تور بېكەت ئېچىش مۇنبىرىمىزنىڭ قۇرۇلۇشىنىڭ ئاساسلىق نىشانى بولۇپ، مۇنبەر قۇرۇلغاندىن بېرى ASP.NET غا دائىر ھېچقانداق دەرسلىك ياكى ماتېرىيال يوللانمىغان ئىدى. ئۆزۈممۇ بۇ پىروگرامما تىلىغا ئالاھىدە ئىشتىياقى بولغاچقا بىر كۈندە ياكى ئىككى كۈندە بىر قىسىم دەرسلىك ئىشلەشنى ئويلىشىپ، ئۇشبۇ دەرسلىكنى داۋاملاشتۇرۇش ئۈچۈن ئىزدىنىۋاتىمەن.
  بۇ دەرسلىكنىڭ ئاخىرىنىڭ داۋاملىشىشى ئۈچۈن مۇنبەرداشلارنىڭ يېقىندىن قوللاپ بېرىشىنى ئۈمىد قىلىمەن. ئەگەر ئانچە قىزىقىش بولمىسا بۇ دەرسلىك ئىزلەشنى توختىتىپ قويۇشىمىز مۇمكىن. شۇڭا مۇنبەرداشلارنىڭ ئاكتىپ ئاۋاز قوشۇشىنى ۋە ئۇستازلارنىڭ يوليورۇقلاردا بولىشىنى ئۈمىد قىلىمىز.
  
  ئەمىسە دەرسنى باشلاپ كەتتۇق. دەرسنى باشلاشتىن ئاۋۋال كومپيۇتېرىمىزغا Visual Studio 2010/2012/2013 نەشرىلىرىنىڭ خالىغان بىرى قاچىلانغان بولىشى كېرەك. ناۋادا بۇ ئېچىش مۇھىتى قاچىلىمىغان بولسىڭىز تۆۋەندىكى ئادرېستىن چۈشۈرۈڭ:
چۈشۈرۈش
مەزمۇننى كۆچۈرۈۋېلىش

  قاچىلاش باشقا يۇمشاق دېتاللارنى قاچىلىغانغا ئوخشايدۇ. لېكىن سىز بۇنى قاچىلىشىڭىز ئۈچۈن كېتىدىغان شەرتلەر: C دىسكا سىغىمىڭىزدىن 20G بوشلۇق بولىشى، شۇنداقلا C دىسكىغا قاچىلىشىڭىز شەرت. قاچىلىغان ۋاقىتتا بىر مۇنچە تاللاش ئورۇنلرى بار، سىز ئامال بار ھەممىسىنى تاللىغان ئاساستا قاچىلىشىڭىزنى تەۋسىيە قىلىمىز.

  خوش، ئەمدى بىز Visual Studio2013 دە ASP.NET  تور بېتى ئېچىش تەييارلىقىنى قىلىمىز.
  1. Visual Studio2013 نى قوززغىتىمىز

2.  ASP.NET پىروگرامما ئېچىش تۈر ھۆججىتىدىن بىرنى قۇرىمىز

3. 1. كۆرستىلغىنى بولسا تۈر ئېچىش تىلىنى كۆرسىتىدۇ. بۇنىڭدا ئىككى تىلدا ئېچىشقا بولىدۇ. بىز C# تىلىنى تاللايمىز. نېمىشقا بۇنى تاللايمىز؟ بۇنىڭ جاۋابى بىلمەكچى بولسىڭىز نەزىرىڭىز ماۋۇ يازمىدا بولسۇن: #C تىلى ئارقىلىق نېمە ئىشلارنى قىلىغىلى بولىدۇ؟ . 2- قۇرۇش ASP.NET تۈرىنى تاللايمىز. 3- تور بېكەت پىروگراممىمىزنىڭ ساقلىنىدىغان ئورنى. بىر نەرسىنى ئۇنتۇپ قاپتىمەن. ئەڭ ئۈستى بۇرجەكتە .NET Framwork 4.0 نى تاللايمىز. بۇ ھەقتە كېيىنچە توختىلىمىز. 4- جەزملەش.

4. ئەمدىكى باسقۇچتا يېڭىدىن Web كۆزنىكىنى ئاكىرىمىز. رەسىمدە كۆرسىتىلگەندەك قىلىڭ.

  5. بۇ كۆزنەك سىزنىڭ ئاچماقچى بولغان تۈرىڭىزنىڭ، يەنى باشبەتنىڭ نامىنى بىلدۈرىدۇ. ASP.NET تىلىدا Default دېگىنى HTML, PHP لاردىكى Index بىلەن ئوخشاش. شۇڭا ئاشتا تۇرۇۋەرسۇن. جەزملەڭ.

  6. خۇش، ئەمدى بىز بىمالال كود يېزىش باسقۇچىغا كىردۇق. كود يېزىش باسقۇچى ئىككىنچى قىسىم دەرسلىكىمىزدە سۆزلىنىدۇ. قىزىقسىڭىز دىققىتىڭىز پىروگراممېرلار مۇنبىرىدە بولسۇن.
