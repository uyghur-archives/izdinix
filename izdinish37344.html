Windows 7 رەسمىي ھەقىقى نەشىرى ۋە قىسقىچە تەسىرات

مىكروسوفت تەرەپ MSDN تور بېكىتىدە Windows 7 نىڭ رەسمىي نەشىرىنى ئېلان قىلغىلى خېلى ئۇزاق بولغان بولسىمۇ، نۆۋەتتىكى ئەھۋالدا ئەڭ تېز سۈرئەتتە بۇ مەشغۇلات سىستېمىسى توغرىلىق مەلۇم چۈشەنچىگە ئىگە بولۇش سەل قىيىنغا توختىدى، شۇنداقتىمۇ بىر ئاماللار بىلەن قىسقىچە بىر يازما تەييارلاپ چىقتىم. ئىشىنىمەنكى يېڭى مەشغۇلات سىستېمىسى تاللاش ئېھتىياجىغا دۇچ كەلگەن، سىستېما ئالماشتۇرماقچى بولىۋاتقان ۋە Windows 7 نى ئىشلىتىش ھەققىدە ئىككىلىنىۋاتقان تورداشلار ئۈچۈن، مەزكۇر يازما چوقۇم ياخشى پايدىلىنىش ماتېرىيالى بولالايدۇ.
گەرچە Windows 7 ئالدىنقى يىلىلا ئېلان قىلىنىپ بولغان بولسىمۇ، پېقىر بۇ سېستىمىنى بۇ يىل بېشىدا ئىشلىتىشكە مۇۋەپپەق بولدۇم. تەدبىركار ئادەتتە سەل ھورۇنراق، شۇڭا ئالدىنقى قېتىم مەشغۇلات سېستىمىسىنى قاچىلىغان ۋاقتىم تەخمىنەن 2007-يىلىنىڭ ئاخىرىغۇ دەيمەن، ئىككى يىلغا يېقىن ئىشلەتكەن Vista نى كومپيۇتېرىمدىن ئۆچۈرۈپ تاشلاپ بىراقلا Windows 7 گە كۆچۈۋالدىم. Windows 7 گەرچە Vista غا قارىغاندا ئىككى يىل كېچىكىپ ئېلان قىلىنغان بولسىمۇ، ئەمما ئۇنىڭ قاتتىق دېتالغا بولغان ئەڭ تۆۋەن تەلىپى ئاساسەن ئۆزگەرمىگەن. ئوخشاش قاتتىق دېتال سەپلىمىسى شارائىتىدا Windows 7 نىڭ خىزمەت ئۈنۈمى Vista دىكىگە قارىغاندا راۋان بولغاچقا، ئەمىلىيەتتە قاتتىق دېتالغا بولغان تەلەپتە «ئۆرلىمەي تۆۋەنلەش» ئەھۋالى بار. مېنىڭ سېتىۋالغىلى 3 يىلدىن ئاشقان PC رىمدىمۇ Windows 7 نىڭ خىزمەت سۈرئىتى Vista دىكىگە قارىغاندا تېخىمۇ راۋان بولۇۋاتىدۇ، ئۇنىڭ ئۈستىگە قوزغىلىش سۈرئىتىمۇ زور دەرىجىدە قىسقىراپتۇ!

مېنىڭ PC رىمغا Win7 نىڭ بەرگەن سېستىما باھا نومۇرى ئارانلا 2.8 نومۇر، سىنكارتىغا بېرىلگەن نومۇر ئەڭ تۆۋەن، بىراق ئىنتايىن راۋان ئىشلىتىۋاتىمەن.
قاچىلاش جەھەتتە، مېنىڭ Pentium D تىپلىق مەركىزى بىر تەرەپ قىلغۇچ سەپلەنگەن كومپيۇتېرىمدا قاچىلاش ئېلىپ باردىم، ئالدى بىلەن بايقىغىنىم، قاچىلاش سۈرئىتى Vista غا قارىغاندا روشەن تېزلىشىپتۇ، گەرچە شۇ چاغدا ۋاقىت خاتىرىلەشنى ئۇنتۇپ قالغان بولساممۇ، ئەمما ئېسىمدە قېلىشىمچە، مەن نۇردېسكىنى سېلىپ قاچىلاشقا بۇيرۇق بېرىشكە باشلىغاندىن تارتىپ، سېستىما قاچىلىنىدىغان قاتتىق دېسكا رايونىنى تاللاپ بېرىش، قاچىلاش ھۆججەتلىرىنى كومپيۇتېرغا كۆچۈرۈش، پۈتكۈل قاچىلاش مەشغۇلات ئاخىرلىشىپ كومپيۇتېرنى قايتا قوزغىتىش ۋە سېستىما ئىشلەتكۈچىسى بەلگىلەشكىچە بولغان جەرياندا تەخمىنەن 20-30 مىنۇتلار ۋاقىت كەتتىغۇ دەيمەن. ئەمما بۇ يەردىكى مۇتلەق كۆپ قىسىم ۋاقىت، قاچىلاش ھۆججەتلىرىنى بىۋاستە ئوپتېك دېسكىدىن قاتتىق دېسكىغا كۆچۈرۈشكە كېتىپ قالدى، ناۋادا كومپيۇتېرنىڭ ئۆزىدە تۇرۇپ قاچىلاش ئېلىپ بارغان بولسام، ئىشىنىمەنكى سۈرئەت بۇنىڭدىن تېز بولۇشى مۇمكىن.
قوزغاتقۇچ پروگراممىسى جەھەتتە، يېڭى قاچىلىغان Windows 7 ئىشقىلىپ مېنىڭ قاتتىق دېتاللىرىمنىڭ قوزغاتقۇچلىرىنى ئاساسەن ئۆزىدە تەمىنلەپتۇ، پەقەت ئۈنكارتا ۋە TV كارتامنى تونۇيالمىدى. ئەمما مەن Vista ئىشلەتكەن چېغىمدا ئىشلەتكەن ئۈنكارتا قوزغاتقۇچى ۋە TV قوزغاتقۇچى ئارقىلىق بۇ قاتتىق دېتاللارنىمۇ نورمال خىزمەت ھالىتىگە ئەپكەلدىم، شۇنىڭ بىلەن سېستىما ئاساسەن مۇكەممەل خىزمەت ھالىتىگە ئۆتتى!
سۈرئەت جەھەتتە، ئوخشاش سەپلىمىدىكى كومپيۇتېرنىڭ Windows 7 دە ئېرىشكەن سېستىما باھا نومۇرى Vista غا قارىغاندا تۆۋەنرەك، بىراق ئەمەلىي خىزمەت سۈرئىتى Vista دىكىگە قارىغاندا خېلىلا تېز بىلىندى! ئاۋال بايقىغىنىم، قوزغىلىش سۈرئىتى خېلى ئوبدانلا تېزلىشىپتۇ، كومپيۇتېر ئويۇنلىرى جەھەتتە، مەن ئۆزۈم ئادەتتە كۆپ ئوينايدىغان PES2009 بىلەن Call Of Duty4 ھازىرقى زامان ئۇرۇشىنى قاچىلاپ كۆرۈپ باقتىم. ئويۇننىڭ راۋانلىقى Vista دىكىگە قارىغاندا سەل راۋان بىلىندى، بۇ ئىككى ئويۇننى ئادەتتە مەن كۆپرەك ئوينايمەن، ھەم ئاساسى كۆرۈنمە يۈز ۋە مەشغۇلاتلارغا پىششىق، شۇڭا تۇيغۇم خاتا ئەمەس دەپ ئويلايمەن…ۋاقىت مۇناسىۋىتى بىلەن تېخىمۇ تەپسىلىي سىناش نەتىجىسىنى تەمىنلەشكە ئامالسىزمەن، مەن پەقەت ئۆزۈمنىڭ ئاساسلىق كۆز قارىشىنى بىلدۈرۈپ ئۆتتۈم، ناۋادا خاتالىق بولسا تورداشلارنىڭ تەنقىد-پىكىرلەرنى بېرىشىنى قارشى ئالىمەن.
يۇمشاق دېتاللارنىڭ ماسلىشىشچانلىقى جەھەتتە، مېنىڭ قاچىلىغىنىم 32 بىتلىق Windows 7 مەشغۇلات سېستىمىسى، ئىشقىلىپ Vista دا ئىشلەتكەن يۇمشاق دېتاللىرىمنىڭ ھەممىسى دېگۈدەك نورمال خىزمەت قىلالىدى، ھەتتا Vista دا نورمال ئىشلىتەلمىگەن بىر قىسىم يۇمشاق دېتاللار، مەسىلەن، يېشىل ۋە كىچىك نەشىردىكى مەۋھۇم ئوپتىك دېسكا قورالى Win7 دا ناھايىتى نورمال خىزمەت قىلدى! تەدبىركار بلوگىنىڭ ئىلگىرىكى ئىخلاسمەن ئوقۇرمەنلىرىنىڭ بەلكىم يادىدا بولسا كېرەك، مەن ئادەتتە ھەپىلىشىدىغان يۇمشاق دېتاللار خېلى كۆپ، ئىشقىلىپ بۇ يۇمشاق دېتاللاردا Windows 7 دە نورمال خىزمەت قىلالمايدىغان ئەھۋالنى بايقىمىدىم، كۆڭۈل ئېچىشتىن تارتىپ كومپيۇتېر ئويۇنلىرى، ۋىرۇسخور، ئىشخانا ئاپتوماتلاشتۇرۇش قوراللىرى، پروگرامما تۈزۈش مۇھىتلىرىغىچە ھەممىسى Windows 7 بىلەن ناھايىتى ياخشى ماسلىشالايدىكەن(64 بىتلىق سېستىمىدىكى ئەھۋالنى بىلمەيمەن)! ئەلۋەتتە بۇنىڭ ئالدىنقى شەرتى، چوقۇم ئاشۇ يۇمشاق دېتاللارنىڭ ئەڭ يېڭى نەشىرى بولۇش، ناۋادا سىز جاھىللىق قىلىپ Win98 دەۋرىدىكى يۇمشاق دېتاللارنى ئىشلىتىمەن دەپ تۇرىۋالسىڭىز، بۇنىڭغا مېنىڭ ئامالىم يوق. 淘宝 ۋە ھەرقايسى بانكىلارنىڭ توردىك پۇل تاپشۇرۇش ئەھۋالىنى سىناق قىلىشىمغا ئاساسلانغاندا، پەقەت IE8 نىڭ تەڭشەك قىسمىدىكى “启用内存保护帮助减少联机攻击” دېگەن تۈر ئېلىۋېتىلسىلا، نورمال پۇل تۆلەش مەشغۇلاتى ئېلىپ بېرىشقا بولىدىكەن. بۇنىڭدىن باشقا، Nokia Pc Studio، كۆك چىش قاتارلىق ئۈسكۈنىلەرنىمۇ نورمال ئىشلىتەلىدىم.

ئىقتىدار جەھەتتە، Windows 7 ئاز بولمىغان يېڭى ئىقتىدارلارنى ئۆزىگە مۇجەسسەملەپتۇ! مەسىلەن يېڭى ۋەزىپە ستونى(قوللىنىشچانلىقى ئاجايىپ يۇقىرى! بۇ يېڭى ۋەزىپە ستونى ماڭا بەكلا يارىدى)، ۋەزىپە ستونى ۋە ھالەت ستونىدىكى ھەرقانداق سىنبەلگىنى خالىغانچە تارتىپ يۆتكەش، يېڭى قوشۇلغان «ئامبار» ئۇقۇمى، يېڭى ئۇسلۇب تاللاش ئىقتىدارى(تەگلىك رەسىمنى ئاپتوماتىك ئالماشتۇرغىلى بولىدۇ)، يېڭى UAC ئىقتىدارى، يەنە ئىزدەش ئىقتىدارى جەھەتتىكى يېڭى ئۆزگىرىشلەر قاتارلىقلارنى مەن ساناپ تۈگىتىپ بولالمايمەن. ئەلۋەتتە Windows 7 نىڭ كىشىگە بېرىدىغان تۇيغۇسىمۇ ئۇنچە بەك يېڭى دەپ كەتكىلى بولمايدۇ، چۈنكى نېمىلا دېمىگەن بىلەن ئۇنىڭ يادروسى Vista بىلەن ئوخشاش. بىراق تېخىچە XP ئىشلىتىۋاتقان تورداشلارغا نىسبەتەن، ئىشىنىمەنكى Windows 7 سىزنى يېپيېڭى بىر دۇنياغا باشلاپ كىرەلەيدۇ، XP دىن ئىبارەت 2001-يىلى ئېلان قىلىنغان 9 ياشتىن ئاشقان بۇ قېرى مەھسۇلات Vista نىڭ مەغلۇبىيىتى سەۋەبىدىن تا ھازىرغىچە خېلى كۆپ قىسىم ئابونتلارنىڭ ئىشەنچىسىگە ئېرىشىۋالغان بولسىمۇ، ھازىرقىدەك تېز ئۇچۇر دەۋرىدە ئىشىنىمەنكى تېزلا بازاردىن شاللىنىپ چىقىلىدۇ، مۇناسىۋەتلىك خەۋەرلەردە كۆرسىتىلىشىچە، بۇ يىل 7-ئايدىن باشلاپ مىكروسوفتمۇ Windows XP گە بولغان قوللاشنى توختىتىدۇ. بۇ دېگەنلىك، مىكروسوفت Windows XP گە نىسبەتەن ھېچقانداق يېڭىلاش ياكى بىخەتەرلىك يامىقى تارقاتمايدۇ، ھەرقايسى سېتىش نۇقتىلىرى تەلتۆكۈس توختىتىلىدۇ دېگەنلىك، شۇڭا مۇشۇ باھانىدە Windows 7 گە كۆچۈشمۇ بىر ئاقىلانە قاراش دەپ قارايمەن.

ئومۇمەن ئېيتقاندا Windows 7 مېنى ئىنتايىن رازى قىلغان بىر مەشغۇلات سېستىمىسى! ئەمما بازار باھاسى بەلكىم بىزلەرگە نىسبەتەن ئانچە تۆۋەن بولماسلىقى مۇمكىن، ناۋادا ئالما شىركىتىنىڭ Mac OSX Snow Leopard قا ئوخشاش 200 نەچچە سوم بولغان بولسا نەقەدەر ياخشى بولاتتى ھە؟ پەرىزىمچە بۇ باھاغا Windows 7 نىڭ Home Basic نەشىرىمۇ كەلمەيدىغۇ دەيمەن. ئىشقىلىپ خېلى كۆپ قىسىم تورداشلارغا نىسبەتەن، Win7 نىڭ رەسمى نەشىرىنى سېتىۋېلىپ ئىشلىتىش ئۇنچە ئوڭايمۇ ئەمەس.
بۇدا كۆپچىلىككە تەۋسىيە قىلىدىغىنىم، MSDN دا ئېلان قىلىنغان Windows 7 RTM رەسمىي ئەسلى نەشىرى بولۇپ، ھېچقانداق ئارتۇقچە ئەخلەتلەر ئارلاشتۇرۇلمىغان، دېسكىغا ئويۇپ ساقلاشتىكى ئەڭ ئېسىل تاللاش بولالايدىغان نۇسخىسى! ئەلۋەتتە، MSDN تەرەپتىن تارقىتىلغان بولغاچقا، مەزكۇر نۇسخا ھېچقانداق ئاكتىپلاش ياكى يېشىۋېتىش ياماقلىرىنى ئۆز ئىچىگە ئالمىغان، ناۋادا قاچىلىسىلا ئاپتوماتىك ئاكتىپلىنىدىغان نەشىرى لازىم دەيدىغان تورداشلار بولسا، كېلەر قېتىمقى يازمامدا بەلكىم مۇشۇ توغرىسىدا تەپسىلىي توختىلىشىم مۇمكىن. ناۋادا تەدبىركارغا ئوخشاش ئەسلى ھەقىقى نەشىردىكى يۇمشاق دېتاللارنىڭ مەستانىسى بولسىڭىز، مەزكۇر سېستىما سىزنىڭ ئەڭ ياخشى تاللىشىڭىز بولالايدۇ! ئىشىنىڭكى، ئەسلى ھەقىقىي نەشىردىكى مەشغۇلات سېستىمىسىلا مەڭگۈ مۇقىم خىزمەت قىلالايدۇ!
