Windows Phone ئۈچۈن پۈتۈك ئېلان قىلىندى

http://bbs.xjcfan.com/data/attachment/forum/201408/20/215354usv8jczqifww3ihh.jpg

     ئەپ نامى: پۈتۈك (Putuk)
     خىزمەت مۇھىتى: +Windows Phone 8.0
     نەشرى: 1.0.0.0
     سىغىمى: 810KB
     رولى: Windows Phone دا ئۇيغۇرچە يېزىق بىر تەرەپ قىلىش
     ئەپ چۈشەندۈرۈلۈشى:
     «پۈتۈك» پەقەت بىرلا مەقسەت – Windows Phone دا ئۇيغۇرچە يېزىق بىر تەرەپ قىلىش ئۈچۈن تۈزۈلگەن.
     «پۈتۈك» ناملىق بۇ ئەپ Windows Phone نىڭ 8.0 ياكى ئۇنىڭدىن يۇقىرى نەشرىلىرىدە ئىشلەيدۇ.
     «پۈتۈك» ناملىق بۇ ئەپ Windows Phone مۇھىتىدا ئۇيغۇرچە يېزىق بىر تەرەپ قىلىش، پۈتۈكنى ئۈندىدار (Wechat) دىكى دوستلارغا، دوستلار چەمبىرىكىگە، قىسقا ئۇچۇرغا ۋە ئېلخەتكە ھەمبەھرىلەش قاتارلىق ئىقتىدارلارغا ئىگە.
     تۇنجى نەشرىدىكى بۇ ئەپتە پۈتۈك پۈتكەندە نۇر بەلگىسى چىقماسلىق خاتالىقى بار. بۇ مەسىلىنى ھەل قىلىشقا ئۇزۇنراق ۋاقىت كېتىپ قالىدىغانلىقى ئۈچۈن، كېيىنكى نەشرىدە ھەل قىلىشنى ئويلىشىپ، تۇنجى نەشرىنى ئېلان قىلدۇق.
     بۇ ئەپتە بىز ھېس قىلمىغان ئۇنداق ياكى مۇنداق خاتالىقلار ساقلانغان بولۇشى تەبىئىي. كېيىنكى نەشرىلىرىنىڭ تېخىمۇ ئىقتىدارلىق بولۇشى ئۈچۈن، ئىشلىتىش جەريانىدا خاتالىق بايقىسىڭىز ياكى ئەپكە نىسبەتەن تەكلىپ-پىكىرلىرىڭىز بولسا، xjcfan@sina.com غا يوللاپ بەرگەيسىز. شۇنىڭ بىلەن بىللە ئەتراپىڭىزدىكى Windows Phone ئىشلىتىۋاتقان دوستلىرىڭىزغا تەۋسىيە قىلىشنى ئۇنتۇمىغايسىز.
     چۈشۈرۈش ئادرېسى 1: http://www.windowsphone.com/zh-cn/store/app/putuk/9f06eb3e-34f7-403a-a5be-833606e3266f
     چۈشۈرۈش ئادرېسى 2: Windows Phone ئەپ بازىرى http://www.windowsphone.com دىن Xjcfan ياكى Putuk دەپ ئىزدەپ، چۈشۈرۈۋالسىڭىزمۇ بولىدۇ.

     تۆۋەندە ئەپتىن كەسمە رەسىملەر بېرىلدى.

ئاخشام كەچ بولۇپ كېتىپ بىرىنچى قەۋەتتىن ئورۇن ئەپ قويسام، تەھرىرلەيدىغان يېرىنى تاپالمىدىم. :(

ئالدى بىلەن ئىككى تۆتتال ۋىنپون ھەۋەسكارلىرى ئۈچۈن جاپا چېكىۋاتقان بىر تۈركۈم ئاغىنىلەنىڭ بار ئىكەنلىكىنى بىلىپ يامان خوش بوپ كەتتىم، سىلەرگە ئاپىرىن.
ئىشلىتىش جەريانىدا ھېس قىلغانلىرىم.
1. بىۋاستە چېكىنىشكە بولمايدىكەن.
2. نۇر بەلگىسىنىڭ بولماسلىقى بەكمۇ مۇھىم ئەمەستەك.
3.ئۈندىدارغا ھەمبەھىرلەنگەندە «来自PUTUK» دېگەن قوشۇمچە چىقىۋالىدىكەن، بۇ يېرىنى پەقەت ياقتۇرمىدىم.
4.ھەرپ كۇنۇپكىرىلىرىنىڭ ئورۇنلاشتۇرۇلىشى سەل زىچتەك ھېس قىلدىم، لۇمىيا 820 ئىدى.
5.تەگلىك رەڭگىنى ئۆزىمىز تەڭشىۋالغىلى بولىدىغان ئىقتىدار قوشۇلسا، پۈتۈن بەت ئاق بولسا سەل بىئارامدەك ھېس قىلدىم.
