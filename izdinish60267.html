windows8 دە سامسۇڭ تېلېفوننى تونۇتقىلى بولماسلىق مەسىلىسىنى ھەل قىلىش

سىستېما نەشىرىنى ۋىندوۋس8 گە كۆتۈرگەندىن باشلاپ تېلېفوننى چاتسام USB ھالەتتە تونىماي جىلە قىلدى، توردىن سامسۇڭنىڭ USB قوزغاتقۇچىلىرىنى چۈشۈرۈپ قاچىلىساممۇ ھىچ بوممايدۇ، تور ئاختۇرۇپ تۆۋەندىكى ئۈچ خىل ئۇسۇلنى تاپتىم.
1. USB قوزغاتقۇچنى قاچىلاشتىن بۇرۇن ئوڭ چېكىپ 兼容性疑难解答 نى بېسىپ كۆرسەتمىگەن ئاساسەن مەشغۇلاتنى تاماملىغاندىن كېيىن، قاچىلىۋەتسەك بولىدۇ.
مېنىڭ مەسىلەم مۇشۇ ئامال بىلەن ھەل بولدى. تۆۋەندىكى ئىككى خىل ئامالنى ئىشلىتىپ باقمىدىم، شۇنداقتىمۇ قالدۇرۇپ قوياي دېدىم.

2. ئىجرا كۆزنىكىگە msconfig نى يېزىپ 系统配置 غا كىرگەندىن كېين 服务 غا كىرىپ، Device Install Sevice نى قوزغىتىۋىتىپ ئاندىن سىستېمىنى قايتا قوزغىتىپ USB قوزغاتقۇچنى قاچىلىساق تامام.

3. سامسۇڭنىڭ ئورگان تور بېكىتىگە چىقىپ Kies يۇمشاق دېتالىنى چۈشۈرۈپ قاچىلىساق بولىدۇ، Kies سامسۇڭ تەمىنلىگەن يۇمشاق دېتال بولۇپ، تېلېفوننى باشۇرۇشقا ئىشلىتىلىدۇ، ئادەتتە سامسۇڭنىڭ تېلېفونلىرىنىڭ USB قوزغاتقۇچلىرىنى ئۆز ئىچىگە ئالغان بولىدۇ.

PS:  مەن ئەسلى دەسلەپتە ئىككىنچى ئامالنى ئىشلەتمەكچى بولغان، بىراق Device Install Sevice نى ئوڭ چەكسەم قوزغىتىش بۇيرۇقى ئېچىلغىلى ئۇنىماي، بىرىنچى ئامالنى ئىشلىتىپ باقسام مەسىلە ھەل بولۇپ قالدى، ئاندىن قارىسام Device Install Sevice قوزغىتىغلىق ھالەتتە تۇرىدۇ. نىگىزىدىن ئېيتقاندا بىرىنچى بىلەن ئىككىنچى ئامال ئىككىسى بىر گەپ بولسا كېرەك.
