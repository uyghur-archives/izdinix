ﺳﺎﻏﻼﻣﻠﯩﻘﻘﺎ ﭘﺎﻳﺪﯨﻠﯩﻖ ﺑﯩﺮ ﻧﻪﭼﭽﻪ ﻣﯩﻨﯘﺗﻠﯘﻕ ﮬﻪﺭﯨﻜﻪﺗﻠﻪﺭ
1. ﺑﺎﺭﻣﺎﻗﻼﺭ ﺑﯩﻠﻪﻥ ﭼﺎﭼﻨﻰ ﺑﯩﺮ ﻣﯩﻨﯘﺕ ﺗﺎﺭﺍﭖ ﺑﯧﺮﯨﺶ : ﺋﯩﻜﻜﻰ ﻗﻮﻟﻨﻰ ﻳﯧﻴﯩﭗ، ﺑﺎﺭﻣﺎﻕ ﺑﯩﻠﻪﻥ ﭘﯩﺸﺎﻧﯩﺪﯨﻦ ﺋﺎﺭﻗﯩﻐﺎ ﻗﺎﺭﯨﺘﯩﭗ ﭼﺎﭼﻨﻰ ﺗﺎﺭﺍﭖ ﺑﻪﺭﮔﻪﻧﺪﻩ، ﺑﺎﺵ ﻗﯩﺴﻤﯩﺪﯨﻜﻰ ﻗﺎﻧﻨﯩﯔ ﺋﺎﻳﻠﯩﻨﯩﺸﯩﻨﻰ ﺗﯧﺰﻟﯩﺘﯩﭗ، ﻣﯧﯖﻪ ﻗﺎﻥ ﺗﻮﻣﯘﺭ ﻛﯧﺴﻪﻟﻠﯩﻜﻠﯩﺮﯨﻨﯩﯔ ﺋﺎﻟﺪﯨﻨﻰ ﺋﺎﻟﻐﯩﻠﻰ ﺑﻮﻟﯩﺪﯗ.
2. ﻗﯘﻻﻗﻨﻰ ﺑﯩﺮ ﻣﯩﻨﯘﺕ ﺋﯘﯞﯗﻻﭖ ﺑﯧﺮﯨﺶ : ﻗﯘﻻﻗﺘﺎ ﭘﯜﺗﯜﻥ ﺑﻪﺩﻩﻧﮕﻪ ﺗﯘﺗﯩﺸﯩﺪﯨﻐﺎﻥ ﺋﯘﺷﺸﺎﻕ ﺗﻮﻣﯘﺭ ﺗﻮﭘﻼﻧﻐﺎﻥ ﺑﻮﻟﯘﭖ، ﺑﺎﺵ ﺑﺎﺭﻣﺎﻕ ﺑﯩﻠﻪﻥ ﻗﯘﻻﻗﻼﺭﻧﻰ ﺋﯘﯞﯗﻻﭖ ﺑﻪﺭﮔﻪﻧﺪﻩ، ﺗﻮﻣﯘﺭﻻﺭ ﺭﺍﯞﺍﻧﻠﯩﺸﯩﭗ، ﻗﯘﻻﻕ ﻏﻮﯕﯘﻟﺪﺍﺵ، ﻛﯚﺯ ﺗﻮﺭﻟﯩﺸﯩﺶ، ﺋﯘﻧﯘﺗﻘﺎﻗﻠﯩﻖ ﻛﯧﺴﻪﻟﻠﯩﻜﻠﯩﺮﯨﻨﯩﯔ ﺋﺎﻟﺪﯨﻨﻰ ﺋﺎﻟﻐﯩﻠﻰ ﺑﻮﻟﯩﺪﯗ.
3. ﻛﯚﺯ ﻗﺎﺭﭼﯘﻗﯩﻨﻰ ﺑﯩﺮ ﻣﯩﻨﯘﺕ ﻣﯩﺪﯨﺮﻟﯩﺘﯩﭗ ﺑﯧﺮﯨﺶ : ﻛﯚﺯﻧﯩﯔ ﮔﯚﮬﯩﺮﯨﻨﻰ ﺋﯚﺯ ﺭﯦﺘﯩﻤﻰ ﺑﯩﻠﻪﻥ ﺋﻮﯓ - ﺳﻮﻟﻐﺎ ﻗﺎﺭﯨﺘﯩﭗ ﻣﯩﺪﯨﺮﻟﯩﺘﯩﭗ ﺑﻪﺭﮔﻪﻧﺪﻩ، ﻛﯚﺭﯛﺵ ﻗﯘﯞﯞﯨﺘﯩﻨﻰ ﺋﺎﺷﯘﺭﻏﯩﻠﻰ ﺑﻮﻟﯩﺪﯗ.
4. ﺑﯘﺭﯗﻧﻨﻰ ﺑﺎﺭﻣﺎﻕ ﺑﯩﻠﻪﻥ ﺑﯩﺮ ﻣﯩﻨﯘﺕ ﺳﯩﻼﭖ ﺑﯧﺮﯨﺶ : ﺋﯩﻜﻜﻰ ﺑﺎﺭﻣﺎﻕ ﺑﯩﻠﻪﻥ ﺑﯘﺭﯗﻧﻨﻰ ﻳﯘﻗﯩﺮﯨﺪﯨﻦ - ﺗﯚﯞﻩﻧﮕﻪ ﻗﺎﺭﯨﺘﯩﭗ ﺳﯩﻼﭖ ﺑﻪﺭﮔﻪﻧﺪﻩ، ﺋﻮﺭﻧﯩﻤﯩﺰﺩﯨﻦ ﺗﯘﺭﯗﭘﻼ ﺳﻮﻏﯘﻗﺘﯩﻦ ﭘﻪﻳﺪﺍ ﺑﻮﻟﻐﺎﻥ ﺑﯘﺭﯗﻥ ﭘﯜﺗﯜﺵ، ﺑﯘﺭﯗﻧﺪﯨﻦ ﺳﯘ ﻛﯧﻠﯩﺶ ﯞﻩ ﺯﯗﻛﺎﻣﺪﺍﭖ ﻗﯧﻠﯩﺸﻨﯩﯔ ﺋﺎﻟﺪﯨﻨﻰ ﺋﺎﻟﻐﯩﻠﻰ ﺑﻮﻟﯩﺪﯗ.
5. ﺑﯩﺮ ﻣﯩﻨﯘﺕ ﭼﯩﺸﯩﻨﻰ ﭼﺎﻛﯩﻠﺪﯨﺘﯩﭗ، ﺗﯩﻠﻨﻰ ﮬﻪﺭﯨﻜﻪﺗﻠﻪﻧﺪﯛﺭﯛﺵ : ﭼﯩﺸﻨﻰ ﻳﯧﻨﯩﻚ ﭼﺎﻛﯩﻠﺪﯨﺘﯩﭗ ﺑﻪﺭﮔﻪﻧﺪﻩ، ﭼﯩﺶ ﻳﯩﻠﺘﯩﺰﻯ ﯞﻩ ﭼﯩﺶ ﺋﺎﺭﯨﻠﯩﻘﻠﯩﺮﯨﺪﯨﻜﻰ ﻗﺎﻧﻨﻰ ﺟﺎﻧﻼﻧﺪﯗﺭﯗﭖ، ﭼﯩﺸﻨﻰ ﺳﺎﻏﻼﻣﻼﺷﺘﯘﺭﻏﯩﻠﻰ ﺑﻮﻟﯩﺪﯗ. ﺗﯩﻠﻨﻰ ﮬﻪﺭﯨﻜﻪﺗﻪﻧﺪﯛﺭﯛﭖ ﺑﻪﺭﮔﻪﻧﺪﻩ، ﺗﯩﻠﻨﯩﯔ ﮬﻪﺭﯨﻜﻪﺗﻠﯩﻨﯩﺸﯩﻨﻰ ﺭﺍﯞﺍﻧﻼﺷﺘﯘﺭﯗﭖ، ﺗﯩﻞ ﺳﯧﺰﯨﻤﯩﻨﻰ ﻛﯜﭼﻪﻳﺘﻜﯩﻠﻰ ﺑﻮﻟﯩﺪﯗ.
6. ﻛﯩﻨﺪﯨﻜﻨﻰ ﺑﯩﺮ ﻣﯩﻨﯘﺕ ﻳﻪﯕﮕﯩﻞ ﺳﯩﻼﭖ ﺑﯧﺮﯨﺶ : ﺋﯩﻜﻜﻰ ﺋﺎﻟﻘﺎﻥ ﺑﯩﻠﻪﻥ ﻛﯩﻨﺪﯨﻜﻨﻰ ﻳﻪﯕﮕﯩﻞ ﺳﯩﻼﭖ ﺑﻪﺭﮔﻪﻧﺪﻩ، ﺷﺎﻣﺎﻝ ﺗﯧﮕﯩﺸﻨﯩﯔ ﺋﺎﻟﺪﯨﻨﻰ ﺋﺎﻟﻐﯩﻠﻰ ﺑﻮﻟﯩﺪﯗ.
7. ﺑﯩﺮ ﻣﯩﻨﯘﺕ ﻗﻮﺭﺳﺎﻗﻨﻰ ﺋﯩﭽﯩﮕﻪ ﺗﺎﺭﺗﯩﭗ، ﺳﻮﯕﯩﻨﻰ ﭼﯩﻘﯩﺮﯨﺶ : ﻗﻮﺭﺳﺎﻗﻨﻰ ﻗﺎﻳﺘﺎ - ﻗﺎﻳﺘﺎ ﺋﯩﭽﯩﮕﻪ ﺗﺎﺭﺗﯩﭗ ﺑﻪﺭﮔﻪﻧﺪﻩ، ﻗﺎﻥ ﺋﺎﻳﻠﯩﻨﯩﺸﻨﻰ ﺋﯩﻠﮕﯩﺮﻯ ﺳﯜﺭﯛﭖ، ﺑﻮﯞﺍﺳﯩﺮ ﻛﯧﺴﻪﻟﻠﯩﻜﯩﻨﯩﯔ ﭘﻪﻳﺪﺍ ﺑﻮﻟﯘﺷﯩﺪﯨﻦ ﺳﺎﻗﻼﻧﻐﯩﻠﻰ ﺑﻮﻟﯩﺪﯗ.
8. ﭘﯘﺕ - ﻗﻮﻟﻼﺭﻧﻰ ﺑﯩﺮ ﻣﯩﻨﯘﺕ ﺳﯘﻧﯩﯟﯦﻠﯩﺶ : ﺑﯘﻧﺪﺍﻕ ﻗﯩﻠﻐﺎﻧﺪﺍ، ﻗﺎﻧﻨﯩﯔ ﺋﺎﻳﻠﯩﻨﯩﺸﯩﻨﻰ ﺗﯧﺰﻟﯩﺸﯩﭗ، ﻳﯜﺭﻩﻙ، ﻣﯧﯖﯩﮕﻪ ﻳﯧﺘﻪﺭﻟﯩﻚ ﺋﻮﻛﺴﯩﮕﯧﻦ ﯞﻩ ﻗﺎﻥ ﻳﻪﺗﻜﯜﺯﯛﭖ ﺑﯧﺮﯨﻠﯩﺪﯗ. ﺑﯘﻧﯩﯔ ﺑﯩﻠﻪﻥ ﺋﺎﺳﺘﺎ ﯞﻩ ﺟﯩﺪﺩﯨﻲ ﺧﺎﺭﺍﻛﺘﯧﺮﻟﯩﻚ ﻳﯜﺭﻩﻙ، ﻣﯧﯖﻪ ﻛﯧﺴﻪﻟﻠﯩﻜﻠﯩﺮﯨﻨﯩﯔ ﺋﺎﻟﺪﯨﻨﻰ ﺋﺎﻟﻐﯩﻠﻰ، ﻗﻮﻝ - ﭘﯘﺗﻼﺭﺩﯨﻜﻰ ﺋﯘﺷﺸﺎﻕ ﺑﻮﻏﯘﻣﻼﺭﻧﯩﯔ ﮬﻪﺭﯨﻜﻪﺗﻠﯩﻨﯩﺸﯩﻨﻰ ﺋﺎﺷﯘﺭﻏﯩﻠﻰ ﺑﻮﻟﯩﺪﯗ.
