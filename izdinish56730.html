شىرئەلىجان CBAچۇلپانلار مۇسابىقىسىگە تاللاندى

   
  ۋاڭيى تەنتەربىيە تورىنىڭ 14-يانۋار بەرگەن خەۋىرى؛ 2014-يىللىق CBAچولپانلار ھەپتىلىگى 2014-يىلى 1-ئاينىڭ 18-19-كۈنى بىيجىڭ ۋەن شىدا تەنتەربىيە مەرگىزىدە ئۆتكۈزىلىدۇ. بۈگۈن، جۇڭگۇ ۋاسكىتبولچىلار جەمئىيىتى  2014-يىللىق CBAچولپانلار مۇسابىقىسى،  يىڭى چولپانلار ۋاسكېتبول مۇسابىقىسى، ئۇچ نۇمۇرلۇق توپ چۇشۇرۇش، ماھارەت رىقابەت مۇسابىقىسى، گارغا توپنى كۆمتۈرۈپ بېسىپ كىرگۈزۈش مۇسابىقىسى قاتارلىق تۈرلەردە مۇسابىقىگە قاتنىشىدىغان تەنھەركەتچىلەرنىڭ ئىسىملىگىنى ئىلان قىلدى. چولپانلار مۇسابىقىسى ئاساسلىق ئەزا بۇلۇپ مەيدانغا چىقىدىغانلار تىزىملىگىدە،  شىمال رايۇندىن ماربول ئەڭ ئاخىرى بۇ قىتىملىق CBAچولپانلار ھەپتىلىگىگە قاتنىشىشدىن مەھرۇم بولغاندىن كىيىن، شىنجاڭ كوماندىسىنىڭ چەتئەللىك ياردەمچى ئەزاسى خاندېرسېن ئۇنىڭ ئورنىغا سەپلەندى.
   
2014-يىللىق CBAچولپانلار ھەپتىلىگى تۇلۇق ئىسىملىگى تۈۋەندىكىچە؛
  شىمال رايۇن چولپانلار كۇماندىسى؛
  باش تىرىنىر  سۈن ۋەنجۈن ( شىنجاڭ)  ياردەمچى تىرىنىر مېن لۇ لېي ( بىيجىڭ)
  ئاساسلىق ئەزالار؛ خەن دىجى، لى شىياۋ شۇ ( لىياۋنىڭ) ،  دىڭ يى يۈخاۋ  ( شەندۇڭ )،  سۈن يۈ  ( بىيجىڭ)، خاندېرسېن ( شىنجاڭ ).
  زاپاس ئەزالار؛ شىرئەلىجان ( شىنجاڭ)، جاڭ بىياۋ ( جىلىن)، مورىس ( بىيجىڭ )،  لى گېن ( بىيجىڭ)،  دۇەن جاڭ فىڭ ( شەنشى)، جاڭ نەن( تىيەنجىن)، خادادى ( سىچۋەن ).
  جەنۇپ رايۇن چولپانلار كۇماندىسى؛
باش تىرىنىر يۇناس ( گۇاڭدۇڭ).
ئاساسلىق ئەزالار؛  ۋاڭ جىجى (1-ئاۋغۇسىت)، جۇفاڭيۈ، ۋاڭ شىپىڭ، يىجىيەنلەن ( گۇاڭدۇڭ)،  لىن جېجى ( گۇاڭشىيا).
زاپاس ئەزالار؛ كۇشى دۇبى ( شاڭخەي )، گۇ چۇەن ( دۇڭگۇەن )، خوشۇ پىڭ ( جىياڭسۇ )، گىپ سۇن ( گۇاڭشىيا )،  دېڭ جىڭ خۇڭ ( جىجىياڭ)،  ۋاڭ جىلىن ( فۇجىيەن)، شۇ جۇڭخاۋ (1-ئاۋغۇسىت).
  يىڭى چولپانلار ۋاسكېتبول مۇسابىقە تېزىملىكى؛
شىمالىي رايون يىڭى چولپانلار كوماندىسى:
كوماندا ئەزالىرى:  سۈي فۇمېڭ ( جىلىن )، جاڭ ياۋ شىڭ ( چىڭداۋ)، جاڭ شۈ ۋىي، لىيۇ جى شۇەن، يۇەن شى ( لياۋنىڭ)، شى چىڭ خاۋ ( شەندۇڭ )، جىيا چىڭ ( سىچۋەن )، تيەن گۈيسېن، چەي چاڭيى( سەنشى)، سۈن مېڭ ياڭ ( شىنجاڭ).
جەنۇبىي رايون يىڭى چولپانلار كوماندىسى:
كوماندا ئەزالىرى:  گاۋ شاڭ، ( گۇاڭدۇڭ)،  لو خەنچېن ( دۇڭۋەن)، ۋۇ چەن ( جېجىياڭ )، لى جىڭ لۇڭ ( گۇاڭشيا)، تىيەن زىشىياڭ ( بىرىنچى ئاۋغۇسىت)، جۇشۇ خاڭ،  جۈ مىڭشىن( فۇشەن).
ئۇچ نۇمۇرلۇق تۇپ سىلىش مۇسابىقىسىگە قاتنىشىدىغان چۇلپانلار تىزىملىگى؛
شىرئەلىجان مۇختەر ( شىنجاڭ، ئۇيغۇر )، خې تىيەن جۇ ( لىياۋنېڭ )،  گۇ چۇەن ( دۇڭگۇەن)، دۇەن جاڭ فىڭ ( شەنشى)، جاڭ تەيلۇڭ( فۇجىيەن). مىڭ شاڭلۇڭ ( تىيەنجىن).
ماھارەت رىقابەت مۇسابىقە تېزىملىكى؛
ۋاڭ زى رۈي ( گۇاڭشىيا)، جاڭ بو ( بىرىنچى ئاۋغۇست)، لو خەنچېن ( دۇڭۋەن)،  جاۋ شۇ ۋېي ( لياۋنىڭ).
گارغا توپنى كۆمتۈرۈپ بېسىپ كىرگۈزۈش مۇسابىقە تېزىملىكى؛
جەي يى ( شاڭخەي).  تيەن گۈيسېن، شىڭ جىچاڭ ( سەنشى)، ماك فېيبا ( سىچۋەن)، يۇ خەن سۈن ( گۇاڭشيا).  
  ئالدىنقى چولپانلار ھەپتىلىگىنىڭ ئۈچ نۇمۇرلۇق توپ چۈشۈرۇش چىمپىيۇنى،  ئۇيغۇر ۋاسكىتبۇل چولپىنمىز شىرئەلىجان مۇختەر ئىككى تۈردىكى مۇسابىقىگە قاتنىشىشقا تاللانغان.  بىز ئۇنىڭ بۇ قىتىمقى چۇلپانلار مۇسابىقىسىدىمۇ يەنە شەرەپ قۇچىشىنى ئۇمۇت قىلىمىز.
