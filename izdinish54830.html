ئەڭ ياخشى نۇسخا ئالغۇ ( فۇيىنجى ) نى تاللىشىپ بىرىڭلار؟

مەن 2 يىل بۇرۇن  SHARP  AR- 4818S   دىن بىرنى خوتەندىن  سىتىۋالغان بولۇپ،  بۇرۇن  نۇسخا ئالغۇ ( فۇيىنجى ) دىن كۆپ چۈشەنچەم بولمىغاچقا، ياخشى دەپلا سىتىۋالغان.  ئەمدى چاتىقى كۆرۆلۈشكە باشلىدى.  رىمۇنىتچىلار گۇمىدا ئوڭشىيالمىدى ( بۇزۇلغان ئۇششاق زاپچاسلىرى يوق بولۇپ قالدى )



بۇزۇلغان ئورنى:  شۈگۇ، دىڭيىڭچى، جۇدىيەنچى قاتارلىقلارنىڭ ھەرىكەت ئۇزىتىش ئايلانما چاقى ئۇپراپ، كۈچىيەلمەس بولۇپ قاپتۇ، شۇنىڭ بىلەن بۇيرۇق بەرگەندە، قەغەر شۈگۈ ئوتتورىدا ۋ دىڭيىڭچىنىڭ ( تەسۋىرنى قىزىتىپ مۇقىملاشتۇرغۇچ ئوخشايدۇ )  ئوتتورىدا قىىلىپ قالىدۇ.



شۇڭا يىڭىدىن ئىسىلراق، زاۋۇت تەرەپتىن يىڭىدىن تۈركۈملەپ بازارغا چىقىرىۋاتقان نۇسخا ئالغۇ ( فۇيىنجى ) بولسا تەۋسىيە قىلساڭلار؟



بولسا جىڭ MADEIN JAPAN   ( ياپۇننىڭ ) بولسا،  ئىزدىنىشتىكى قىرىنداشلىرىمدى  ئۈمىتۋارمەن.



مەن بىر قىتىمدا ئىككى يۈزنى بىرلا بۇيرۇق بىلەن چىقىرىدىغان نۇسخا ئالغۇ ( فۇيىنجى ) كۆرگەن ئىدىم.  ماركىسىغا دىققەت قىلماپتىكەنمەن، ئىسىمدە قىلىشىچە X-          بىر ھەرىپ باردەك قىلغان.



يەنى:  بىر قىتىمدا قوش نۇسخىلىيالايدىغان ئىقتىدارى بار، چىدامچانلىقى يۇقىرى، باھاسى مۇئاپىق بولغانلىرى بولسا تىخىمۇ ياخشى، مەن ۋە باشقا ئىھتىياجلىق دوسلارنىڭ پايدىلىنىشى ئۈچۈن، تىخىمۇ مول بىلىمىڭلارنى ئىزدىنىشقا چىقىرىشىڭلارنى ئۈمۈد قىلىمەن.
