Excel ھۆججەتنى Oracle غا ئەكىرىش

ماڭا كۆپىنچە Excel ھۆججەتنى Oracle ساندانغا كىرگۈزۈش ئېھتىياجى تۇغۇلۇپ قالىدۇ. ئادەتتە مېنىڭ ئۇسۇلۇم مانداق:


1. ئاۋال Excel نى Access كە كىرگۈزىۋالىمەن، مىكروسوفىتنىڭ بۇ ئىككى قوشمىقى بىر بىرىنى بەك ياخشى قوللايدۇ.


2. MySQL نىڭ Access ئۇچۇرىنى ئەكىرىش ئۇسۇلىدىن پايدىلىنىپ، بۇ ئۇچۇرلارنى MySQL غا ئەكىرىۋالىمەن.


3. MySQL نى Oracle غا ئايلاندۇرۇش قورالىدىن پايدىلىنىپ، MySQL دىكى ئۇچۇرنى Oracle غا ئەكىرىمەن. لېكىن چاتاق يېرى مەن ئىشلىتىۋاتقان ئالماشتۇرۇش قورالىنىڭ سىناق ئىشلىتىش ۋاقتى توشۇپ، پەقەت بەش قۇرلا كىرگۈزۈپ بېرەلەيدىغان بولۇپ قاپتۇ. باشقا ئوخشاش يۇمشاق دېتاللارنى ئىزدەپ باققان بولساممۇ، كۆپىنچىسى ھەقلىقكەن، ئۇنىڭ ئۈستىگە ئۇيغۇرچە ئۇچۇرلارنى ياخشى بىر تەرەپ قىلالمايدىكەن. شۇنىڭ بىلەن ئامالسىز مانداق قىلدىم:


1. MySQL دىكى مەن Oracle غا ئەكىرمەكچى بولغان جەدۋەلنى SQL ھۆججەتكە ئايلاندۇرىۋالدىم، بۇ ھۆججەتنىڭ ئىچىدە ھەر بىر قۇرنىڭ Insert جۈملىسى باركەن(ئەگەر Oracle غا كۆپ قۇرلۇق كىرگۈزۈشنىڭ ئامالى بولسا مەسىلە سەل ئاسانلىشاتتى)، بۇ جۈملىلەرنىڭ ھەممىسىنى UltiraEdit قا ئەكىرىپ، بىرە قۇر ياساپ جابدۇغاندىن كېيىن، Oracle نىڭ SQL ئىجرا قىلىش كۆزنىكىدە ئىجرا قىلىپ كىرگۈزىۋالدىم.


لېكىن كىرگۈزىدىغان ئۇچۇرنىڭ مىقدارى تېخىمۇ كۆپ بولغان ئەھۋالدا، بۇ لەقۋا ئۇسۇل بەك ئاۋارچىلىق بولغۇدەك. شۇنىڭ بىلەن Excel نى Oracle غا كىرگۈزىدىغان بىرەر ئۈنۈملۈك ئۇسۇل بارمىكىن دەپ بۇ تېمىنى يېزېپ سالدىم، ھە ھۇ دىيىشسەڭلار!
