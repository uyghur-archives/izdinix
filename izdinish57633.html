#C پروگرامما لايىھىلەش مۇھىتى ھازىرلارش

بۈگۈنكى يازمىمىزدا #C پروگرامما لايىھىلەش مۇھىتى ھازىرلارش توغرىسىدا توختىلىمىز.
#C تىلىدا پروگرامما لايىھىلەش ئۈچۈن ئادەتتە بىز مىكروسوفت شىركىتىنىڭ توپلاشتۇرۇلغان ئېچىش مۇھىتى (IDE) بولغان Visual Sutio يۇمشاق دېتالىنى ئىشلىتىمىز، بۇ جۈملىدىكى ئادەتتە دېگەن سۆزگە دىققەت قىلىڭلار، بۇنداق دېيىشمچە باشقا تاللاشلارمۇ بار دېگەن گەپ، مەسىلەن خاتىرە دەپتەر يۇمشاق دېتالىدا يازساقمۇ بولىدۇ، ھى ھى، ئەلۋەتتە ئۇ بەك قولايسىز، ئۇندىن باشقا بىر قەدەر ياخشى بولغانلىرىدىن ئوچۇق كودلۇق پۈتۈنلەي #C تىلىدا يېزىپ چىقىلغان ئېچىش مۇھىتى SharpDevelop نى ئىشلەتسەكمۇ بولىدۇ. بۇنىڭ ئورگان تەرەپ بېتى تۆۋەندە بېرىلدى، قىزىققۇچىلار چۈشۈرۈپ قاچىلاپ سىناپ باقساڭلا بولىدۇ.
http://www.icsharpcode.net/OpenSource/SD/Default.aspx
ئەمما بىز بۇگۈن تونۇشتۇرماقچى بولغىنى مىكروسوفت شىركىتىنىڭ ئېچىش مۇھىتى Visual Sutio نى چۈشۈرۈش ۋە قاچىلاش جەريانىدۇر.
بىزنىڭ ئىشلىتىشىمىز ئۈچۈن Visual Sutio نىڭ ئىككى خىل شەكلى بار، بىرى كەسپىي نۇسقىسى بۇ ھەم ھەقلىق نۇسقىسىدۇر، يەنە بىرى ئۆگەنگۈچىلەر ئۈچۈن تەمىنلىگەن ئېچىش مۇھىتى ئىقتىدارى قىسقارتىلغان نۇسقىسى بولۇپ، بۇ نۇسقىسىنى مىكروسوفت تورىدىن بىۋاستە چۈشۈرۈپ ھەقسىز ئىشلىتىشكە بولىدۇ، بىز بۇ يەردە كەسپىي نۇسقىسىنى قەيەردىن قانداق چۈشۈرۈپ قانداق يېشىپ دېگەندەك مەزمۇنلارنى سۆزلىمەيمىز، يەنى ھەقسىز نۇسقىسىنى چۈشۈرۈش ۋە قاچىلاش ئۇسۇلىنى تونۇشتۇرىمىز.
Visual Sutio نىڭ نەشر مەسىلىسىگە كەلسەك ھازىرقى ئەڭ يېڭى نەشرى 2013 بولۇپ، بىز سۆزلەۋاتقان دەرسنى 2010 مۇھىتىدا ئېلىپ بېرىۋاتقان بولغاچقا 2010 نەشرىنى چۈشۈرۈپ قاچىلاشنى تونۇشتۇرىمىز، 2013 كە قىزىقساڭلار ئوخشاش ئۇسۇل بىلەن قاچىلاپ ئىشلىتىپ باقساڭلا بولىدۇ.
1. مىكروسوفت ئورگان تورىغا كىرىپ Visual Studio 2010 Express Download دەپ ئىزدىسەڭلا چۈشۈرۈش بېتى چىقىدۇ، ياكى بىۋاستە مۇشۇ ئادرېستىنلا چۈشۈرسەڭلا بولىدۇ:
http://www.visualstudio.com/downloads/download-visual-studio-vs#DownloadFamilies_4
يۇقۇرىدىكى بەتنى ئاچساڭلا تۆۋەندىكى رەسىمدىكىدەك مەزمۇنلار كۆرىنىدۇ:
http://www.koklen.net/wp-content/uploads/2014/03/1.jpgئەگەر تور شۈرئىتىڭلار ياخشى بولسا 2010 Visual C# Express ياكى Visual Web Developer 2010 Express نى بېسىپ 立即安装 دېگەن يەرنى بېسىپ چۈشۈرسەڭلا بولىدۇ، بۇنداق بولغان ئاۋال كىچىك بىر دېتال چۈشىدۇ، شۇنى قوزغاتساڭلا ئاندىن رەسمىي يوشۇندا بىز تاللىغان نۇسقىدىكى VS قورالى قاچىلىنىدۇ.
http://www.koklen.net/wp-content/uploads/2014/03/2.jpgمېنىڭ تەۋسىيە قىدىغىنىم يەنىلا Visual Studio 2010 Express All-In-One بولۇپ، بۇنى تاللاپ چۈشۈرسەڭلا بارلىق يۇقۇردىكى تىزىملىكتىكى ھەممىسىنى ئۆز ئىچىگە ئالغان مەھۇم دىسكا ھۆججىتى چۈشىدۇ، بۇنى چۈشۈرۈپ بولۇپ قاچىلىغاندا قايسىنى ئىشلەتسەڭلا شۇنى قاچىلاپ ئىشلەتسەڭلا بولىدۇ.
http://www.koklen.net/wp-content/uploads/2014/03/3.jpgبۇ ھۆججەت تەخمىنەن 1.76GB ئەتراپىدا بولۇپ، چۈشۈرۈپ بولۇپ مەلۇم بىر خىل مەھۇم نۇر دىسكا ھاسىل قىلىش دېتالى ئارقىلىق نۇر دىسكا قوزغاتقۇچىسى ھاسىل قىلىۋېلىپ شۇنىڭ ئىچىگە تىقىپ قوزغىتىپ قاچىلىساڭلار مۇ ياكى شۇ ھۆججەتنى بىۋاستە پرىستىن يېشىش دېتالى ئارقىلىق يېشىپ setup قوزغىتىپ قاچىلاشنى باشلىساڭلارمۇ بولىدۇ.
بىز بۇ يەردە DAEMON Tools دىن پايدىلىنىپ مەھۇم نۇر دىسكا قوزغاتقۇچىسى ئارقىلىق قاچىلاشنى تونۇشتۇرىمىز.
DAEMON Tools توردىن چۈشۈرسەڭلا بولىدۇ ناھايىتى كىچىك قولاي دېتال، قاچىلاپ قوزغىتىپ تۆۋەندىكى رەسىمدىكىدەك مەشغۇلات قىلىپ ئالدىنقى قەدەمدە چۈشۈرۈپ تەييار قىلىۋالغان  Visual Studio 2010 Express All-In-One نىڭ مەھۇم نۇر دىكسا ھۆججىتى (iso ھۆججىتىنى) نى تاللاپ بەرسەڭلار بولىدۇ
http://www.koklen.net/wp-content/uploads/2014/03/4.jpg
، ئاندىن ئادەتتە نۇر دىسكىنى ئاچقان ئۇسۇل بويىچە قوش چىكىپ ئاچساڭلا ياكى ئىچىگە كىرىپ setup نى قوش چىكىپ

http://www.koklen.net/wp-content/uploads/2014/03/6.jpgيۇقۇرىدىكى بىر قانچە تۈرنىڭ ئىچىدىن Visual C# 2010 学习版 دېگەننى تۈرنى تاللاپ قاچىلايمىز، بۇ #C  تىلىدا Windows پروگراممىسى يېزىش ئۈچۈن ئىشلىتىدىغان نۇسقىسى بولۇپ،باشقىلىرىنى ئىسمىغا قارىساڭلارغا بىلىسىلە، بۇنىڭ ئىچىدىكى Visual Web Developer دېگىنى بولسا مەخسۇس تور پروگراممىسى يەنى تور بەت لايھىلەش ئۈچۈن ئىشلىتىدىغان نۇسقىىدۇر، Visual Studio نىڭ كەسپىي نۇسقىسىدا بۇنداق ئايرىۋىتىلمىگەن، بۇ نۇسقىسى ئۆگەنگۈچى ۋە سەخسلەر ئۈچۈن ھەقسىز تەمىنلەنگەن بولغاچ مۇشۇنداق ئىقتىدار ۋە رولىغا قاراپ ئايرىۋىتىلگەن، ئەمما بىزنىڭ ئادەتتىكى ئھتىياجىمىزدىن پۈتۈنلەي چىقالايدۇ.
