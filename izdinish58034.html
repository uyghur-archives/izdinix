Ubuntu دا java مۇھىتى ھازىرلاش

1.ئاۋۋال jdk نى چۈشۈرىمىز
http://www.oracle.com/technetwork/java/javase/downloads/jdk7-downloads-1880260.html
سىستېما نەشىرىگە ئاساسەن 32 ياكى 64 لىكنى تاللاپ چۈشۈرىمىز.
كىڭەيتىلگەن نامى tag.gz نى تاللاپ چۈشۈرۈڭ.



2. سىستېمىنىڭ Home ھۆججەت قىسقۇچىنىڭ ئېچىگە يېڭى ھۆججەت قىسقۇچتىن بىرنى قۇرۇڭ، ئىسمىنى java دەپ بىكىتىڭ.




3.چۈشۈرگەن djk نى java دېگەن ھۆججەت قىسقۇچنىڭ ئىچىگە كۆچۈرۈڭ.




4. مۇھىتنى تەڭشەڭ.sudo gedit /etc/environmentexport JAVA_HOME=/home/halil/java/jdk1.7.0_55
export JRE_HOME=/home/halil/java/jdk1.7.0_55/jre
export CLASSPATH=.:$CLASSPATH:$JAVA_HOME/lib:$JAVA_HOME/jre/libsudo gedit /etc/profileexport JAVA_HOME=/home/halil/java/jdk1.7.0_55
export JRE_HOME=//home/halil/java/jdk1.7.0_55/jre
export CLASSPATH=.:$CLASSPATH:$JAVA_HOME/lib:$JAVA_HOME/jre/lib
export PATH=$JAVA_HOME/bin:$JAVA_HOME/jre/bin:$PATH:$HOME/bin

مانا مۇشۇنداق ئاسان. شۇنىڭ بىلەن java مۇھىتى تەييارلاندى.

ئەمدى كىچىككىنە سىناق

public class Hello{
    public static void main(String args[]){
      System.out.println("hello wrold");
    }
}

دېمەك مۇۋاپىقىيەتلىك بولدى.java -version
