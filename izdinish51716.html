مەن نېمە كويدا، سىلەر نېمە كويدا؟



ئالىم ئەھەت


Windows 8 نىڭ ئۇيغۇرچە تىل بولىقى ئورگان تەرەپتىن ئېلان قىلىنىپ ھەممەيلەن خۇشاللىققا چۆمگەن بۇ كۈنلەردە، ھەرقايسى تور بېكەتلەردىن تورداشلارنىڭ بۇ ھەقتە بەس-بەستە يازغان تېما ئىنكاسلىرىنى كۆرۈپ چىقتىم. ئەمما بەزى تور بېكەتلەرنى كۆرۈپ ھەيران قالدىم. مەن گەپنى سەل باشتىن باشلاي.


Windows 7 خەنزۇچىدىن باشقا يەنە جۇڭگودىكى 7 مىللەتنىڭ تىل-يېزىقىنى بىر تەرەپ قىلالايتتى. ئۇلار زاڭزۇچە (藏语)، مۇڭغۇلچە (蒙古语)، يىزۇچە (彝语)، ئۇيغۇرچە (维吾尔语)، ئۇنىڭدىن باشقا يەنە 德宏傣文, 西双版纳新傣文, 八思巴文 لاردىن ئىبارەت. Microsoft نىڭ ئۇيغۇرچە تىل بولىقىغا قانداق كۆڭۈل بۆلۈۋاتقانلىقىنى مەن بەش قولدەك ئوبدان بىلىمەن، ئورگان تەرەپ تىل بولىقىنى چىقىرىمەن دېسە، قائىدە بويىچە Windows قا كىرگەن تەرتىپ بويىچە ئويلىشىدۇ. ھېچبولمىغاندا جۇڭگودا خەنزۇچىدىن باشقا يەنە بىرەر تىلنى تاللايمەن دېسە زاڭزۇچە (تىبەت) ياكى مۇڭغۇلچىنى تاللىغان بولاتتى. زاڭزۇچە ۋە مۇڭغۇلچە يۇمشاق دېتاللارنىڭ تېخنىكىسىمۇ خېلە پىشىپ يېتىلگەن، بەزى دېتاللىرىنىڭ تېخنىكىسى تېخى بىزنىڭ ئالدىمىزدا. ئەلۋەتتە دۆلەتمۇ ئۇيغۇرچە يۇمشاق دېتاللارنىڭ تەرەققىياتى ئۈچۈن نۇرغۇن مەبلەغ سېلىۋاتىدۇ. ئەمما مەيلى قانداقلا بولمىسۇن بۇ قېتىم ئورگان تەرەپ ئۇيغۇر تىلىنى ئالدىدا ئويلىشىپ، ئىلتىماسنى تەستىقلىدى. كېيىن چوقۇم زاڭزۇچە (藏语) ياكى مۇڭغۇلچە (蒙古语)نى تاللىشى ئېنىق.


Microsoft نىڭ ئۇيغۇرچە تىل بولىقىنى بەلگىلەنگەن پىلاندىن بۇرۇن ئېلان قىلىۋېتىشىنى مەنمۇ بىلمىدىم. چۈنكى، تىل قوشۇشتىن پارسچە (ئىران)، ئۆزبېكچە (ئۆزبېكىستان)، قازاقچە (قازاقىستان)،…. تىل بولىقىنى ياكى ئەرەب يېزىقى ئاساسىدىكى باشقا تىل بولىقى كۇرد (كۇردىستان)، پوشتۇ (پەنجاپ)،…نى چۈشۈرۈپ سىناپ بېقىڭلار. بۇلارنىڭ ئىچىدە ئەرەبچە ۋە ئۇيغۇرچە تىل بولىقىدىن باشقىلىرىنى تېخىچە ئىشلەتكىلى بولمايدۇ. بەلكىم ئۇلار تەييار بولمىغان بولۇشى مۇمكىن، سۈپىتى تەكشۈرۈلۈۋاتقان بولۇشى مۇمكىن.


ئەمدى گېپىمىزنىڭ مۇھىم مەزمۇنىغا كەلسەك، بۇ يەردە بەزى تور بېكەت (مۇنبەر ياكى بىلوگلار)نى ئاشكارە تىلغا ئالغۇم كەلمىدى، ئەمما بۇ تور بېكەتلەر Windows 8 نىڭ ئۇيغۇرچە تىل بولىقىنى قايتىدىن پىششىقلاپ ئىشلەپ، بەزى ئۇچۇرلارنى قوشۇپ، ئېنىق ئېيتقاندا پېتىغداپ ياكى بۇلغاپ بولغاندىن كېيىن بەيدۇ تور دىسكىسىغا يوللاپ، ئۆز تور بېكىتىدىن چۈشۈرۈش ئۇلانمىسىنى تەمىنلەپتۇ، ھەتتا ئۆز تور بېكىتىنىڭ ئېلانى سۈپىتىدە بىلوگىمغا ئىنكاس يېزىشقىمۇ ئۈلگۈرۈپتۇ. مەن نېمە كويدا، سىلەر زادى نېمە كويدا؟ ئەمما بۇنداق قىلىشنىڭ ياخشى بولمايدىغانلىقىنى ھېس قىلغان بەزى پىكىرى ئۆتكۈر تورداشلارمۇ بۇ ئىشقا ۋاقتىدا دىققەت قىلىپ، ئۇنداق قىلماسلىقنى تاپىلاپتۇ.


بۇنداق قىلمىش «Microsoft Windows 8 تىل ئارايۈزى بولىقىنى ئىشلىتىش ئىجازەتنامىسى»گە ئاشكارە خىلاپلىق قىلغانلىق بولۇپ، Microsoft بۇنداق قىلمىشنى ئەڭ ئېغىر كۆرىدۈ، ۋاقتى كەلگەندە قانۇنىي جاۋابكارلىقىنى سۈرۈشتە قىلالايدۇ. كېيىن تىل بولىقىنى توردىن چۈشۈرۈش تەمىنلەنگەندە، چۈشۈرۈشتىن ئىلگىرى «ئىشلىتىش ئىجازەتنامىسى»(ئۇيغۇرچە)نى توردىن كۆرەلەيسىز. ھەرقانداق ئادەمنىڭ تىل بولىقىنى ئۆز ئالدىغا قايتا پىششىقلاپ ئىشلەشكە، ئۆز ئېلانلىرىنى قوشۇشقا،…ئاشكارە تارقىتىشقا قەتئىي رۇخسەت قىلىنمايدۇ.


يەنە بىرسى بۇ كەسپىي ئەخلاق جەھەتتىمۇ قەبىھ قىلمىشتۇر. ناۋادا Microsoft مەھسۇلاتلىرىغا زىيادە ئامراق بولساڭلار، مەن ئالاقىلىشىپ بېرەي، بېيجىڭدىكى Microsoft ئاسىيا-تىنچ ئوكيان قۇرۇلۇش ئاكادېمىيەسىگە بېرىپ 1-2 ئايلىق پىراكتىكانت خىزمەتچى بولساڭلارمۇ بولىدۇ. لېكىن، بۇنداق ئەخلاقسىزلىق قىلماڭلار. باشقىلار كۆرسە بۇلاردىمۇ بۇنداق ئىش بار ئىكەن دەپ ھەيران قالىدۇ.


يەنە مۇھىم بىرسى ھەممە ئادەم ئورگان تور بېكىتىدىن ئۇيغۇرچە تىل بولىقىنى چۈشۈرۈپ قاچىلىسا، ئۇيغۇر تىل يېزىقىغا بولغان ئېھتىياجنى ۋە قىزغىنلىقنى ئېنىق بىلەلەيدۇ. بۇ Microsoft نىڭ باشقا مۇھىم مەھسۇلاتلىرىغىمۇ ئۇيغۇرچە تىل بولىقىنى كېڭەيتىشكە بەلگىلىك پايدىسى بولىدۇ.


ئەگەر تور بېكىتىڭىز بولسا، ئازىراق ئېلان ئورنىدىن ۋاز كەچسىڭىزمۇ Windows 8 نى تەشۋىق قىلىپ قويۇڭ. مەن تېخى بارلىق ئۇيغۇرچە تور بېكەتلەردە بەس-بەستە Windows 8 توغرىلىق تەشۋىقات قىلامدىكىن دەپ ئويلاپتىمەن. ھازىرمۇ كېچىكمەيسىز، تور بېكىتىڭىزدە بەس-بەستە ئىشلىتىۋاتقان Microsoft نىڭ ئاشۇ كىچىككىنە eot نىڭ يۈزىنى قىلىپ بولسىمۇ، 2-3 ئاي تەشۋىق قىلىپ قويۇڭ. سېتىۋېلىشنى خالايدىغانلار ئېتىبار مەزگىلىدىن پايدىلىنىپ سېتىۋالسۇن. ئىناۋەت كارتىسى سەۋەبىدىن سېتىۋالالمىغانلارغا ئىناۋەت كارتىسى بارلار ياردەملىشىپ ئېلىشىپ بەرسۇن. لېكىن ئىمكان بار پايدا ئېلىشنى مەقسەت قىلماڭ. چۈنكى دۇرۇس يۇمشاق دېتال سېتىۋالىمەن دەپ ئالدىڭىزغا كەلدى، بۇ ھەرگىزمۇ ئاسان ئەمەس. 250 يۈەن بەرسىمۇ 2 يۈەننى قايتۇرۇپ بېرىڭ. ئەجرىڭىزنى باشقا يەردىن تېپىڭ. بىرسىگە ياخشىلىق قىلسىڭىز، چوقۇم شۇ كىشى قايتۇرۇشى كېرەك دەيدىغان تامادا بولماڭ، ياخشىلىق بەلكىم ئويلاپ باقمىغان باشقىلار تەرىپىدىن سىزگە قايتىشى مۇمكىن.


ياكى بۇ دېگەن تىجارەت دېسىڭىز، دوللارچى تونۇشلىرىڭىزغا دەپ قويۇڭ. مۇشۇ قارا سوغۇقتىمۇ 100 دوللار ئالماشتۇرسا، ئاران نەچچە پۇڭ ياكى نەچچە موچەن پايدا ئالىمەن دەپ بانكىنىڭ ئالدىدا ئولتۇرىدۇ. ماۋۇ 40 دوللارلىق نەرسىنى ئېلىپ ساتىمەن دېسە گەپ يوق 1-2 دوللار پايدا ئالالايدۇ. ئۇلاردا دوللار چىقىم قىلالايدىغان نۇرغۇن كارتا بار.


ئارتۇق ۋاقتىڭلار بولسا، باشقىلارنىڭ نەرسىسىنى بۇزىمەن دەپ يۈرمەي يېڭى نەرسىلەرنى ئۆگىنىڭلار، مەسىلەن: Windows 8 دۇكىنىغا ئەپ يېزىشنى مەشىق قىلىڭلار. شىركەت بولساڭلار Windows دۇكىنىغا تىزىملىتىشقا ياردەملىشىمەن. شەخس بولۇپ ئەپ يېزىپ قالساڭلار Windows دىكى ئۇيغۇرسوفت دۇكىنىدا ئېلان قىلىپ سېتىۋالساڭلار بولىدۇ.


قېنى تور بېكەتلىرىڭىزدە بىرنەچچە ئاي بولسىمۇ Windows 8 توغرىسىدا تەشۋىق قىلالامسىز-يوق؟
