UI لايىھەلىگۈچى قوبۇل قىلىنىدۇ

بۇ يازمىنى ئاخىرىدا   tadubeg تەھرىرلىگەن. ۋاقتى  2015-1-25 13:11  




     ئالدىمىزدىكى نەچچە كۈن ئىچىدە يېڭى ئىشخانىغا كۆچىمىز ۋە بەزى يېڭى تۈرلەرنى ئېچىش خىزمىتىنى چىڭ تۇتۇپ باشلاپ كېتىمىز.
     شىركىتىمىزنىڭ UI لايىھەلىگۈچىسى ئابلىمىت ئابلەت (گىگانت) ئىتالىيەدە لايىھەلەش كەسپى بويىچە ماگىستىرلىقتا ئوقۇۋاتىدۇ. ئۇ تۇرۇشلۇق شەھەر بىلەن ئۈرۈمچىنىڭ ئالتە سائەت ئەتراپىدا ۋاقىت پەرقى بولغاچقا، بەزى جىددىي ۋاقىتلاردا ئالاقىلىشىش قولايسىز بولۇش ئەھۋاللىرى كۆرۈلدى. شۇڭا ئۆزىمىزنىڭ ھەم باشقىلارنىڭ تۈرلىرىنى ۋاقتىدا تاماملاشقا كاپالەتلىك قىلىش ئۈچۈن، ئابلىمىتقا ياردەمچى قىلىپ، شىركىتىمىزدە مۇقىم ئىشلەيدىغان يەنە بىر نەپەر UI لايىھەلىگۈچى قوبۇل قىلىشنى قارار قىلدۇق.
     ۋەزىپىسى: شىركىتىمىزنىڭ خىزمەت ئورۇنلاشتۇرۇشى بويىچە ئەپ، يۇمشاق دېتال، تور بېكەتلەرنىڭ UI (كۆرۈنمە يۈز) نى ۋە «كومپيۇتېر ۋە تۇرمۇش» ژۇرنىلىنىڭ گۈزەل-سەنئەت لايىھەسىنى ئىشلەش قاتارلىق خىزمەتلەرنى قىلىدۇ.
     خىزمەت ئورنى: شىركىتىمىزدە خىزمەت قىلىدۇ.
     خىزمەت ۋاقتى: ھەپتىدە 6 كۈن، كۈندە 8 سائەت خىزمەت قىلىدۇ. قانۇندا بەلگىلەنگەن ھېيت-بايراملاردا ئارام ئالىدۇ.
     تەلەپ: رەڭ قارىشى يۇقىرى بولۇشى، مۇستەقىل لايىھە ئىشلەش ئېڭى بولۇشى، ئىلگىرى قىلىپ باققان ئەمەلىي ئەمگىكى بولۇشى، Photoshop ، Illustrator، Coreldraw قاتارلىق تەكشىلىكتە سۈرەت بىر تەرەپ قىلىش يۇمشاق دېتاللىرىنى پۇختا بىلىشى، ھەمكارلىشىپ تۈر ئېچىشقا ماھىر بولۇشى، ۋاقىت قارىشى يۇقىرى، مەسئۇلىيەتچان، تىرىشچان، ئەخلاقلىق، راستچىل، سەمىمىي بولۇشى، شىركىتىمىز بىلەن كەم دېگەندە ئىككى يىللىق ئەمگەك توختامى تۈزۈشى كېرەك.
     يىللىق كىرىمى: UI لايىھەلىگۈچىنىڭ يىللىق كىرىمى «ئاساسىي مائاش (24 مىڭ يۈەندىن 120 مىڭ يۈەنگىچە) + ئالتە خىل سۇغۇرتا + تۈر مۇكاپاتى + قەرەلسىز مۇكاپات + ھېيتلىق تەمىنات + يىللىق مۇكاپات + پاي ئايرىش» ئۆلچىمى بويىچە بولىدۇ.
UI لايىھەلىگۈچى شىركەتتىكى تۆھپىسىگە ئاساسەن، شىركەت پېيىغا ئېرىشەلەيدۇ. ئايرىپ بېرىلىدىغان پاي ئەڭ تۆۋەن بولغاندا %1، ئەڭ يۇقىرى بولغاندا %49 بولىدۇ.
     ئەسكەرتىش:
     1. ئوقۇش تارىخى بولۇشى تەلەپ قىلىنمايدۇ. تېخنىكا، ئەمەلىي ئىقتىدار ۋە ئەمەلىي كۈچكە ئىگە ھەرقانداق كىشى ئىشلىسە بولىدۇ.
     2. مىللەت، جىنس، ياش چەكلەنمەيدۇ.
     3. UI لايىھەلىگۈچى رەسمىي خىزمەتكە چۈشۈشتىن ئىلگىرى ئىككى ئاي سىناق ئىشلەيدۇ. سىناق مەزگىلىدە 1500 يۈەن ئىش ھەققى بېرىلىدۇ. سىناقتىن كېيىن UI لايىھەلىگۈچى شىركىتىمىزدىن، شىركىتىمىز UI لايىھەلىگۈچىدىن رازى بولسا، ئەمگەك توختامى تۈزۈلىدۇ. [ئەسكەرتىش: (1) UI لايىھەلىگۈچىنىڭ ئەمەلىي ئەھۋالىغا قاراپ، سىناق ۋاقتىنى بىر ئايغا قىسقارتىشقا بولىدۇ؛ (2) UI لايىھەلىگۈچىنىڭ ئەمەلىي ئەھۋالىغا قاراپ، سىناق مەزگىلىدىكى مائاشىنى ئۆستۈرۈشكە بولىدۇ.]
     ئۆزىڭىزنى ياكى ئەتراپىڭىزدىكى تونۇش-بىلىشلىرىڭىزنى بۇ خىزمەتكە مۇناسىپ دەپ قارىسىڭىز، بىز بىلەن ئالاقىلىشىڭ.
     بۇ ئۇچۇردا ئېنىقسىز نۇقتىلار بولسا، تەپسىلاتىنى خىزمەت ۋاقتىدا بىۋاسىتە ئىشخانىمىزغا كېلىپ ياكى تېلېفون قىلىپ سۈرۈشتۈرۈڭ.
     شىركەت ئادرېسى: ئۈرۈمچى شەھىرى يەنئەن يولى 154-نومۇر جىلى سودا ئىشلىرى بازىسى 3-قەۋەت 37-ئىشخانا
