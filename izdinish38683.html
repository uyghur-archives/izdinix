نېمە ئۈچۈن Windows 7 نى قاچىلىغاندا 100MB يۇشۇرۇن بۆل

نېمە ئۈچۈن Windows 7 نى قاچىلىغاندا 100MB يۇشۇرۇن بۆلەك (隐藏分区) ھاسىل بولىدۇ؟ Windows 7 يېتەكلەش ھۆججختى (引导文件) ۋە WinRE نى بۇ بۆلەككە ساقلاپ يۇشۇرۇپ قويىدۇ. بۇنىڭدىن مەقسەت پەقەت يېتەكلەش بۆلىكىنى قوغداشلا بولۇپلا قالماستىن يەنە، Bitlocker قوزغاتقۇچ مەخپىليەشتۈرۈش (Bitlocker 驱动器加密) ئىقتىدارىنى ئىشلىتىدىغان ئىشلەتكۈچىلەرنىڭ قاتتىق دېسكا قۇرۇلمىسىنى قايتىدىن ئورۇنلاشتۇرۇش ئاۋارىچىلىقىدىن قۇتۇلدۇرۇش. ئاددىيراق قىلىپ ئېيتقاندا Bitlocker قوزغاتقۇچ مەخپىليەشتۈرۈش (Bitlocker驱动器加密) قاتتىق دىسكىدا ھۆججەت مەخپىلەشتۈرگەندە يېتەكلەش ھۆججىتىنى مەخپىيلەشتۈرەلمەيدۇ، ئەگەر يېتەكلەش ھۆججىتىنى مەخپىيلەشتۈرسە، ئېنىقكى سېستىما قوزغىلالمايدۇ، ئەگەر ئىشلەتكۈچى قوزغاتقۇچ مەخپىليەشتۈرۈش (Bitlocker驱动器加密) ئىشلەتمەكچى بولسا جەزمەن سېستىما يېتەكلەش ھۆججىتىنى باشقا بۆلەككە ساقلاش كېرەك. شۇڭا بۇ 100MB لىق بۆلەك بىھاجەت ئەمەس. ئەلۋەتتە، مەخپىيلەشتۈرۈش ئىقتىدارىنى ئىشلەتمىسەك بۇ بۆلەك ئۇنچىۋالا مۇھىم بولمايدۇ. ئۆچۈرىۋەتسەك ياكى قۇرمىساقمۇ بولىدۇ.

Windows 7 نى قاچىلاش داۋامىدا تۆۋەندىكىدەك ئىككى خىل ئەھۋالدا 100MB لىق سېستىما قالدۇرۇش بۆلىكى(系统保留) قۇرىدۇ:

1. Windows 7 نى يېڭىدىن قاچىلاش.

2. Windows 7 دىن پايدىلىنىپ قاتتىق دېسكىنى بۆلەككە ئايرىش ۋە فورماتلاش

يۇقىرىقى ئىككى خىل ئەھۋال ئاستىدا، Windows 7 نىڭ بۆلەككە ئايرىش ۋە فورماتلاش قۇرالىنى ئىشلەتكىلى بولىدۇ. تۆۋەندە بۇ بۆلەكنى ھاسىل قىلماسلىق ئۇسۇلى تۇنۇشتۇرىلىدۇ.

1) Windows 7 نى قاچىلاش دىسكىنى ئوپتىك دېسكا قوزغاتقۇچقا سېلىپ، Windows 7 قاچىلاش باسقۇچىغا كىرىمىز. بۇ يەردە جەزملەشتۈرۈ  (确定) كۇنۇپكىسىنى باسىمىز.

تۆۋەندىكى رەسىمدىكىدەك تاللاپ، باش رايۇن (主分区) بۆلەكنى ئۆچۈرىمىز.

3. سېستىما قالدۇرۇش بۆلىكى تاللاپ ئاستىدىن كېڭەيتىش (扩展) نى چىكىمىز.

4. كېڭەيتىش چوڭ –كىچىكلىكى (扩展大小) غا بايىقى C دىسكىنىڭ سىغىمىنى كىرگۈزۈپ ئىشلىتىش (应用) كۇنۇپكىسىنى باسىمىز.

5. قاراڭ، ئەمدى بولدى. مۇشۇنداق قىلساقلا بولدى. ئەمدى خاتىرجەم سېستىمىنى قاچىلاڭ.

يۇقارقى جەريانلارنى كۆرۈپ چۈشىنىۋالدىڭىزغۇ دەيمەن. ئىككىنچى بۆلەكنى ئۆچۈرىمىز. بىرىنچى بۆلەك 100MB نى كېڭەيتىمىز.
