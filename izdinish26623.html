turbu c 2.0 دىكى دائىم كۆرۈلىدىغان خاتالىق ئۇچۇرى
operator not followed by maco argument name#    (ئەمەل بەلگىسى # نىڭ ئارقىسىغا ئۆزگەرگۈچى ئۇلانمىغان)
تەھلىل ۋە بىر تەرەپ قىلىش: ماكرو ئېنىقلىما(宏定义)دا ، #بولسا بىر ماكرو ئۆزگەرگۈچىلەر تىزمىسىنى (宏变串) پەرىقلەندۈرۈشكە ئىشلىتىلىدۇ.
'#' بەلگىسى ئارقىسىغا چوقۇم بىر ماكرو ئۆزگەرگۈچىلەر تىزمىسى ئۇلانغان بولىشى شەرت.

'xxxxxx' not anargument    (كۆرسىتىلگەن  'xxxxx' بولسا فونكىسيە پارامتىرى ئەمەس.)
تەھلىل ۋە بىر تەرەپ قىلىش: ئەسلى پروگراممىدا xxxxx نى بىر فونكىسيە پارامتىرى دەپ تونىغان،ئەمما بۇ بەلگە فونكىسيىدە كۆرۈلمىگەن.

Argument # missing name  (فونكىسيە # نامى يوقالغان)
تەھلىل ۋە بىر تەرەپ قىلىش:  پارامتىر نامى بىكىتىلگەن فونكىسيىنىڭ فونكىسيىلىك ئەسلى تىپىدا(函数原型)ئىشلىتىلگەن.
ناۋادا فونكىسيەنى ئەسلى تىپ شەكىلدە ئېنىقلىساق ، بۇ فونكىسيە بارلىق پارامتىر نامىنى ئۆز ئىچىگە ئېلىشى شەرت.

Argument list syntax error   (پارامتىردا گىرامماتىكىلىق خاتالىق كۆرۈلگەن)
تەھلىل ۋە بىر تەرەپ قىلىش: فونكىسيە يۆتكەپ ئىشلەتكەن پارامتىرلار چوقۇم پەش بەلگىسى بىلەن ئايرىلىشى ۋە تىرناقىنىڭ يىپىلىشى بىلەن
ئاخىرلىشىشى كىرەك. ناۋادا ئەسلى ھۆججەتتە پەش بەلگىسى بىلەن ئاخىرلاشمىغان ياكى تىرناق يېپىلمىغان پارامتىرلار بولسا مۇشۇ خاتالىق كۆرۈلىدۇ.

Array bounds missing   (سانلار گۇرۇپپىسىنىڭ چىگرا بەلگىسى «[» يوقالغان).
تەھلىل ۋە بىر تەرەپ قىلىش: بىر سانلار گۇرۇپپىسى بىكىتىلىپ ، يۇقىردىكى بەلگە بىلەن ئاخىرلاشتۇرۇلمىغان بولسا مۇشۇ خاتالىق كۆرۈلىدۇ.

Array size too large   (سانلار گۇرۇپپىسى بەك چوڭ)
تەھلىل ۋە بىر تەرەپ قىلىش: ئېنىقلانغان سانلار گۇرۇپپىسى بەك چوڭ بولۇپ، ئىشلىتىشكە بولىدىغان ئىچكى ساقلىغۇچ سېغىمىدىن ئېشىپ كەتكەن.

Assembler statemet too long    (مەجمۇئە جۈمىلىسى(汇编语句) بەك ئۇزۇن)
تەھلىل ۋە بىر تەرەپ قىلىش: ئىچكى مەجمۇئە جۈمىلىسى(内部汇编语句) ئەڭ ئۇزۇن بولغاندىمۇ 480بايىتتىن ئېشىپ كەتسە بولمايدۇ.
