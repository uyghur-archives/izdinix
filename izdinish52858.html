win8 ئاكىتىپلاش ھەققىدە ياردەم

ھۆرمەتلىك تورداشلار، بۈگۈن كۆزنەك 8 نى توردىن چۈشۈرۈپ، بارماق دېسكا بىلەن قاچىلىغان ئىدىم، لېكىن ھېچقانداق ئاچقۇچ نۇمىرىنى سورىماي قاچىلاپ كەتتى، قاچىلاپ بولغاندىن كىيىنمۇ، شۇنچە قىلىپ ئاكىتىپلايدىغان ئورۇننى تاپالمىدىم. ئاكىتىپلاشنى باسساممۇ خاتالىق ئۇچۇرى چىقىۋالدى،

زىلتار ئەپەندى دېگەن تۆۋەندىكىدەك ئۇسۇلمۇ ئاقمىدى.

زىلتار ئەپەندىنىڭ كۆرسەتمىسى:

ndows8نىڭ مەھسۇلات ئاچقۇچنى سېتىۋالغان بولسىڭىزمۇ،Windows8نى قاچىلاپ ئاكتىپلاش جەريانىدا

Invalid Product Key Windows 8 provided by Microsoft- Windows 8 installer not functioning code: Oxc004f061

خاتالىق كۆرۈلسە، يەنى ئىناۋەتسىز مەھسۇلات ئاچقۇچى دېگەن خاتالىق.

بۇ خىل خاتالىق بىر قانچە خىل ئەھۋالدا كۆرۈلىشى مۇمكىن ئىكەن.

I. باشقا كومپىيۇتېردا مەھسۇلات سېتىۋىلىپ يەنە باشقا كومپىيۇتېردا قاچىلىغاندا

II. سىستېمىنى پۈتۈنلەي ئۆچۈرۋېتىپ دېسكىدىن قاچىلىغاندا

III. ماك كومپىيۇتېرلارغا دېسكىدىن قاچىلىغاندا

ۋە باشقىلار

ھەل قىلىش ئۇسۇلى:

1-باسقۇچ: سىستېما دېسكىسى ئاستىدىكى يەنى س پەن ئاستىدىكى Windows ئاستىدىكى Registry Editor نى قوزغىتىمىز.

2-باسقۇچ: MediaBootInstall نى ئىزدەپ، قىممىتىنى 0 قىلىمىز.

3-باسقۇچ:

(command prompt (run as administrator

نى قوزغىتىمىز. يەنى C:WindowsSystem32 ئاستىدىكى CMD نى قوزغۇتۇپ slmgr /rearm بۇيرۇقىنى ئىجرا قىلىمىز.

4-باسقۇچ: كومپىيۇتېرنى قايتا قوزغۇتىمىز.

5-باسقۇچ: سېتىۋالغان مەسھۇلات ئاچقۇچنى سىلىپ ئاكتىپلايمىز.





شۇڭا ئورگان تەرەپكە تېلفۇن قىلماي تۇرۇپ، سېستمىنى قايتا قاچىلىماستىن ھەل قىلىدىغان ئۇسۇل بارمىدۇ ؟





    2. پۈتۈن چۈشۈرىدىغاننى چۈشۈرۈپ  بولۇپ، سېستىمىنى بىر كۇنۇپكا بىلەن ئەسلىگە كەلتۈرۋالساقمۇ بولامدۇ ياكى بىھاجەتمۇ ؟  一键恢复
