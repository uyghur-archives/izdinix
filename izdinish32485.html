Win XP نىڭ پارامىتىرلىرىنى تەڭشەش ئارقىلىق ئىشلەش سۈر

مەشغۇلات سېستىمىسى Win XP دا ، 90 نەچچە خىل مۇلازىمەت (Service) بار بولۇپ ، ئەدەتتە 30 نەچچە خىل مۇلازىمەت تۈرى ئوچۇق بولىدۇ. ئەمەلىيەتتە بىز ئۇنىڭ ئىچىدىكى بىر قانچىسىنىلا ئىشلىتىمىز. ئىشلەتمەيدىغان مۇلازىمەت تۈرلىرىنى ئىتىۋەتكەندە خېلى كۆپ سىستىما مەنبەسىنى ۋە ئېچكىسى سىغىم بوشلۇقىنى تىجەپ قالغىلى بولىدۇ.
بۇ مۇلازىمەتلەرنىڭ قايسىسىنىڭ ئىشلەۋاتقان ، قايسىسىنىڭ ئوچۇق ئىكەنلىكىنى تۆۋەندىكى ئۇسۇل بىلەن كۆرگىلى، ئېتىۋەتكىلى ۋە ئاچقىلى بولىدۇ.
«باشلاش تىزىملىكى» -«ئىجرا» غا services.msc نى كىرگۈزىمىز. بۇ ۋاقىتتا سېستىما Xp نىڭ مۇلازىمەت تىزىملىكى ئېچىلىدۇ. بۇ يەردىن ھەرقايسى مۇلازىمەتلەرنى قوزغاتقىلى ، ئېتىۋەتكىلى ، ئاپتوماتىك قوزغىلدىغان قىلىپ تەڭشەپ قويغىلى بولىدۇ.

تۆۋەندە مۇلازىمەتلەرنىڭ رولى ۋە بىر تەرەپ قىلىش توغرىسىدا توختىلىمىز.
1. alerter مۇلازىمىتى : خاتالىق مەلۇم قىلغۇچ . تەۋسىيە : ئېتىۋىتىڭ.
2. application layer gateway service مۇلازىمىتى : ئۇچىنچى تەرەپ بىلەن توردىن تەڭ بەھىرلىنىش / ئوتتىن مۇداپىئە تېمىنى قوللاش مۇلازىمىتى (بەزى مۇداپىئە تاملىرى 防火墙 / توردىن تەڭ بەھىرلىنىش دىتاللىرى مۇشۇنى ئىشلىتىدۇ) MB1.5 ئىچكى سىغىمنى ئىگىلەيدۇ .
3. application management مۇلازىمىتى : يۇمشاق دىتاللارنى تەڭشەش ، ئۆچۈرۈش مۇلازىمىتى .
4. automatic updates مۇلازىمىتى : windows نىڭ ئاپتوماتىك يېڭىلاش مۇلازىمىتى.
5. automatic updates مۇلازىمىتى : بۇ مۇلازىمەت ئارقىلىق http1.1 مۇلازىمىتىر ئارسىدا ئۇچۇر يەتكۈزۈشنى ئەمەلگە ئاشۇرۇلىدۇ ، windows نى يېڭىلىغاندا ئۈزۈلۈپ قالغان يەردىن داۋاملاشتۇرۇشنى ئەمەلگە ئاشۇرىدۇ.
6. clipbook مۇلازىمىتى : دائىرلىك توردىكى كومپيۇتېر ئارىقلىق مەزمۇننى چاپلاش /كىسىشتىن ئورتاق بەھىرلىنىشكە ئىشلىتىلىدۇ. تەۋسىيە : ئىتىۋىتىڭ.
7. com+Event system مۇلازىمىتى : بەزى COM+ يۇمشاق دىتالغا كىرەكلىك . سىز كومپيۇتېرڭىزدىكى تۆۋەندىكى مۇندەرىجىدە(c:\program files\ComPlus Applications) ھېچقانداق ھۆججەت بولمىسا بۇ مۇلازىمەتنى ئېتىۋەتسىڭىز بولىدۇ.

8. COM+Event system application مۇلازىمىتى : يۇقۇرقىغا ئوخشاش.

9. Computer browser مۇلازىمىتى : دائىرلىك توردىكى كومپيۇتېرلارنى زىيارەت قىلىش مۇلازىمىتى. لىكىن ئىتىۋەتسىمۇ زىيارەت قىلىشقا تەسىر يەتكۈزمەيدۇ.

10. cryptographic services : windows نى توردىن ئاپتوماتىك يېڭىلغاندا windows ھۆججەتلىرىگە بەلگە سېلىشقا ئىشلىتىلىدۇ. windows نى يېڭىلىغاندا ئېچىۋىتىڭ.

11. DHCP client مۇلازىمىتى : مۇقۇم IP ئادرىسى بارلار ئىشلىتىدۇ. (DSLقاتارلىقلار ) . ئەگەر مودىم ئارقىلىق تورغا چىقسىڭىز ئىشلەتمەيسىز.

12. Distributed link tracking client مۇلازىمىتى : دائىرلىك توردىكى ئۇلىنىشنى ئۇچۇرنى يېڭىلاش ئۈچۈن ئىشلىتىلىدۇ. مەسلەن بىر كومپيۇتېردىكى بىر ھۆججەت ، يەنە بىر كومپيۇتېرغا ئۇلىنىش قىلىنغان بولسا ، ئەگەر ھۆججەت يۆتكەلسە ، بۇ مۇلازىمەت ئۇلىنىش ئۇچۇرىنى يېڭىلايدۇ. 4MB سىغىمنى ئىگىلەيدۇ.

13. DNS Client مۇلازىمىتى : DNS نى ئانالىز قىلىش مۇلازىمىتى.

14. Error reporting service مۇلازىمىتى : خاتالىق مەلۇم قىلغۇچ ، windows تا يۈز بەرگەن خاتالىقلارنى مىكروسوفىتقا مەلۇم قىلىدۇ. تەۋسىيە : ئېتىۋىتىڭ.

15. Event Log مۇلازىمىتى : سېستىما خاتىرىسى تىزىملاش مۇلازىمىتى.

16. Fast user switching compatibility مۇلازىمىتى : كۆپ ئابونىت ئارسىدا تىز سۈرئەتتە ئالماشتۇرۇش مۇلازىمىتى.

17. help and support مۇلازىمىتى : يارىدەم مۇلازىمىتى . تەۋسىيە : ئېتىۋىتىڭ.

18. Human interface device access مۇلازىمىتى : كومپيۇتېردىكى ئالاھىدە ئورنىتىلغان سەپلىمىلەرنى قوللاش ، مەسلەن كونۇپكا تاختىسىدىكى ئاۋازنى كونترول قىلىش كونۇپكىسى ......قاتارلىقلار.

19. IMAPI CD-burning COM service مۇلازىمىتى : XP نىڭ دىسكا ئويۇش مۇلازىمىتى .

20. Indexing service مۇلازىمىتى : مەزكۇر كومپيۇتېر ۋە توردىكى باشقا كومپيۇتېرلاردىكى ھۆججەتنىڭ ئاچقۇچلۇق مەزمۇن ۋە خاراكتىرى بىلەن تەمىنلەپ ئابونىتلارنى تىز ئىزدەپ تېپىش مۇلازىمىتى بىلەن تەمىنلەيدۇ.

21. Internet Connection Firewall(ICF) مۇلازىمىتى : XP نىڭ ئوتتىن مۇداپىئە تېمى مۇلازىمىتى . تەۋسىيە : ئىشلەتمىسىڭىز ئىتىۋىتىڭ.

22. IPSEC Services مۇلازىمىتى : IP بىخەتەرلىك سىتراتىگىيسى مۇلازىمىتى . ئادەتتە كۆپ ئىشلىتىلمەيدۇ. تەۋسىيە : ئېتىۋىتىڭ.

23. Logical Disk manager مۇلازىمىتى : دىسكا باشقۇرۇش مۇلازىمىتى .

24. Logical Disk manager administrative service : ئۈستىدىكى بىلەن ئوخشاش.

25. messengerمۇلازىمىتى: msn ئەمەس ، مۇلازىمىتىر بىلەن يەككە كومپيۇتېر ئوتتۇرسىدا خاتالىق ئۇچۇر يەتكۈزۈش مۇلازىمىتى.

26. MS software shadow copy provider مۇلازىمىتى : ساپاسلاش ئىقتىدارى.

27. Net Logon : ئابونىت تىزىملتىشنى باشقۇرۇش مۇلازىمىتى. تەۋسىيە : ئېتىۋىتىڭ.

28. Netmeeting remote desktop sharing : Netmeeting ئارقىلىق باشقا باشقا كومپيۇتېرنى يىراقتىن زىيارەت قىلىش مۇلازىمتى. Netmeeting نى ئىشلەتمىسڭىز ئېتىۋىتىڭ.

29. Network Connections : تورغا ئۇلىنىش ۋە نۇمۇر بۇراپ ئۇلاش مۇلازىمىتىنى باشقۇرۇش مۇلازىمتى.

30 . Network locetion Awareness : ئەگەر توردىن ئورتاق بەھىرلەنمەكچى بولسىڭىز لازىم بولىدۇ.

31. NT LM Security support provider : telnet مۇلازىمىتى.

32. NVIDIA Driver Helper service : nvidia كۆرسىتىش كارتىسنىڭ يارىدەم مۇلازىمىتى. تەۋسىيە : ئېتىۋىتىڭ.

33. PerFORMance logs and alerts : كومپيۇتېرنىڭ ئىشلەش ئەھۋالىنى خاتىرلەش ۋە قەرەرلىك خاتىرگە يېزىش ياكى ئاگاھلاندۇرۇش مۇلازىمىتى.

34. Plug and Play : يېڭى قاتتىق دىتاللارنى ئاپتوماتىك بايقاش ، چىتپلا ئىشلىتىش مۇلازىمىتى.

35. Portable media serial number : كومپيۇتېرغا چېتىلغان ھەرقانداق مئېدىيا ئۈسكۈنىلىرنىڭ نۇمۇرىنى تونۇش مۇلازىمىتى.

36. Print Spooler : ھۆججەتنى كىچىكتۈرۈپ بېسىش مۇلازىمىتى.

37. Protected Storageيەرلىك شىفىر ۋە تورغا چىققاندىكى شىفىرلارنى ساقلاش مۇلازىمىتى ، توردىكى ئاپتوماتىك تولدۇرۇش ئىقتىدارى.

38. Remote access auto connection manager : كەڭ بەلۋاغلىق تورغا چىققۇچىلار ۋە توردىن ئورتاق بەھىرلىندىغانلارغا كىرەك بولىدۇ.

39. Remote desktop help session manager : يىراقتىن يارىدەم مۇلازىمىتنى باشقۇرىدۇ. 4MB سىغىنمنى ئىگىلەيدۇ.

40. Remote Procedure Call (RPC) : سېستىمىنىڭ يادرولۇق مۇلازىمىتى.

41 . Remote Procedure Call LOCATOR : RPC سانلىق مەلۇماتىنى باشقۇرۇش مۇلازىمىتى. 1MB سىغىمنى ئىگىلەيدۇ.

42. remote registry : يىراقتىن يەرلىك كومپيۇتېردىكى تىزىملاش خاتىرسىنى ئىجرا قىلىش ۋە ئۆزگەرتىش مۇلازىمىتى . چوڭ يوچۇق. تەۋسىيە : دەرھال ئېتىۋىتىش كىرەك.

43 . removable storage : دىسكىنى زاپاسلاش مۇلازىمىتى . تەۋسىيە : ئېتىۋىتىڭ.

44. secondary logon : administrator دىن باشقا ئابونىتلارغا ھوقۇقلارنى تەقسىملەش مۇلازىمىتى.

45. security accounts manager : يەرلىك كومپيۇرنىڭ ئابونىتلارنىڭ بىخەتەرلىك ئۇچۇرلىرىنى ساقلايدۇ.

46. server : دائىرلىك توردا ھۆججەت ۋە پىرىنتىلارنى ئېچىش ۋە ئورتاق بەھىرلىنىش مۇلازىمىتى.

47. shell hardware detection : دىتاللارنىڭ ئاپتوماتىك قوزغۇلۇش ئىقتىدارىنى ئۇقتۇرۇش ۋە باشقۇرۇش. ئوپتىك دىسكا ، بارماق دىسكا قاتارلىقلار......

48. smart card : مەزكۇر كومپيۇتېرنىڭ ئىقتىدار كارتىسىنى ئوقۇشىنى باشقۇرۇش مۇلازىمىتى.

50. smart card helper : smart card ياردەمچىسى.

51. SSDP Discovery service : ئائىلە تورىدىكى UPnP ئۈسكىنىسىنى تونۇش مۇلازىمىتى.

52. system event notification : ئابونىتنىڭ تىزىملىتىش /تىزىمدىن چىقىش/قايتا قوزۇغۇتۇش / كومپيۇتېرنى ئېتىش قاتارلىق ئۇچۇرلىرىنى خاتىرلەش مۇلازىمىتى.

53. system restore service : سېستىمىنى ئەسلىگە قايتۇرۇش مۇلازىمىتى. ئىچكى سىغىمنى ۋە سېسىتىما مەنبەسىنى بەك كۆپ ئىگىلەيدۇ.
