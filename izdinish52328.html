بىل گايتىس: Windows 8 كەلگۈسىدىكى مەشغۇلات سىستېمىسىنىڭ ئاساسى

گەرچە پۈتكۈل يەر شارى تەھلىلچىلىرى ۋە مۇتەخەسسىسلەر مىكروسوفتنىڭ Windows 8
ئىشلەتكۈچىلەر ئارايۈزىگە زور دەرىجىدە ئۆزگىرىش كىرگۈزگەنلىكىگە تېخىچە زەربە
بېرىۋاتقان بولسىمۇ، ئەمما مىكروسوفتنىڭ قۇرغۇچىسى بىل گايتىس Windows 8
ئەمەلىيەتتە كەلگۈسىدىكى مىكروسوفت مەشغۇلات سىستېمىسىنىڭ ئاساسى ئىكەنلىكىنى
بىلدۈردى.




بىل گايتىس يېقىندا بىر
زىيارەتنى قوبۇل قىلغاندا، Windows 8 نىڭ بىر قەلىپلەرنى روھلاندۇرىدىغان يېڭى
دەۋرنىڭ باشلىنىشى ئىكەنلىكىنى، مەزكۇر مەشغۇلات سىستېمىسىنىڭ چەكمە سېزىمچان
ئېكراننى قوللايدىغانلىقى ۋە مۇناسىۋەتلىك ئەپلەرنىڭ داۋاملىق كۆپىيىۋاتقانلىقىنى
بىلدۈردى.

Windows 8 نىڭ سېتىلىش مىقدارىنىڭ 60 مىليونغا بارغانلىقىدىن رازى
بولىدىغان-بولمايدىغانلىقى سورالغاندا، بىل گايتىس Win8 مەشغۇلات سىستېمىسىنىڭ
چەكمە سېزىمچان ئېكراننى قوللايدىغانلىقى ۋە ئەپ بازىرىدىكى مۇناسىۋەتلىك ئەپلەرنىڭ
داۋاملىق ئېشىۋاتقانلىقى، بۇنىڭ ناھايىتى ياخشى باشلىنىش ئىكەنلىكىنى، Windows 8
مەشغۇلات سىستېمىسىنىڭ ناھايىتى ياخشى بىر يولدا كېتىۋاتقانلىقىنى
بىلدۈردى.

گەرچە مىكروسوفت Windows 8 مەشغۇلات سىستېمىسىنىڭ سېتىلىش ئەھۋالىنىڭ
ناھايىتى ياخشى ئىكەنلىكىنى بىلدۈرگەن بولسىمۇ، ئەمما يېقىندا مىكروسوفت يەنىلا
بازارنى ياخشىلاشنىڭ نۇرغۇن بوشلۇقى بارلىقىنى ئېتىراپ قىلغان. مىكروسوفت
ھەمكارلاشقۇچىلار بىلەن تېخىمۇ زىچ ھەمكارلىشىشنىڭ 2013-يىلى ئەڭ مۇھىم ئىشلارنى
بىر ئىكەنلىكىنى، شۇڭا بۇ يىل Windows 8 غا قارىتا مىكروسوفتنىڭ تېخىمۇ كۆپ
كۈچەيدىغانلىقى ھەققىدە ۋەدە بەردى.
