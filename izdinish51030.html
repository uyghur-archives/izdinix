Windows 8 نى سېتىۋېلىش (248 يۈەن)



Windows 8 نى سېتىۋېلىش (ياكى ئىشلىتىش) ئۈچۈن، تۆۋەندىكىدەك 2 خىل ئۇسۇل بار


(1) دەرىجە ئۆستۈرۈش

248 يۈەن ئېتىبار باھا بىلەن ھازىر ئىشلىتىۋاتقان Windows نىڭ نەشرىنى Windows 8 Pro (كەسپىي نەشرى)گە ئۆستۈرەلەيسىز. ناۋادا، 2012-يىل 6-ئايدىن باشلاپ، ماركىلىق كومپيۇتېر ئالغان بولسىڭىز، 98 يۈەن تاپشۇرۇپ، 7 Windows نىڭ نەشرىنى Windows 8 Pro (كەسپىي نەشرى)گە ئۆستۈرەلەيسىز.


(2) Windows 8 قاچىلانغان ماركىلىق كومپيۇتېر سېتىۋېلىش

مۇشۇ ئايلاردا يېڭىدىن بازارغا سېلىنغان Sony, Dell, 联想،….قاتارلىق ماركىلىق كومپيۇتېر سېتىۋالسىڭىز، ئىچىدە Windows 8 ھەقىقىي نەشرى قاچىلانغان بولىدۇ. بۇ يەردە (2)-ئۇسۇلنى تونۇشتۇرمايمەن، ئۆزى ياقتۇرىدىغان ماركىدىكى خالىغان كومپيۇتېرنى سېتىۋالسا بولىدۇ.


بىرىنچى: دەرىجە ئۆستۈرۈش ئارقىلىق Windows 8 Pro (كەسپىي نەشرى)نى سېتىۋېلىش


ئالدى بىلەن سىزدە ئىناۋەت كارتىسى (信用卡) ياكى Paypal ھېساباتى بولۇشى كېرەك (مۇشۇنداق ئېنىق دەپ تۇرساممۇ ئادەتتىكى بانكا كارتىلىرىدا سېتىۋېلىشقا بولمامدۇ دەپ سوئال سوراپ يۈرمەڭلار)، مەن پەقەت ئىناۋەت كارتىسى (信用卡) ئارقىلىق سېتىۋېلىشنى تونۇشتۇرىمەن (مۇشۇنداق دەپ تۇرساممۇ يەنە Paypal دا سېتىۋېلىشنى نېمە ئۈچۈن تونۇشتۇرمىدىڭىز دەپ سوراپ يۈرمەڭلار)، شۇڭا ئىناۋەت كارتىسى (信用卡) بېجىرمىگەنلەرنىڭ تۆۋەندىكى مەزمۇنلارنى كۆرمىسىمۇ بولىدۇ. ئىناۋەت كارتىسى (信用卡) بېجىرىش ئۈچۈن، بانكىلارغا بېرىپ سۈرۈشتە قىلىڭ، بۇ يەردە جاۋاب بېرىپ بولالمايمەن، رەھمەت!


(1) ئالدى بىلەن تور كۆرگۈچ ئادرېس بالدىقىغا تۆۋەندىكى ئادرېسىنى كىرگۈزۈپ، Microsoft تور بېكىتىگە يۆتكىلىڭ.

(2) Windows8-UpgradeAssistant (دەرىجە ئۆستۈرۈش ياردەمچىسى)نى قوش چېكىپ ئىجرا قىلىڭ.


(3) Windows8-UpgradeAssistant (دەرىجە ئۆستۈرۈش ياردەمچىسى) كومپيۇتېرىڭىزدىكى مەشغۇلات سىستېمىسىنى تەكشۈرۈپ، دەرىجە ئۆستۈرۈشكە بولىدىغان ياكى بولمايدىغانلىقىنى تەكشۈرۈپ بېرىدۇ ھەمدە ماسلاشمايدىغان (ياكى سىغىشمايدىغان) ئۈسكۈنە ياكى پىروگرامما (دېتاللار)نى كۆرسىتىپ بېرىدۇ.


(4) كېيىنكى (下一步)نى باسسىڭىز، ساقلاپ قالىدىغان مەزمۇنلارنى تاللاش دىيالوگ رامكىسى ئېچىلىدۇ. بۇ يەردە 3 خىل تاللاش بار.

ھېچقانداق مەزمۇنلارنى ساقلىمايمەن


بۇنىڭ ئىچىدىن 1-تاللاش يەنى Windows تەڭشەكلىرى، خۇسۇسىي ھۆججەت ۋە ئەپلەرنى ساقلاپ قېلىشنى تاللاڭ (تەۋسىيە قىلىمەن)، ئاندىن كېيىنكى (下一步)نى بېسىڭ.


(5) سىزگە ماس كېلىدىغان Windows 8 نىڭ باھاسى 248 يۈەن دېگەن ئۇچۇر چىقىدۇ، بۇنىڭ ئىچىدىن سېتىۋېلىش (订购)نى تاللاڭ.


(6) سىز زاكاز قىلغان مەزمۇن كۆرۈنىدۇ، ھېسابات قىلىش (结账)نى تاللاڭ.


(7) پوچتا ئارقىلىق ئەۋەتىش ئادرېسىنى تولدۇرۇپ، كېيىنكى (下一步)نى بېسىڭ


(8) پۇل تۆلەش ئۇسۇلىنى تاللايسىز، بۇنىڭ ئىچىدىن ئىناۋەت كارتىسى (信用卡)نى تاللاپ، كېيىنكى (下一步)نى بېسىڭ


(9) پۇل تۆلەش ئۇچۇرىدا يەنە ئىناۋەت كارتىسى (信用卡) ئۇچۇرىنى تولدۇرۇپ، كېيىنكى (下一步)نى بېسىڭ. ئېسىڭىزدە بولسۇن ئېلېكتىرونلۇق خەت ئادرېسىنى چوقۇم توغرا كىرگۈزۈڭ. مەھسۇلات ئاچقۇچى بۇ يەرگىمۇ ئەۋەتىلىدۇ.


(10) ئاخىرقى باسقۇچتا سىز زاكاز قىلغان ئۇچۇرلارنى جەزملەشتۈرىسىز.


(11) سىز بەرگەن زاكاز بىر تەرەپ قىلىنىدۇ، يەنى بانكا كارتىڭىزدىن 248 يۈەننى تۇتۇپ قالىدۇ.


(12) مەھسۇلات ئاچقۇچىنى تەمىنلەپ بېرىدۇ. بۇ ئاچقۇچنى پۇختا ساقلاڭ، ئۇ يەنە ئېلېكتىرونلۇق خەت ئادرېسىڭىزغا ئەۋەتىلىدۇ.


(13) Windows 8 نى چۈشۈرۈشكە باشلايدۇ، ئادەتتىكى ئەھۋالدا 2Mbs لىق توردىمۇ 1 سائەت 30 مىنۇت ئىچىدە چۈشۈرۈپ بولىدۇ، سەۋر قىلىپ ساقلاپ تۇرۇڭ.


مەن يەنە ISO ھۆججىتى ھاسىل قىلىش، ئوپتىك دىسكىغا ئويۇش ۋە بارماق دىسكىدىن قاچىلاش ئۇسۇللىرىنى يازىمەن. ئاندىن يەنە بۇ ئىنگلىزچە Windows 8 نىڭ تىلىنى خەنزۇچە ۋە ئۇيغۇرچىگە تەڭشەش ئۇسۇللىرىنى مۇشۇ يەردە تونۇشتۇرىمەن. داۋامىغا دىققەت قىلغايسىز. لېكىن تۈزۈك ئىنكاس چۈشمىسە، بۇ تېمىغا قىزىقمايدىكەن دەپ داۋامىنى يازمايمەن.

ھەممىمىز كۆڭۈل بۆلىۋاتقان Windows 8 مۇ ئېلان قىلىندى. بۇ مەشغۇلات سىستېمىسى ئۆزىدىكى ئۆزگىچە يېڭىلىقلار ۋە ئالاھىدە ياخشى بولغان ئىقتىدارلار ئارقىلىق كەڭ كومپيۇتېر ھەۋەسكارلىرىنى ئۆزىگە تارتىپ كەلدى. ئەينى ۋاقىتتا Windows 7 ئېلان قىلىنغاندا نۇرغۇن تورداشلار بىراقلا Windows XP دىن Windows 7 غا كۆچۈشنى خالىمىدى. چۈنكى بۇ يېڭىچە مەشغۇلات سىستېمىسىنىڭ Windows XP بولغان پەرقى زىيادە چوڭ بولغاچقا، كەڭ ھەۋەسكارلار دەماللىققا بۇ يېڭى سىستېما Windows 7 دا مەشغۇلات ئېلىپ بارالماي، گاڭگىراپ قالدى ۋە ئاخىرى كۆپ ساندىكى ھەۋەسكارلار سىستېمىسىنى يەنىلا Windows XP غا قايتۇرۇپ ئىشلەتتى. بۇ يەردە ھەۋەسكارلارنىڭ Windows 7 غا دەرھال كۆچەلمەسلىكى بۇ شاش ئاتنى قانداق مىنىشنى بىلەلمىدى. چۈنكى بۇ شاش ئاتنى قانداق كۆندۈرۈش، ئۆزىمىزگە قانداق خىزمەت قىلدۇرۇش ھەققىدە بىرەر جايدا سىستېمىلىق ماتېرىيال يوق ئىدى. شۇ ۋەجىدىن تا ھازىرغىچە ئارىمىزدا Windows 7 دىن ئىبارەت بۇ شاش ئاتنى مىنىپ باقمىغانلار نۇرغۇن. مانا ئەمدى نۇرغۇنلىرىمىز بۇ Windows 7 دىن ئىبارەت شاش ئاتنى مىنىشكە ئۈلگۈرمەي تۇرۇپ ئۇنىڭدىنمۇ چاپقۇر ئارغىماق ھېسابلانغان Windows 8 مەيدانغا چۈشتى. بۇ ئارغىماق ھەممىمىزنىڭ ئۇنى تېزدىن كۆندۈرۈپ ئۆزىمىزگە خىزمەت قىلدۇرۇشىمىزغا مۇھتاج. ھەممىمىز بىلىمىز، ئات ھەر قانچە ياخشى بولسىمۇ، ئۇنى مىنىشنى بىلمىسەك ئۇ ھالدا بىز ئۈچۈن ئادەتتىكى ئات بىلەن ئارغىماقنىڭ ھېچقانداق پەرقى بولمايدۇ. شۇڭا بلوگگىر تورداشلارنىڭ بۇ چاپقۇر ئارغىماقنى تېزدىن كۆندۈرۈپ، ھەقىقىي پەيزىنى سۈرۈشىگە پايدىلىق بولۇشىنى كۆزدە تۇرۇپ، بۈگۈندىن باشلاپ، مەزكۇر چاپقۇر ئارغىماققا ئائىت ئورگان تەرەپ تەمىنلىگەن «ئارغىماقنى كۆندۈرۈش قوللانمىسى (Windows 8 产品指南)» نى تەرجىمە قىلىپ، كەڭ تورداشلارغا سۇنۇشقا تۇتۇش قىلدى. تورداشلارنىڭ مۇۋاپىق تۈزىتىش پىكىرى بېرىشىنى ئۇمىد قىلىمەن.
